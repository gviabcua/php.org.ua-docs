# RecursiveCachingIterator::getChildren

(PHP 5> = 5.1.0, PHP 7, PHP 8)

RecursiveCachingIterator::getChildren — Повертає дочірні елементи внутрішнього ітератора як об'єкт RecursiveCachingIterator

### Опис

```methodsynopsis
public RecursiveCachingIterator::getChildren(): ?RecursiveCachingIterator
```

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Дочірні елементи внутрішнього ітератора у вигляді об'єкта RecursiveCachingIterator; Повертає **`null`**, якщо елементи внутрішнього оператора відсутні.
