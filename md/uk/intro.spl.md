# Вступ

Стандартна бібліотека PHP (Standard PHP Library, або SPL) - це набір інтерфейсів та класів, які вирішують спільні завдання.

Бібліотека SPL пропонує розробнику набір стандартних структур даних, ітераторів для обходу об'єктів, набори інтерфейсів та стандартних винятків, ряд класів для роботи з файлами та набір функцій на кшталт [spl\_autoload\_register()](function.spl-autoload-register.md)
