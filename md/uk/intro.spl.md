# Вступ

Стандартна бібліотека PHP (SPL) - це набір інтерфейсів та класів, призначених для вирішення стандартних завдань.

SPL надає ряд стандартних структур даних, ітераторів для оббігу об'єктів, інтерфейсів, стандартних винятків, кілька класів для роботи з файлами і надає ряд функцій, наприклад [splautoloadregister()](function.spl-autoload-register.html)
