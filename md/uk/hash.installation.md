## Встановлення

З версії PHP 5.1.2 модуль Hash за замовчуванням поставляється і збирається разом з PHP.

Цю поведінку легко вимкнути за допомогою опції конфігурації --disable-hash. Для попередніх версій потрібно встановлювати [» модуль PECL](https://pecl.php.net/package/hash)

Починаючи з PHP 7.4.0, модуль Hash доданий до ядра PHP і включений за замовчуванням.
