# rnp\_backend\_version

(PECL rnp >= 0.1.1)

rnp\_backend\_version — Повертає версію криптографічної бібліотеки бекенду

### Опис

```methodsynopsis
rnp_backend_version(): string
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає версію у вигляді рядка.
