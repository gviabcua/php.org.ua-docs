# stats\_rand\_gen\_ipoisson

(PECL stats >= 1.0.0)

stats\_rand\_gen\_ipoisson - Обчислює випадкове відхилення від розподілу Пуассона

### Опис

```methodsynopsis
stats_rand_gen_ipoisson(float $mu): int
```

Обчислює випадкове відхилення від розподілу Пуассона з параметром `mu`

### Список параметрів

`mu`

Параметр розподілу Пуассона

### Значення, що повертаються

Випадкове відхилення
