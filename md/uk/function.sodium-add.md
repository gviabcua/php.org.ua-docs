# sodiumadd

(PHP 7> = 7.2.0, PHP 8)

sodiumadd - Додає великі числа

### Опис

```methodsynopsis
sodium_add(string &$string1, string $string2): void
```

Додає параметр `string2` до `string1`, перезаписуючи значення, що зберігається в `string1`. Функція передбачає, що обидва параметри є двійковими рядками, які представляють цілі числа без знака у порядку байтів.

### Список параметрів

`string1`

Рядок, що представляє ціле число довільної довжини без знака у зворотному порядку байтів. Цей параметр передається за посиланням та міститиме суму двох параметрів.

`string2`

Рядок, що представляє ціле число довільної довжини без знака у зворотному порядку байтів.

### Значення, що повертаються

Функція не повертає значення після виконання.
