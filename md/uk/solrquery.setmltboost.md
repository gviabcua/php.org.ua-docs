# SolrQuery::setMltBoost

(PECL solr >= 0.9.2)

SolrQuery::setMltBoost — Встановлює, чи буде запит посилено релевантністю цікавого виразу

### Опис

```methodsynopsis
public SolrQuery::setMltBoost(bool $flag): SolrQuery
```

Встановлює, чи буде запит посилено релевантністю цікавого виразу

### Список параметрів

`value`

Установите\*\*`true`** або **`false`\*\*

### Значення, що повертаються

Повертає поточний об'єкт SolrQuery, якщо використовується значення, що повертається.
