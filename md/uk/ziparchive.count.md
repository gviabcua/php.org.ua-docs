# ZipArchive::count

(PHP 7 >= 7.2.0, PHP 8, PECL zip >= 1.15.0)

ZipArchive::count — Підраховує кількість файлів в архіві

### Опис

```methodsynopsis
public ZipArchive::count(): int
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає кількість файлів у архіві.
