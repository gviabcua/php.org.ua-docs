# pcntlasyncsignals

(PHP 7> = 7.1.0, PHP 8)

pcntlasyncsignals — Увімкнути/вимкнути асинхронну обробку сигналів або отримати поточний статус

### Опис

```methodsynopsis
pcntl_async_signals(?bool $enable = null): bool
```

Якщо пропущено параметр `enable`, функція **pcntlasyncsignals()** поверне поточне значення налаштування. А якщо заданий, асинхронна обробка буде включена, або відключена, залежно від заданого значення.

### Список параметрів

`enable`

Увімкнути або вимкнути асинхронну обробку сигналів.

### Значення, що повертаються

Якщо при використанні значення параметра `enable` одно **`null`**, повертає чи включена асинхронна обробка сигналів. Якщо під час використання параметр `enable` не дорівнює **`null`**, повертає, чи була включена асинхронна обробка сигналів *до* виклик функції.
