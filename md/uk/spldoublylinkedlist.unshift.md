# SplDoublyLinkedList::unshift

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SplDoublyLinkedList::unshift — Вставляє елемент на початок двозв'язкового списку

### Опис

```methodsynopsis
public SplDoublyLinkedList::unshift(mixed $value): void
```

Вставляет`value`в начало двусвязного списка.

### Список параметрів

`value`

Значення, яке додається на початок.

### Значення, що повертаються

Функція не повертає значення після виконання.
