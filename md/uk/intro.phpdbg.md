# Вступ

Оскільки phpdbg реалізований у вигляді модуля SAPI, він може повністю контролювати оточення без вплив на функціональність і швидкість виконання вашого коду.

Phpdbg претендує на те, щоб вважатися легковажним, потужним та простим у використанні відладчиком для PHP. Він надає такі можливості:

-   Покрокове налагодження
-   Гнучкі точки переривання (метод класу, функція, рядок файлу, адреса, опкод)
-   Легкий доступ до PHP за допомогою вбудованої функції eval()
-   API користувача
-   Незалежність від SAPI – легко інтегрується
-   Підтримка конфігураційних файлів PHP
-   Визначення власних суперглобальних змінних JIT
-   Опціональна підтримка введення з клавіатури – зручна робота через термінал
-   Легкий у використанні - читайте документацію :)

**Опції командного рядка**

| Опция | Пример аргумента | Описание                                                                                         |
|-------|------------------|--------------------------------------------------------------------------------------------------|
| з     | c/my/php.ini     | Вказує php.ini, який треба завантажити                                                           |
| д     | dmemorylimit=4G  | Встановлення відповідної директиви php.ini                                                       |
| н     |                  | Заборона використання стандартного php.ini                                                       |
| до    |                  | Не показувати вітальний банер                                                                    |
| в     |                  | Дозволити висновок oplog                                                                         |
| в     |                  | Заборонити використання квітів                                                                   |
| і     | imy.init         | Задати файл .phpdbginit                                                                          |
| І     |                  | Ігнорувати стандартний .phpdbginit                                                               |
| Про   | Omy.oplog        | Задати файл для запису oplog                                                                     |
| р     |                  | Запустити контекст виконання                                                                     |
| рр    |                  | Запустити контекст виконання та вийти після запуску (не звертати увагу на точки зупинки)         |
| е     |                  | Генерувати розширену інформацію для відладчика/профілювальника                                   |
| Е     |                  | Дозволити покроковий eval. Будьте уважні!                                                        |
| з     | s=, -s=foo       | Прочитати код для виконання із стандартного потоку введення з опціональним роздільником          |
| З     | Scli             | Перевизначити ім'я SAPI. Будьте уважні!                                                          |
|       |                  |                                                                                                  |
|       |                  |                                                                                                  |
| л     | l4000            | Задати порт віддаленої консолі                                                                   |
| а     | a192.168.0.3     | Вказати адресу віддаленої консолі                                                                |
| з     |                  | Включити висновок у форматі xml (замість стандартного текстового)                                |
| п     | p, -p = func, -p | Вивести опкоди та вийти                                                                          |
| х     |                  | Вивести довідковий розділ                                                                        |
| У     |                  | Надрукувати номер версії                                                                         |
|       | \- arg1 arg2     | Використовується для поділу аргументів phpdbg та php $argv; будь-які $argv додавайте після нього |
