# Вступ

LZF - це дуже швидкий алгоритм стиснення, ідеальний для економії місця за вкрай низьких витрат часу. Він може бути оптимізований під швидкість або ефективність стиснення на момент компіляції. Цей модуль використовує бібліотеку [»liblzf](http://oldhome.schmorp.de/marc/liblzf.html), створену для своєї мети Марком Лейманом.
