## Класи символів

**Класи символів**

| Последовательность | Описание                                                                                                                                                                                                                                |
|--------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|                    | Одиночний символ, що вказаний або міститься у вказаному діапазоні. Діапазони можна комбінувати із операторами `{+}` і `{-}`. Наприклад, `[a-z]{+}[0-9]` те саме, що і `[0-9a-z]`, а `[a-z]{-}[aeiou]` те саме, що і `[b-df-hj-np-tv-z]` |
|                    | Одиночний символ не вказано та не міститься у вказаному діапазоні.                                                                                                                                                                      |
|                    | Будь-який символ за замовчуванням `[^\n].`                                                                                                                                                                                              |
| д                  | Будь-яка десяткова цифра, `[0-9]`                                                                                                                                                                                                       |
| Д                  | Будь-який символ, крім десяткової цифри, `[^0-9]`                                                                                                                                                                                       |
| з                  | Будь-який символ пробілу, `[ \t\n\r\f\v]`                                                                                                                                                                                               |
| З                  | Будь-який непробільний символ, `[^ \t\n\r\f\v]`                                                                                                                                                                                         |
| в                  | Будь-який символ, що утворює "слово", `[a-zA-Z0-9_]`                                                                                                                                                                                    |
| У                  | Будь-який символ, що не утворює "слово", `[^a-zA-Z0-9_]`                                                                                                                                                                                |
