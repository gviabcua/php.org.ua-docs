# ImagickDraw::getGravity

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getGravity — Повертає гравітацію тексту

### Опис

```methodsynopsis
public ImagickDraw::getGravity(): int
```

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

Повертає гравітацію розміщення тексту, що використовується під час його інструкції.

### Значення, що повертаються

Повертає константу [GRAVITY](imagick.constants.md#imagick.constants.gravity) `imagick::GRAVITY_*`) у разі успішного виконання або 0, якщо гравітація не встановлена.
