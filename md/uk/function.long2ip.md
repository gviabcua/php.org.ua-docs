# long2ip

(PHP 4, PHP 5, PHP 7, PHP 8)

long2ip - Конвертує ціле число в IPv4-адресу

### Опис

```methodsynopsis
long2ip(int $ip): string|false
```

Функция**long2ip()** генерує інтернет-адреси у форматі розділених точками чисел (тобто aaa.bbb.ccc.ddd) з довгого цілісного представлення.

### Список параметрів

`ip`

Правильне подання адреси як цілого числа.

### Значення, що повертаються

Повертає IP-адресу у вигляді рядка або \*\*`false`\*\*в случае возникновения ошибки.

### список змін

| Версия | Опис |
| --- | --- |
| 7.1.0 | Тип параметра `ip` змінений з рядкового (string) на цілий (int). |

### Примітки

> **Зауваження** :
> 
> На 32-бітових архітектурах приведення цілих уявлень IP-адрес з рядка (string) в ціле число (int) не дасть правильних результатів для чисел, що перевищують значення константи **`PHP_INT_MAX`**
