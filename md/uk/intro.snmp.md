# Вступ

Модуль SNMP надає простий та зручний набір інструментів для керування віддаленими пристроями через Simple Network Management Protocol (простий протокол керування мережами).

Так як цей модуль є обгорткою над бібліотекою Net-SNMP, всі основні концепції ідентичні і функції PHP змінюють їхню поведінку залежно від конфігурації Net-SNMP та змінних середовища.

Більше інформації про Net-SNMP можна знайти за адресою [» http://www.net-snmp.org/](http://www.net-snmp.org/)
