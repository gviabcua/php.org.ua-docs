# ReflectionFunctionAbstract::isStatic

(PHP 5, PHP 7, PHP 8)

ReflectionFunctionAbstract::isStatic — Перевіряє, чи є статична функція

### Опис

```methodsynopsis
public ReflectionFunctionAbstract::isStatic(): bool
```

Перевіряє, чи є статична функція.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Метод возвращает\*\*`true`\*\*, якщо функція є статичною, інакше повертає **`false`**
