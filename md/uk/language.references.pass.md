## Передача за посиланням

Ви можете передати змінну в функцію посилання, щоб вона могла змінювати значення аргументу. Синтаксис виглядає так:

```php
<?php
function foo(&$var) {
    $var++;
}

$a = 5;
foo($a);
// $a здесь равно 6
?>
```

> **Зауваження**: У виклику функції відсутній знак посилання - він є лише у визначенні функції. Цього достатньо для коректної передачі аргументів на засланні.

За посиланням можна передавати:

-   Змінні, наприклад `foo($a)`
    
-   Посилання, що повертаються функцією, наприклад:
    
    ```php
    <?php
    function foo(&$var) {
        $var++;
    }
    function &bar() {
        $a = 5;
        return $a;
    }
    foo(bar());
    ?>
    ```
    
    Дивіться також пояснення [возврата по ссылке](language.references.return.html)
    

Будь-який інший вираз не повинен передаватися за посиланням, оскільки результат не визначений. Наприклад, наступна передача за посиланням є неправильною:

```php
<?php
function foo(&$var) {
    $var++;
}
function bar() { // Операция & отсутствует
    $a = 5;
    return $a;
}
foo(bar()); // Вызывает предупреждение

foo($a = 5); // Выражение, а не переменная
foo(5); // Константа, а не переменная

class Foobar
{
}

foo(new Foobar()) // Вызывает уведомление с PHP 7.0.7
                  // Notice: Only variables should be passed by reference
?>
```
