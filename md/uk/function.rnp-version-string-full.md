# rnp\_version\_string\_full

(PECL rnp >= 0.1.1)

rnp\_version\_string\_full — Повертає рядок повної версії бібліотеки RNP

### Опис

```methodsynopsis
rnp_version_string_full(): string
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає рядок повної версії бібліотеки RNP.
