# GearmanTask::dataSize

(PECL gearman >= 0.5.0)

GearmanTask::dataSize — Отримати розмір даних, що повертаються

### Опис

```methodsynopsis
public GearmanTask::dataSize(): int|false
```

Повертає розмір даних завдання, що їх повернув обробник.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Розмір даних або **`false`**, якщо даних не надано.
