# Вступ

Ці функції є основою доступу до баз даних типу Berkeley DB.

Це основний рівень абстракції для файлових баз даних. Таким чином, функціональність обмежена загальним підмножиною функцій, що підтримуються сучасними базами даних такими як [» Oracle Berkeley DB](https://www.oracle.com/database/berkeley-db/db.md)
