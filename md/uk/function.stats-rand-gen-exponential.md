# statsrandгенexponential

(PECL stats >= 1.0.0)

statsrandгенexponential - Обчислює випадкове відхилення від експоненційного розподілу

### Опис

```methodsynopsis
stats_rand_gen_exponential(float $av): float
```

Обчислює випадкове відхилення від експоненційного розподілу із заданим масштабним коефіцієнтом `av`

### Список параметрів

`av`

Масштабний коефіцієнт

### Значення, що повертаються

Випадкове відхилення
