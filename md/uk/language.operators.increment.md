## Оператори інкременту та декременту

PHP підтримує префіксні та постфіксні оператори інкременту та декременту в стилі C.

> **Зауваження**: Оператори інкременту/декременту впливають тільки на числа та рядки. Масиви, об'єкти, булеві значення та ресурси не будуть змінені. Декремент **`null`** також не дасть жодного ефекту, проте інкремент дасть значення `1`

**Оператори інкременту та декременту**

| Пример | Название              | Действие                                              |
|--------|-----------------------|-------------------------------------------------------|
| ++$a   | Префіксний інкремент  | Збільшує $a на одиницю, а потім повертає значення $a. |
| $a++   | Постфіксний інкремент | Повертає значення $a, а потім збільшує $a на одиницю. |
| \-$a   | Префіксний декремент  | Зменшує $a на одиницю, а потім повертає значення $a.  |
| $a--   | Постфіксний декремент | Повертає значення $a, а потім зменшує $a на одиницю.  |

Наведемо приклад простого скрипту:

```php
<?php
echo "<h3>Постфиксный инкремент</h3>";
$a = 5;
echo "Должно быть 5: " . $a++ . "<br />\n";
echo "Должно быть 6: " . $a . "<br />\n";

echo "<h3>Префиксный инкремент</h3>";
$a = 5;
echo "Должно быть 6: " . ++$a . "<br />\n";
echo "Должно быть 6: " . $a . "<br />\n";

echo "<h3>Постфиксный декремент</h3>";
$a = 5;
echo "Должно быть 5: " . $a-- . "<br />\n";
echo "Должно быть 4: " . $a . "<br />\n";

echo "<h3>Префиксный декремент</h3>";
$a = 5;
echo "Должно быть 4: " . --$a . "<br />\n";
echo "Должно быть 4: " . $a . "<br />\n";
?>
```

PHP слідує угодам Perl (на відміну від С) щодо виконання арифметичних операцій із символьними змінними. Наприклад, у PHP та Perl `$a = 'Z'; $a++;` присвоїть `$a` значення `'AA'`, у той час як у C `a = 'Z'; a++;` присвоїть `a` значення `'['` (ASCII-значення `'Z'` дорівнює 90, а ASCII-значення `'['` одно 91). Слід врахувати, що до символьних змінних можна застосовувати операцію інкременту, тоді як операцію декременту застосовувати не можна, крім того, підтримуються лише символи ASCII (a-z і A-Z). Спроба інкременту/декременту інших символьних змінних не матиме жодного ефекту, вихідний рядок залишиться незмінним.

**Приклад #1 Арифметичні операції із символьними змінними**

```php
<?php
echo '== Буквы ==' . PHP_EOL;
$s = 'W';
for ($n=0; $n<6; $n++) {
    echo ++$s . PHP_EOL;
}
// С цифрами несколько по другому
echo '== Цифры ==' . PHP_EOL;
$d = 'A8';
for ($n=0; $n<6; $n++) {
    echo ++$d . PHP_EOL;
}
$d = 'A08';
for ($n=0; $n<6; $n++) {
    echo ++$d . PHP_EOL;
}
?>
```

Результат виконання цього прикладу:

```
== Буквы ==
X
Y
Z
AA
AB
AC
== Цифры ==
A9
B0
B1
B2
B3
B4
A09
A10
A11
A12
A13
A14
```

Інкрементування або декрементування булевих змінних не призводить до жодного результату.
