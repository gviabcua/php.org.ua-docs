# ImagickPixelIterator::getIteratorRow

(PECL imagick 2, PECL imagick 3)

ImagickPixelIterator::getIteratorRow - Повертає поточний піксель ітератора ряду

### Опис

```methodsynopsis
public ImagickPixelIterator::getIteratorRow(): int
```

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

Повертає поточний піксель ряду ітератора.

### Значення, що повертаються

Повертає зсув ряду у вигляді числа, у разі виникнення помилки викидає виняток ImagickPixelIteratorException.
