# Вступ

Модуль xdiff дозволяє створювати та застосовувати файли виправлень, що містять різницю між різними версіями файлів.

Модуль підтримує два режими роботи: з рядками та з файлами, також два різних формати патчів - уніфікований і бінарний. Уніфіковані патчі хороші для текстових файлів та зручні для читання людиною. Для бінарних файлів, таких як архіви або зображення, слід використовувати бінарний формат, оскільки він коректно обробляє недруковані символи.

Починаючи з версії 1.5.0, існують два різні набори функцій для створення бінарних патчів. Нові функції - [xdiffstringrabdiff()](function.xdiff-string-rabdiff.html) і [xdifffilerabdiff()](function.xdiff-file-rabdiff.html) створюють висновок, що підтримується старими функціями, але працюють швидше і результат займає менше місця. Детальніше про методи створення бінарних патчів та різницю між ними читайте на сайті [» libxdiff](http://www.xmailserver.org/xdiff-lib.md)
