# cos

(PHP 4, PHP 5, PHP 7, PHP 8)

cos — Косінус

### Опис

```methodsynopsis
cos(float $num): float
```

**cos()** повертає косинус параметра `num`. Параметр `num` задається у радіанах.

### Список параметрів

`num`

Кут у радіанах

### Значення, що повертаються

Косинус кута `num`

### Приклади

**Приклад #1 Приклад використання **cos()****

```php
<?php

echo cos(M_PI); // -1

?>
```
