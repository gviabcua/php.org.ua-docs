# tradermult

(PECL trader >= 0.2.0)

tradermult — Арифметичний вектор твору

### Опис

```methodsynopsis
trader_mult(array $real0, array $real1): array
```

Обчислює твір векторної точки для `real0` за допомогою `real1` та повертає отриманий вектор.

### Список параметрів

`real0`

Масив, що містить реальні значення.

`real1`

Масив, що містить реальні значення.

### Значення, що повертаються

Повертає масив з обчисленими даними або false у разі виникнення помилки.
