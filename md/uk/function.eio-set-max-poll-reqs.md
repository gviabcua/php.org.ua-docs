# eiosetmaxpollreqs

(PECL eio >= 0.0.1dev)

eiosetmaxpollreqs — Встановлює максимальну кількість запитів, що обробляються.

### Опис

```methodsynopsis
eio_set_max_poll_reqs(int $nreqs): void
```

### Список параметрів

`nreqs`

Кількість запитів

### Значення, що повертаються

Функція не повертає значення після виконання.
