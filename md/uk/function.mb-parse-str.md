# мбparsestr

(PHP 4> = 4.0.6, PHP 5, PHP 7, PHP 8)

мбparsestr — Розбір даних запитів GET/POST/COOKIE та встановлення значень глобальних змінних

### Опис

```methodsynopsis
mb_parse_str(string $string, array &$result): bool
```

Розбирає запити GET/POST/COOKIE та встановлює значення глобальних змінних. Оскільки PHP не працює з вихідними даними POST/COOKIE, на даний момент його можна використовувати лише для роботи з даними запиту GET. Функція розбирає рядок URL, визначає кодування, перетворює символи у внутрішнє кодування скрипта та заносить значення змінних на масив `result` або у глобальні змінні.

### Список параметрів

`string`

Кодовані дані в URL.

`result`

Масив (array), що містить декодовані дані та символи у внутрішньому кодуванні скрипта.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.
