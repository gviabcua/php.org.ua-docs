# mb\_parse\_str

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

mb\_parse\_str — Розбирає дані запитів GET/POST/COOKIE та встановлює значення глобальних змінних

### Опис

```methodsynopsis
mb_parse_str(string $string, array &$result): bool
```

Розбирає запити GET/POST/COOKIE та встановлює значення глобальних змінних. Оскільки PHP не працює з вихідними даними POST/COOKIE, її можна використовувати лише для роботи із даними GET-запиту. Функція розбирає URL-кодовані дані, визначає кодування, перетворює символи у внутрішнє кодування скрипту та заносить значення змінних у масив `result` або у глобальні змінні.

### Список параметрів

`string`

URL-кодовані дані.

`result`

Масив (array), що містить декодовані та перетворені у внутрішнє кодування значення.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або \*\*`false`\*\*в случае возникновения ошибки.
