# ps\_open\_memory\_image

(PECL ps >= 1.1.0)

ps\_open\_memory\_image — Бере зображення GD та повертає зображення для розміщення в документі PS

### Опис

```methodsynopsis
ps_open_memory_image(resource $psdoc, int $gd): int
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

### Список параметрів

`psdoc`

Ідентифікатор ресурсу файлу postscript, повернутий [ps\_new()](function.ps-new.md)

`gd`
