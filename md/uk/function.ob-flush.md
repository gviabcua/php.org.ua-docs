# ob\_flush

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

ob\_flush — Скидає (відправляє) повернене активним обробником виводу значення

### Опис

```methodsynopsis
ob_flush(): bool
```

Функція викликає обробник виводу (з прапором the **`PHP_OUTPUT_HANDLER_FLUSH`**), скидає (відправляє) повернене їм значення та видаляє вміст активного буфера виведення.

Функція не відключає активний буфер виводу, як це роблять функції [ob\_end\_flush()](function.ob-end-flush.md) або [ob\_get\_flush()](function.ob-get-flush.md)

Функция**ob\_flush()** завершиться невдало, якщо активний буфер виведення було запущено без прапора **`PHP_OUTPUT_HANDLER_FLUSHABLE`**

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або \*\*`false`\*\*в случае возникновения ошибки.

### Помилки

Якщо функція завершується невдало, вона видає помилку рівня **`E_NOTICE`**

-   [ob\_start()](function.ob-start.md) \- Включає буферизацію виводу
-   [ob\_get\_contents()](function.ob-get-contents.md) \- Повертає вміст буфера виводу
-   [ob\_end\_flush()](function.ob-end-flush.md) \- Скидає (відправляє) значення активного оброблювача виводу, що повертається, і відключає активний буфер виводу
-   [ob\_get\_flush()](function.ob-get-flush.md) \- Скидає (відправляє) повернене активним обробником виводу значення, повертає вміст активного буфера виводу та відключає його
-   [ob\_clean()](function.ob-clean.md) \- Очищає (стирає) вміст активного буфера виводу
