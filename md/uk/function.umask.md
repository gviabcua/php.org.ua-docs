# umask

(PHP 4, PHP 5, PHP 7, PHP 8)

umask — Змінює поточну umask

### Опис

```methodsynopsis
umask(?int $mask = null): int
```

**umask()** встановлює PHP umask на значення `mask` & 0777 та повертає стару umask. Якщо PHP використовується як серверний модуль, umask буде відновлюватися після закінчення кожного запиту.

### Список параметрів

`mask`

Новий umask.

### Значення, що повертаються

Якщо параметр `mask` дорівнює **`null`**, функція **umask()** просто повертає поточну umask, інакше повертається стара umask.

### список змін

| Версия | Описание |
| --- | --- |
|  | Параметр `mask` тепер припускає значення **`null`** |

### Приклади

**Приклад #1 Приклад використання **umask()****

```php
<?php
$old = umask(0);
chmod("/path/some_dir/some_file.txt", 0755);
umask($old);

// Проверка
if ($old != umask()) {
    die('При восстановлении umask произошла ошибка');
}
?>
```

### Примітки

> **Зауваження**
> 
> Уникайте використання цієї функції у багатопотокових веб-серверах. Найкраще змінити права файлу за допомогою функції [chmod()](function.chmod.html) після його створення. Використання **umask()** може призвести до несподіваної поведінки одночасно працюючих скриптів і веб-сервера, т.к. вони всі будуть використовувати одну і ту ж umask.
