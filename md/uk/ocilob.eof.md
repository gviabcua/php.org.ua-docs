# OCILob::eof

(PHP 5, PHP 7, PHP 8, PECL OCI8> = 1.1.0)

OCILob::eof — Перевіряє, чи вказівник LOB знаходиться на кінці об'єкта.

### Опис

```methodsynopsis
public OCILob::eof(): bool
```

Повідомляє про досягнення внутрішнім покажчиком кінця об'єкта LOB.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає \*\*`true`\*\*якщо внутрішній покажчик об'єкта знаходиться на кінці LOB. Інакше повертає **`false`**

### список змін

| Версия                 | Описание                                                                                                |
|------------------------|---------------------------------------------------------------------------------------------------------|
| 8.0.0, PECL OCI8 3.0.0 | Клас **OCI-Lob** перейменований на [OCILob](class.ocilob.html) відповідно до стандартів іменування PHP. |

### Примітки

> **Зауваження**
> 
> Функція повертає Oracle помилку у разі, якщо [OCILob::setBuffering](ocilob.setbuffering.html) включено для LOB.
