# FilterIterator::current

(PHP 5> = 5.1.0, PHP 7, PHP 8)

FilterIterator::current — Отримує значення поточного елемента

### Опис

```methodsynopsis
public FilterIterator::current(): mixed
```

**Увага**

Отримує значення поточного елемента.
