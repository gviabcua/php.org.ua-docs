# Ev::sleep

(PECL ev >= 0.2.0)

Ev::sleep — Блокувати процес задану кількість секунд

### Опис

```methodsynopsis
final
   public
   static
   Ev::sleep(
    float
     $seconds
   ): void
```

Блокувати процес задану кількість секунд

### Список параметрів

`seconds`

Дробна кількість секунд

### Значення, що повертаються

Функція не повертає значення після виконання.
