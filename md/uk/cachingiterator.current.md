# CachingIterator::current

(PHP 5, PHP 7, PHP 8)

CachingIterator::current — Повертає поточний елемент

### Опис

```methodsynopsis
public CachingIterator::current(): mixed
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.
