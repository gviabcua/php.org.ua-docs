# rnp\_backend\_string

(PECL rnp >= 0.1.1)

rnp\_backend\_string — Повертає ім'я бібліотеки криптографічного бекенду

### Опис

```methodsynopsis
rnp_backend_string(): string
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Рядок імені бекенда. В даний час підтримуються такі: "Botan" та "OpenSSL".
