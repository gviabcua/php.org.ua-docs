# ImagickDraw::getClipRule

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getClipRule — Повертає поточне правило заливки багатокутника

### Опис

```methodsynopsis
public ImagickDraw::getClipRule(): int
```

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

Повертає поточне правило заливки багатокутника, яке використовуватиметься відсічний контур.

### Значення, що повертаються

Повертає одну з констант [FILLRULE](imagick.constants.html#imagick.constants.fillrule) `imagick::FILLRULE_*`
