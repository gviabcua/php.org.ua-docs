## Установка

Підтримка цих функцій за замовчуванням вимкнена. Для включення підтримки семафорів System V зберіть PHP з опцією **\--enable-sysvsem**Для включения поддержки разделяемой памяти System V соберите PHP с опцией**\--enable-sysvshm**Для включения поддержки сообщений System V соберите PHP с опцией**\--enable-sysvmsg**
