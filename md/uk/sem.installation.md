## Встановлення

Підтримка цих функцій за замовчуванням вимкнена. Для включення підтримки семафорів System V зберіть PHP з опцією **\-enable-sysvsem**. Для включення підтримки пам'яті System V, що розділяється, зберіть PHP з опцією **\-enable-sysvshm**. Щоб увімкнути підтримку повідомлень System V, зберіть PHP з опцією **\-enable-sysvmsg**
