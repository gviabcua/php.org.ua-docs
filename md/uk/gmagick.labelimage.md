# Gmagick::labelimage

(PECL gmagick >= Unknown)

Gmagick::labelimage — Додає позначку до зображення

### Опис

```methodsynopsis
public Gmagick::labelimage(string $label): mixed
```

Додає позначку до зображення.

### Список параметрів

`label`

Мітка, яку потрібно додати
