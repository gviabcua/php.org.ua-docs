# usleep

(PHP 4, PHP 5, PHP 7, PHP 8)

usleep — Затримка виконання у мікросекундах

### Опис

```methodsynopsis
usleep(int $microseconds): void
```

Відкладає виконання програми на вказану кількість мікросекунд.

### Список параметрів

`microseconds`

Час виконання, що відкладається в мікросекундах. Мікросекунда – це одна мільйонна секунди.

> **Зауваження**: Значення більше `1000000` (тобто очікування більше секунди) можуть не підтримуватись операційною системою. Натомість використовуйте [sleep()](function.sleep.md)

> **Зауваження**: Відкладене виконання може бути трохи довшим (тобто може бути довшим, ніж параметр `microseconds`) за рахунок будь-якої активності системи або часу, витраченого на обробку виклику або гранулярністю системних таймерів.

### Значення, що повертаються

Функція не повертає значення після виконання.

### Приклади

**Пример #1 Пример использования**usleep()\*\*\*\*

```php
<?php

// Текущее время
echo (new DateTime('now'))->format('H:i:s.v'), "\n";

// ждать 2 миллисекунды
usleep(2000);

// вернуться к исполнению
echo (new DateTime('now'))->format('H:i:s.v'), "\n";

// ждать 30 миллисекунд
usleep(30000);

// вернуться к исполнению ещё раз!
echo (new DateTime('now'))->format('H:i:s.v'), "\n";

?>
```

Результат виконання наведеного прикладу:

```
11:13:28.005
11:13:28.007
11:13:28.037
```
