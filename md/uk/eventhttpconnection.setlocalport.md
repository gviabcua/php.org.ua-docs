# EventHttpConnection::setLocalPort

(PECL event >= 1.2.6-beta)

EventHttpConnection::setLocalPort — Встановлює локальний порт, з якого з'являються з'єднання.

### Опис

```methodsynopsis
public
   EventHttpConnection::setLocalPort(
    int
     $port
   ): void
```

Встановлює локальний порт, з якого виконуються з'єднання.

### Список параметрів

`port`

Номер порту.
