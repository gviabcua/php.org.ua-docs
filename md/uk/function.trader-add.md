# trader\_add

(PECL trader >= 0.2.0)

trader\_add — Арифметична додавання вектора

### Опис

```methodsynopsis
trader_add(array $real0, array $real1): array
```

Обчислює векторне додавання `real0`к`real1` та повертає отриманий вектор.

### Список параметрів

`real0`

Масив, що містить реальні значення.

`real1`

Масив, що містить реальні значення.

### Значення, що повертаються

Повертає масив з обчисленими даними або false у разі виникнення помилки.
