# Yac::get

(PECL yac >= 1.0.0)

Yac::get - Геттер

### Опис

```methodsynopsis
public Yac::__get(string $key): mixed
```

Витягує значення з кешу

### Список параметрів

`key`

Ключ (string)

### Значення, що повертаються

Змішане значення у разі успішного виконання, **`null`** у разі виникнення помилки
