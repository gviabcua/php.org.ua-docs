# statsrandphraseтоseeds

(PECL stats >= 1.0.0)

statsrandphraseтоseeds - Генерує два початкові числа для генератора випадкових чисел RGN

### Опис

```methodsynopsis
stats_rand_phrase_to_seeds(string $phrase): array
```

Генерує два початкові числа для генератора випадкових чисел з `phrase`

### Список параметрів

`phrase`

Вхідна фраза

### Значення, що повертаються

Повертає масив із двох цілих чисел.
