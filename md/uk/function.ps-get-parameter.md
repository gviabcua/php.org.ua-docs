# ps\_get\_parameter

(PECL ps >= 1.1.0)

ps\_get\_parameter — Отримує певні параметри

### Опис

```methodsynopsis
ps_get_parameter(resource $psdoc, string $name, float $modifier = ?): string|false
```

Отримує кілька параметрів, які були прямо встановлені [ps\_set\_parameter()](function.ps-set-parameter.md) або опосередковано однією з інших функцій. Параметри визначення є строковими значеннями. Функцію не можна використовувати для отримання ресурсів, які також були встановлені за допомогою [ps\_set\_parameter()](function.ps-set-parameter.md)

Параметр`name` може мати такі значення:

`fontname`

Ім'я поточного активного шрифту або шрифту, ідентифікатор якого передається у параметрі `modifier`

`fontencoding`

Кодування активного шрифту.

`dottedversion`

Версія базової бібліотеки pslib у форматі

`scope`

Поточний обсяг креслення. Можливо: object, document, null, page, pattern, path, template, prolog, font, glyph.

`ligaturedisolvechar`

Символ, що розчиняє лігатуру. Якщо ви використовуєте шрифт, що містить лігатури \`ff' і символ для розчинення лігатури \`|', тоді \`f|f' призведе до двох \`f' замість лігатури \`ff'.

`imageencoding`

Кодування, яке використовується для кодування зображень. Може мати значення `hex`или`85`. Шістнадцяткове кодування використовує два байти у файлі postscript на кожен байт зображення. 85 – це кодування Ascii85.

`linenumbermode`

Установите значение`paragraph`якщо рядки пронумеровані всередині абзацу або `box`якщо вони пронумеровані в межах навколишнього поля.

`linebreak`

Використовується лише якщо текст виводиться за допомогою [ps\_show\_boxed()](function.ps-show-boxed.md)Если установлено значение\*\*`true`\*\*, повернення каретки додасть розрив рядка.

`parbreak`

Використовується лише якщо текст виводиться за допомогою [ps\_show\_boxed()](function.ps-show-boxed.md)Если установлено значение\*\*`true`\*\*, повернення каретки розпочне новий абзац.

`hyphenation`

Використовується лише якщо текст виводиться за допомогою [ps\_show\_boxed()](function.ps-show-boxed.md)Если установлено значение\*\*`true`\*\*, абзац буде розставлено через дефіс, якщо дефісний словник встановлено та існує.

`hyphendict`

Ім'я файлу словника, який використовується для розміщення переносів.

### Список параметрів

`psdoc`

Ідентифікатор ресурсу файлу postscript, повернутий функцією [ps\_new()](function.ps-new.md)

`name`

Назва параметра.

`modifier`

Ідентифікатор, необхідний, якщо запитує параметр ресурсу, наприклад, розмір зображення. У такому разі передається ідентифікатор ресурсу.

### Значення, що повертаються

Возвращает значение параметра или\*\*`false`\*\*в случае возникновения ошибки.
