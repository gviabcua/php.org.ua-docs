# traderacos

(PECL trader >= 0.2.0)

traderacos — Вектор тригонометричний арккосинус

### Опис

```methodsynopsis
trader_acos(array $real): array
```

Обчислює арккосинус дуги для кожного значення `real` та повертає результуючий масив.

### Список параметрів

`real`

Масив, що містить реальні значення.

### Значення, що повертаються

Повертає масив з обчисленими даними або false у разі виникнення помилки.
