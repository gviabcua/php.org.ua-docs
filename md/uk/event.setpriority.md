# Event::setPriority

(PECL event >= 1.2.6-beta)

Event::setPriority — Задати пріоритет події

### Опис

```methodsynopsis
public
   Event::setPriority(
    int
     $priority
   ): bool
```

Встановлює пріоритет події.

### Список параметрів

`priority`

Пріоритет.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.
