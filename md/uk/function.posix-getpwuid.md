# posixgetpwuid

(PHP 4, PHP 5, PHP 7, PHP 8)

posixgetpwuid — Повертає інформацію про користувача, використовуючи його ID

### Опис

```methodsynopsis
posix_getpwuid(int $user_id): array|false
```

Повертає array з інформацією про користувача за допомогою ідентифікатора користувача.

### Список параметрів

`user_id`

Ідентифікатор користувача.

### Значення, що повертаються

Повертає асоціативний масив із такими елементами:

**Масив з інформацією про користувача**

| Элемент | Описание |
| --- | --- |
| name | Елемент name містить ім'я користувача. Це короткий, зазвичай менше 16 символів "дескриптор" користувача, що не є фактичним повним ім'ям користувача. |
| passwd | Елемент passwd містить пароль групи у зашифрованому вигляді. Часто, наприклад, у системах, що використовують "shadow" файли для зберігання інформації про паролі, це поле містить зірочку. |
| uid | Числовий ідентифікатор користувача повинен збігатися з параметром `user_id`, переданому під час виклику функції. Є надлишковим. |
| gid | Ідентифікатор групи користувача. Використовуйте функцію [posix\_getgrgid()](function.posix-getgrgid.html) для отримання імені групи та списку користувачів- її членів. |
| gecos | GECOS є застарілим членом, який вказує на інформаційне поле у ​​системі пакетної обробки Honeywell. Однак це поле існує, і його вміст був формалізований у стандарті POSIX. Це поле містить список, розділений комами, і містить повне ім'я користувача, робочий телефон, номер офісу та домашній телефон. У більшості систем доступне лише повне ім'я користувача. |
| dir | Цей елемент містить абсолютний шлях до домашньої директорії користувача. |
| shell | Елемент shell містить абсолютний шлях до виконуваного файлу командної оболонки користувача за промовчанням. |

Функція повертає **`false`** у разі виникнення помилки.

### Приклади

**Приклад #1 Приклад використання **posixgetpwuid()****

```php
<?php

$userinfo = posix_getpwuid(10000);

print_r($userinfo);
?>
```

Результатом виконання цього прикладу буде щось подібне:

```
Array
(
    [name]    => tom
    [passwd]  => x
    [uid]     => 10000
    [gid]     => 42
    [gecos]   => "tom,,,"
    [dir]     => "/home/tom"
    [shell]   => "/bin/bash"
)
```
