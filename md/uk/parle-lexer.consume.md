# ParleLexer::consume

(PECL parle >= 0.5.1)

ParleLexer::consume — Передає дані на обробку

### Опис

```methodsynopsis
public Parle\Lexer::consume(string $data): void
```

Обробляє дані для лексування.

### Список параметрів

`data`

Дані для лексування.

### Значення, що повертаються

Функція не повертає значення після виконання.
