# filectime

(PHP 4, PHP 5, PHP 7, PHP 8)

filectime — Повертає час зміни індексного дескриптора файлу

### Опис

```methodsynopsis
filectime(string $filename): int|false
```

Повертає час зміни індексного дескриптора (inode) файлу.

### Список параметрів

`filename`

Шлях до файлу.

### Значення, що повертаються

Возвращает время последнего изменения файла или\*\*`false`\*\* у разі виникнення помилки. Час повертається у форматі тимчасової мітки Unix.

### Помилки

У разі невдалого завершення роботи генерується помилка рівня **`E_WARNING`**

### Приклади

**Приклад #1 Приклад використання функції** filectime()\*\*\*\*

```php
<?php

// Приклад вывода: Файл somefile.txt в последний раз был изменён: December 29 2002 22:16:23.

$filename = 'somefile.txt';
if (file_exists($filename)) {
    echo "Файл $filename в последний раз был изменён: " . date("F d Y H:i:s.", filectime($filename));
}

?>
```

### Примітки

> **Зауваження** :
> 
> Примітка. На більшості платформ Unix файл вважається зміненим, якщо змінені дані його індексного дескриптора, що включає інформацію про права на файл, про його власника, групу та інші метадані, що містяться в індексному дескрипторі. Зверніться також до описів функцій [filemtime()](function.filemtime.md) (Ця функція корисна для створення повідомлень типу: "Останнє оновлення від ..." на веб-сторінках) та [fileatime()](function.fileatime.md)

> **Зауваження** :
> 
> Зверніть увагу також, що в деяких описах роботи Unix ctime представляється як час створення файлу. Це не вірно. У більшості файлових систем Unix немає часу створення файлу.

> **Зауваження** :
> 
> Зверніть увагу, що обробка часу може відрізнятися в різних файлових системах.

> **Зауваження**: Результати цієї функції кешуються Більш детальну інформацію дивіться у розділі [clearstatcache()](function.clearstatcache.md)

**Підказка**

Починаючи з PHP 5.0.0, ця функція також може бути використана з *деякими* обгортками url. Список обгорток, що підтримуються сімейством функцій [stat()](function.stat.md), смотрите в разделе[Підтримувані протоколи та обгортки](wrappers.md)
