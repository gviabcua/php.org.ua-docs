## php-config

**php-config** — це простий скрипт командного рядка для отримання інформації про конфігурацію PHP.

Коли модулі компілюються на системах з кількома встановленими версіями PHP, установка, для якої розпочато складання, може бути визначена під час конфігурації командою **php-config**с параметром**\--with-php-config**, в якому вказують шлях до потрібного сценарію.

Список доступних для скрипту **php-config** параметрів командного рядка можна отримати у будь-який час, запустивши команду **php-config**с параметром**\-h** :

```
Usage: /usr/local/bin/php-config [OPTION]
Options:
  --prefix            [...]
  --includes          [...]
  --ldflags           [...]
  --libs              [...]
  --extension-dir     [...]
  --include-dir       [...]
  --php-binary        [...]
  --php-sapis         [...]
  --configure-options [...]
  --version           [...]
  --vernum            [...]
```

**Опції скрипту**

| Опция | Опис |
| --- | --- |
| \--prefix | Директорія, в якій встановлено PHP, наприклад /usr/local |
| \--includes | перелік `-I` параметрів з усіма файлами, що підключаються |
| \--ldflags | `LD`, з якими був скомпільований PHP |
| \--libs | Зовнішні бібліотеки, з якими було скомпільовано PHP |
| \--extension-dir | Директорія, де за замовчуванням шукаються модулі |
| \--include-dir | Префікс директорії, в яку за умовчанням встановлюються заголовні файли |
| \--php-binary | Повний шлях до файлів php CLI або CGI |
| \--php-sapis | Показує всі доступні модулі SAPI |
| \--configure-options | Конфігураційні опції для відтворення налаштувань поточної установки PHP. |
| \--version | Версія PHP |
| \--vernum | Версія PHP у вигляді цілого числа |
