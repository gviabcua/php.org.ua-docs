# HRTime\\StopWatch::getLastElapsedTicks

(PECL hrtime >= 0.4.3)

HRTime\\StopWatch::getLastElapsedTicks — Отримати кількість тиків, що пройшли, за останній інтервал

### Опис

```methodsynopsis
public HRTime\StopWatch::getLastElapsedTicks(): int
```

Повертає кількість тиків, що пройшли, за останній закритий інтервал.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає ціле число, що вказує на минулі тики.
