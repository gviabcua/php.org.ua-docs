### Опис

```methodsynopsis
trader_roc(array $real, int $timePeriod = ?): array
```

### Список параметрів

`real`

Масив, що містить реальні значення.

`timePeriod`

Номер періоду. Допустимі значення від 2 до 100000.

Повертає масив з обчисленими даними або false у разі виникнення помилки.
