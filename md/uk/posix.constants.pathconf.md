## Константи функції [posix\_pathconf()](function.posix-pathconf.md)

**`POSIX_PC_LINK_MAX`**(int)

Максимальна кількість посилань, які можуть містити вказаний файл або директорія. Доступна з PHP 8.3.0.

**`POSIX_PC_MAX_CANON`**(int)

Максимальний розмір вбудованого канонічного буфера термінала в байтах (тоді шлях стає символьним спеціальним файлом). Доступна з PHP 8.3.0.

**`POSIX_PC_MAX_INPUT`**(int)

Максимальний розмір черги введення терміналу в байтах (тоді шлях стає символьним спеціальним файлом). Доступна з PHP 8.3.0.

**`POSIX_PC_NAME_MAX`**(int)

Максимальна кількість символів тільки для імені файлу, не його шляху. Доступна з PHP 8.3.0.

**`POSIX_PC_PATH_MAX`**(int)

Максимальна кількість символів для повного імені шляху. Доступна з PHP 8.3.0.

**`POSIX_PC_PIPE_BUF`**(int)

Максимальна кількість байтів, які можуть бути записані до каналу за одну операцію. Доступна з PHP 8.3.0.

**`POSIX_PC_CHOWN_RESTRICTED`**(int)

Перевіряє, чи потрібні привілеї для роботи функції [chown()](function.chown.md)Доступна с PHP 8.3.0.

**`POSIX_PC_NO_TRUNC`**(int)

Перевіряє, чи не перевищує довжина імені файлу (або файлів усередині директорії) значення константи \*\*`POSIX_PC_NAME_MAX`\*\*Доступна с PHP 8.3.0.

**`POSIX_PC_ALLOC_SIZE_MIN`**(int)

Мінімальний розмір пам'яті в байтах, виділений для будь-якої частини файлу. Доступна з PHP 8.3.0.

**`POSIX_PC_ALLOC_SYMLINK_MAX`**(int)

Максимальна кількість символьних посилань, які можуть містити цей файл або директорія. Доступна з PHP 8.3.0.
