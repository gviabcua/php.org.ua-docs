# ImagickPixelIterator::getCurrentIteratorRow

(PECL imagick 2, PECL imagick 3)

ImagickPixelIterator::getCurrentIteratorRow — Повертає поточний ряд об'єкту ImagickPixel

### Опис

```methodsynopsis
public ImagickPixelIterator::getCurrentIteratorRow(): array
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

Повертає поточний ряд у вигляді масиву об'єктів ImagickPixel із піксельного ітератора.

### Значення, що повертаються

Повертає ряд у вигляді масиву об'єктів ImagickPixel, який сам може бути ітерований.
