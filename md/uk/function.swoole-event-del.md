# swooleeventdel

(PECL swoole >= 1.9.0)

swooleeventdel — Видалити всі callback функції сокету

### Опис

```methodsynopsis
swoole_event_del(int $fd): bool
```

### Список параметрів

`fd`

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.
