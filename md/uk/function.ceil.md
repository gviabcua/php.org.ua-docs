# ceil

(PHP 4, PHP 5, PHP 7, PHP 8)

ceil - Округлює дріб у велику сторону

### Опис

```methodsynopsis
ceil(int|float $num): float
```

Повертає найближче більше ціле від `num`

### Список параметрів

`num`

Значення, яке потрібно округлити у велику сторону

### Значення, що повертаються

Повертає найближче більше ціле від `num`. Тип значення, що повертається залишається float т.к. діапазон float більше int.

### список змін

| Версия | Описание                                                                        |
|--------|---------------------------------------------------------------------------------|
|        | `num` більше не приймає внутрішні об'єкти, що підтримують числове перетворення. |

### Приклади

**Приклад #1 Приклад використання **ceil()****

```php
<?php
echo ceil(4.3);    // 5
echo ceil(9.999);  // 10
echo ceil(-3.14);  // -3
?>
```
