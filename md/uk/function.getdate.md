# getdate

(PHP 4, PHP 5, PHP 7, PHP 8)

getdate — Повертає інформацію про дату/час

### Опис

```methodsynopsis
getdate(?int $timestamp = null): array
```

Повертає асоціативний масив (array), що містить інформацію про дату, подану міткою часу `timestamp` або поточним системним часом, якщо `timestamp` не було передано або **`null`**

### Список параметрів

`timestamp`

Необов'язковий параметр `timestamp` являє собою мітку часу типу int, за умовчанням рівну поточному локальному часу, якщо `timestamp` не вказано або **`null`**. Іншими словами, значення за замовчуванням дорівнює результату функції [time()](function.time.html)

### Значення, що повертаються

Повертає асоціативний масив (array) з інформацією про параметр `timestamp`, який містить такі елементи:

**Індекси асоціативного масиву, що повертається.**

| Индекс      | Описание                                                                                                                                                                                                  | Пример значения                                                       |
|-------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| `"seconds"` | Числове уявлення секунд                                                                                                                                                                                   | від `0` до `59`                                                       |
| `"minutes"` | Числове уявлення хвилин                                                                                                                                                                                   | від `0` до `59`                                                       |
| `"hours"`   | Числове уявлення годинників                                                                                                                                                                               | від `0` до `23`                                                       |
| `"mday"`    | Порядковий номер дня місяця                                                                                                                                                                               | від `1` до `31`                                                       |
| `"wday"`    | Порядковий номер дня тижня                                                                                                                                                                                | від `0` (неділя) до `6` (субота)                                      |
| `"mon"`     | Порядковий номер місяця                                                                                                                                                                                   | від `1` до `12`                                                       |
| `"year"`    | Номер року, 4 цифри                                                                                                                                                                                       | Приклади: `1999` `2003`                                               |
| `"yday"`    | Порядковий номер дня у році                                                                                                                                                                               | від `0` до `365`                                                      |
| `"weekday"` | Повне найменування дня тижня                                                                                                                                                                              | від `Sunday` до `Saturday`                                            |
| `"month"`   | Повне найменування місяця, наприклад January або March                                                                                                                                                    | від `January` до `December`                                           |
| `0`         | Кількість секунд, що пройшли з початку епохи Unix (The Unix Epoch), подібно до значення, що повертається функцією [time()](function.time.html) та використовуваному функцією [date()](function.date.html) | Залежить від платформи, як правило, від `-2147483648` до `2147483647` |

### Приклади

**Приклад #1 Приклад використання функції **getdate()****

```php
<?php
$today = getdate();
print_r($today);
?>
```

Результатом виконання цього прикладу буде щось подібне:

```
Array
(
    [seconds] => 40
    [minutes] => 58
    [hours]   => 21
    [mday]    => 17
    [wday]    => 2
    [mon]     => 6
    [year]    => 2003
    [yday]    => 167
    [weekday] => Tuesday
    [month]   => June
    [0]       => 1055901520
)
```
