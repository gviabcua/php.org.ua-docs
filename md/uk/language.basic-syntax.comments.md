## Коментарі

PHP підтримує коментарі у стилі 'C', 'C++' та оболонки Unix (стиль Perl). Наприклад:

```php
<?php
    echo "Это тест"; // Это однострочный комментарий в стиле C++
    /* Это многострочный комментарий
       ещё одна строка комментария */
    echo "Это ещё один тест";
    echo "Последний тест"; # Это комментарий в стиле оболочки Unix
?>
```

Однорядкові коментарі йдуть лише до кінця рядка або поточного блоку PHP-коду, залежно від того, що йде перед ними. Це означає, що HTML-код після `// ... ?>` або `# ... ?>` БУДЕ надруковано: ?> завершує режим PHP і повертає режим HTML, а `//` або `#` не можуть вплинути на це.

```php
<h1>Это <?php # echo "простой";?> пример</h1>
<p>Заголовок вверху выведет 'Это пример'.</p>
```

'C'-коментарі закінчуються при першій же виявленій послідовності `*/`. Переконайтеся, що ви не вкладаєте один одного 'C'-коментарі. Дуже легко припуститися цієї помилки при коментуванні великого блоку коду.

```php
<?php
 /*
    echo "Это тест"; /* Этот комментарий вызовет проблему */
 */
?>
```
