# statsстатиindependentт

(PECL stats >= 1.0.0)

statsстатиindependentt — Отримати значення t для двовибіркового t-критерію Стьюдента для незалежних вибірок

### Опис

```methodsynopsis
stats_stat_independent_t(array $arr1, array $arr2): float
```

Отримати значення t для двовибіркового t-критерію Стьюдента для незалежних вибірок `arr1` і `arr2`

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

### Список параметрів

`arr1`

Перший набір значень

`arr2`

Другий набір значень

### Значення, що повертаються

Повертає значення t або **`false`** у разі виникнення помилки.
