# stats\_stat\_independent\_t

(PECL stats >= 1.0.0)

stats\_stat\_independent\_t — Отримати значення t для двовибіркового t-критерію Стьюдента для незалежних вибірок

### Опис

```methodsynopsis
stats_stat_independent_t(array $arr1, array $arr2): float
```

Отримати значення t для двовибіркового t-критерію Стьюдента для незалежних вибірок `arr1`и`arr2`

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

### Список параметрів

`arr1`

Перший набір значень

`arr2`

Другий набір значень

### Значення, що повертаються

Повертає значення t або \*\*`false`\*\*в случае возникновения ошибки.
