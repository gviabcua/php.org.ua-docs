# apcuclearcache

(PECL apcu >= 4.0.0)

apcuclearcache — Очистити кеш APCu

### Опис

```methodsynopsis
apcu_clear_cache(): bool
```

Очищує кеш APCu.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Завжди повертає **`true`**
