# eio\_nready

(PECL eio >= 0.0.1dev)

eio\_nready — Повертає кількість ще не опрацьованих запитів

### Опис

```methodsynopsis
eio_nready(): int
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

**eio\_nready()** повертає кількість ще не опрацьованих запитів
