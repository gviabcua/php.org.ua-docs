# MultipleIterator::setFlags

(PHP 5> = 5.3.0, PHP 7, PHP 8)

MultipleIterator::setFlags — Встановлює прапори

### Опис

```methodsynopsis
public MultipleIterator::setFlags(int $flags): void
```

Встановлює прапори.

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

### Список параметрів

`flags`

Прапори для встановлення, згідно [Обумовленим константам](class.multipleiterator.html#multipleiterator.constants)

### Значення, що повертаються

Функція не повертає значення після виконання.
