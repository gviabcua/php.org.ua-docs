## Установка

Модуль увімкнено за замовчуванням. Він може бути вимкнений під час виконання опцією: **\--disable-session**

Щоб використовувати квотування пам'яті, що розділяється (mm) для зберігання даних сесії необхідно налаштувати PHP з опцією **\--with-mm\[=DIR\]**

Версія PHP для Windows має вбудовану підтримку цього модуля. Тобто для виклику цих функцій не потрібно завантаження додаткових модулів.

> **Зауваження** :
> 
> За замовчуванням усі дані, пов'язані з конкретною сесією, зберігатимуться у файлі в директорії, вказаній в опції session.save\_path файл конфігурації. Файл створюється кожної сесії (незалежно від наявності пов'язаних із нею даних). Це з тим, що сесія відкрита (створюється файл), але жодні дані ще записані. Слід звернути увагу, що така поведінка є побічним ефектом обмежень у роботі з файловою системою і цілком можливо, що користувач сесії (наприклад той, що використовує базу даних) не відстежує сесії, в яких немає даних.
