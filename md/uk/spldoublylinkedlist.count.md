# SplDoublyLinkedList::count

(PHP 5> = 5.3.0, PHP 7, PHP 8)

SplDoublyLinkedList::count — Підраховує кількість елементів у двозв'язному списку

### Опис

```methodsynopsis
public SplDoublyLinkedList::count(): int
```

### Список параметрів

Повертає кількість елементів у списку.
