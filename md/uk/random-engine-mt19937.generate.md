# Random\\Engine\\Mt19937::generate

(PHP 8 >= 8.2.0)

Random\\Engine\\Mt19937::generate — Створює 32 біти випадкової послідовності

### Опис

```methodsynopsis
public Random\Engine\Mt19937::generate(): string
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає рядок, що становить 32-бітове ціле число без знака в порядку little-endian.
