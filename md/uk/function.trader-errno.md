# tradererrno

(PECL trader >= 0.3.0)

tradererrno — Отримати код помилки

### Опис

```methodsynopsis
trader_errno(): int
```

Отримати код помилки останньої операції.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає код помилки, ідентифікований однією з [TRADERERR](trader.constants.html) констант.
