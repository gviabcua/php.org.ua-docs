## NULL

Тип null - це одиничний тип PHP, тобто він має тільки одне значення: **`null`**

Невизначені та видалені мовною конструкцією [unset()](function.unset.md) змінні перетворюються на значення **`null`**

### Синтаксис

Існує тільки одне значення типу null - реєстронезалежна константа **`null`**

```php
<?php
$var = NULL;
?>
```

### Приведение к\*\*`null`\*\*

**Увага**

Ця функціональність оголошена *застарілої* починаючи з PHP 7.2.0 і була *ВИДАЛЕНО* у версії PHP 8.0.0. Використовувати цю функцію не рекомендується.

Приведення змінної до null через вираз `(unset) $var` *не* видаляє змінну та її значення. Цей вираз лише повертає значення **`null`**
