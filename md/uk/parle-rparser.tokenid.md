# ParleRParser::tokenId

(PECL parle >= 0.7.0)

ParleRParser::tokenId — Отримує ідентифікатор токена

### Опис

```methodsynopsis
public Parle\RParser::tokenId(string $tok): int
```

Отримує ідентифікатор заданого токена.

### Список параметрів

`tok`

Ім'я токена, що використовується в [ParleRParser::token()](parle-rparser.token.html)

### Значення, що повертаються

Повертає ціле число (int), що є ідентифікатором токена.
