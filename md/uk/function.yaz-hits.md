# yaz\_hits

(PHP 4 >= 4.0.1, PECL yaz >= 0.9.0)

yaz\_hits — Повертає кількість збігів для останнього пошуку

### Опис

```methodsynopsis
yaz_hits(resource $id, array &$searchresult = ?): int
```

**yaz\_hits()** повертає кількість збігів останнього пошуку.

### Список параметрів

`id`

Ресурс підключення, що повертається [yaz\_connect()](function.yaz-connect.md)

`searchresult`

Масив результатів для детальної інформації про результати пошуку.

### Значення, що повертаються

Повертає кількість збігів для останнього пошуку або 0, якщо пошук не здійснювався.

Масив результатів пошуку (якщо надано) містить інформацію, що повертається сервером Z39.50 щодо відповіді на пошуковий запит у форматі SearchResult-1. Формат SearchResult-1 може використовуватися для отримання інформації про кількість збігів різних частин запиту (підзапиту). Зокрема, можна отримати кількість збігів окремих умов пошуку в запиті. Інформація для першого запиту знаходиться в $array\[ \], другого підзапиту - у $array\[ \]и т.д.

**searchresult members**

| Элемент | Опис |
| --- | --- |
| `id` | Підзапит ID2 (рядок) |
| `count` | Кількість результатів/збігів (ціле число) |
| `subquery.term` | Умова підзапиту (рядок) |
| `interpretation.term` | Інтерпретована умова підзапиту (рядок) |
| `recommendation.term` | Рекомендована умова підзапиту (рядок) |

> **Зауваження** :
> 
> Для роботи SearchResult потрібен PECL YAZ 1.0.5 або новіший та YAZ 2.1.9 або новіший.

> **Зауваження** :
> 
> Дуже небагато реалізації Z39.50 підтримують функцію SearchResult.
