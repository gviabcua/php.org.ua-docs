# SolrInputDocument::reset

(PECL solr> = 0.9.2)

SolrInputDocument::reset — Псевдонім SolrInputDocument::clear

### Опис

```methodsynopsis
public SolrInputDocument::reset(): bool
```

Псевдонім SolrInputDocument::clear

### Список параметрів
