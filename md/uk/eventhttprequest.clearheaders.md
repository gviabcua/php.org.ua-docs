# EventHttpRequest::clearHeaders

(PECL event >= 1.4.0-beta)

EventHttpRequest::clearHeaders — Видалення всіх вихідних заголовків зі списку заголовків запиту

### Опис

```methodsynopsis
public
   EventHttpRequest::clearHeaders(): void
```

Видаляє всі вихідні заголовки зі списку заголовків запиту.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Функція не повертає значення після виконання.
