# MultipleIterator::containsIterator

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

MultipleIterator::containsIterator — Перевіряє, чи приєднано ітератора

### Опис

```methodsynopsis
public MultipleIterator::containsIterator(Iterator $iterator): bool
```

Перевіряє, чи приєднаний ітератор, чи ні.

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

### Список параметрів

`iterator`

Ітератор для перевірки.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або \*\*`false`\*\*в случае возникновения ошибки.
