# ReflectionProperty::getModifiers

(PHP 5, PHP 7, PHP 8)

ReflectionProperty::getModifiers — Отримання модифікаторів властивостей класу

### Опис

```methodsynopsis
public ReflectionProperty::getModifiers(): int
```

Отримує список модифікаторів

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Числове уявлення модифікаторів. Фактичне значення цих модифікаторів описано у розділі [предопределённые константы](class.reflectionproperty.html#reflectionproperty.constants.modifiers)
