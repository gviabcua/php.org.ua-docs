# HRTime\\StopWatch::getElapsedTicks

(PECL hrtime >= 0.4.3)

HRTime\\StopWatch::getElapsedTicks — Отримати кількість тиків, що пройшли, для всіх інтервалів

### Опис

```methodsynopsis
public HRTime\StopWatch::getElapsedTicks(): int
```

Повертає кількість минулих тиків всіх раніше закритих інтервалів.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає ціле число (int), що вказує на минулі тики.
