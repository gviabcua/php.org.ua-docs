## Установка для PHP < 7.4

Модуль libxml включений за замовчуванням, але може бути вимкнений за допомогою директиви **\--disable-libxml**

Необов'язкова директива **\--with-libxml-dir** використовується для завдання розташування модуля `libxml` у системах, на яких PHP скомпільовано, якщо директива не використовується, пошук буде здійснюватись лише за стандартними шляхами. Процес `configure`производит поиск libxml (особенно,`xml2-config`) по коліях у такому порядку:

1.  Розташування (\[DIR\]), зазначене за допомогою директиви**\--with-libxml-dir** \[DIR\]=/bin/xml2-config)
    
2.  /usr/local/bin/xml2-config
    
3.  /usr/bin/xml2-config
    

Якщо `configure` не знайде xml2-config у директорії, заданій директивою **\--with-libxml-dir**, він продовжить пошук за стандартними шляхами.
