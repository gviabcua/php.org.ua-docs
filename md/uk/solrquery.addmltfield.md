# SolrQuery::addMltField

(PECL solr> = 0.9.2)

SolrQuery::addMltField — Встановлює поле для використання для подібності

### Опис

```methodsynopsis
public SolrQuery::addMltField(string $field): SolrQuery
```

Відповідає mlt.fl. Вказує, що поле має використовуватись для подібності.

### Список параметрів

`field`

Назва поля

### Значення, що повертаються

Повертає поточний об'єкт SolrQuery, якщо використовується значення, що повертається.
