# statskurtosis

(PECL stats >= 1.0.0)

statskurtosis - обчислює ексцес даних в масиві

### Опис

```methodsynopsis
stats_kurtosis(array $a): float
```

Обчислює ексцес даних у масиві `a`

### Список параметрів

`a`

Масив вихідних значень

### Значення, що повертаються

Повертає ексцес масиву значень `a` або **`false`**, якщо `a` порожній чи не є масивом.
