# rnp\_ffi\_destroy

(PECL rnp >= 0.1.1)

rnp\_ffi\_destroy — Знищує об'єкт верхнього рівня, який використовується для взаємодії з бібліотекою

### Опис

```methodsynopsis
rnp_ffi_destroy(RnpFFI $ffi): void
```

### Список параметрів

`ffi`

Об'єкт FFI, що повертається функцією rnp\_ffi\_create.
