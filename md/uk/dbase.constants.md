# Обумовлені константи

Цей модуль не визначає жодних константів.

**`DBASE_VERSION`**(string)

Версія модуль. (Доступно з dbase 7.0.0)

**`DBASE_RDONLY`**(int)

Відкрийте базу даних тільки для читання. Використовується з [dbase\_open()](function.dbase-open.md)(Доступно с dbase 7.0.0)

**`DBASE_RDWR`**(int)

Відкрити базу даних для читання та запису. Використовується з [dbase\_open()](function.dbase-open.md)(Доступно с 7.0.0)

**`DBASE_TYPE_DBASE`**(int)

Створити базу даних типу dBASE. Використовується з [dbase\_create()](function.dbase-create.md)(Доступно с 7.0.0)

**`DBASE_TYPE_FOXPRO`**(int)

Створити базу даних типу FoxPro. Використовується з [dbase\_create()](function.dbase-create.md)(Доступно с 7.0.0)
