# мбstrtoupper

(PHP 4> = 4.3.0, PHP 5, PHP 7, PHP 8)

мбstrtoupper — Приведення рядка до верхнього регістру

### Опис

```methodsynopsis
mb_strtoupper(string $string, ?string $encoding = null): string
```

Повертає рядок `string`, літерні символи в якій наведено верхній регістр.

### Список параметрів

`string`

Початковий рядок (string).

`encoding`

Параметр `encoding` є символьним кодуванням. Якщо він опущений або дорівнює **`null`**, замість нього буде використано значення внутрішнього кодування.

### Значення, що повертаються

`string`, літери у якій наведено до верхнього регістру.

### Приклади

**Приклад #1 Приклад використання **мбstrtoupper()****

```php
<?php
$str = "У Мэри Был Маленький Ягнёнок и Она Его Очень ЛЮБИЛА";
$str = mb_strtoupper($str);
echo $str; // Выведет У МЭРИ БЫЛ МАЛЕНЬКИЙ ЯГНЕНОК И ОНА ЕГО ОЧЕНЬ ЛЮБИЛА
?>
```

**Приклад #2 Приклад використання **мбstrtoupper()** з латинськими літерами**

```php
<?php
$str = "Τάχιστη αλώπηξ βαφής ψημένη γη, δρασκελίζει υπέρ νωθρού κυνός";
$str = mb_strtoupper($str, 'UTF-8');
echo $str; // Выведет ΤΆΧΙΣΤΗ ΑΛΏΠΗΞ ΒΑΦΉΣ ΨΗΜΈΝΗ ΓΗ, ΔΡΑΣΚΕΛΊΖΕΙ ΥΠΈΡ ΝΩΘΡΟΎ ΚΥΝΌΣ
?>
```

### Примітки

На відміну від [strtoupper()](function.strtoupper.md), що символ є буквою визначається на підставі властивостей символу Юнікоду. Таким чином, на поведінку функції не впливають регіональні налаштування системи, а також функція може перетворювати символи, що мають літерні особливості, такі як а-умляут (ä).

За додатковою інформацією про властивості Юнікоду звертайтесь до [» http://www.unicode.org/reports/tr21/](http://www.unicode.org/reports/tr21/)
