# SplHeap::top

(PHP 5> = 5.3.0, PHP 7, PHP 8)

SplHeap::top — Повертає вузол, що знаходиться на вершині купи

### Опис

```methodsynopsis
public SplHeap::top(): mixed
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Вузол, що знаходиться на вершині купи.

### Помилки

Якщо структура даних вузла виявиться порожньою, буде викинуто виняток [RuntimeException](class.runtimeexception.md)
