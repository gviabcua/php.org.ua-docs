# intltz\_create\_default

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlTimeZone::createDefault -- intltz\_create\_default — Створити нову копію часового поясу за промовчанням для поточного хоста

### Опис

Об'єктно-орієнтований стиль (метод):

```methodsynopsis
public static IntlTimeZone::createDefault(): IntlTimeZone
```

Процедурний стиль:

```methodsynopsis
intltz_create_default(): IntlTimeZone
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

### Список параметрів

Ця функція не має параметрів.
