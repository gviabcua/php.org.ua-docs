# SplDoublyLinkedList::top

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SplDoublyLinkedList::top — Отримує вузол, що знаходиться в кінці двозв'язкового списку

### Опис

```methodsynopsis
public SplDoublyLinkedList::top(): mixed
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Значення останнього вузла.

### Помилки

Викидає виняток [RuntimeException](class.runtimeexception.md)коли структура даних порожня.
