# Вступ

PHP підтримує функції прямого введення/виводу, як описано в стандарті Posix (секція 6), для виклику функцій введення/виводу на нижчому рівні, ніж потокові функції мови C ([fopen()](function.fopen.md) [fread()](function.fread.md),..). Ці функції можуть бути задіяні лише якщо необхідний прямий контроль пристроїв. В інших випадках більш адекватно буде використовувати стандартні функції [файлової системи](book.filesystem.md)
