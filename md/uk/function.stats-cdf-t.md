# stats\_cdf\_t

(PECL stats >= 1.0.0)

stats\_cdf\_t — Обчислює один із параметрів розподілу Стьюдента за рештою

### Опис

```methodsynopsis
stats_cdf_t(float $par1, float $par2, int $which): float
```

Повертає кумулятивну функцію розподілу Стьюдента, обернену їй чи один із своїх параметрів. Вигляд значення і параметрів (`par1`и`par2`) определяются параметром`which`

У наступній таблиці перераховані значення і параметри, що повертаються, в залежності від `which`. CDF, x та nu позначає функцію кумулятивного розподілу, значення випадкової змінної та кількість ступенів свободи відповідно.

**Значення, що повертається, та параметри**

| `which` | Возвращаемое значение | `par1` | `par2` |
| --- | --- | --- | --- |
|  | CDF | x | nu |
|  | x | CDF | nu |
| 3 | nu | x | CDF |

### Список параметрів

`par1`

Перший параметр

`par2`

Другий параметр

`which`

Прапор, що визначає, що буде повернено

### Значення, що повертаються

Повертає CDF, x чи nu, залежно від значення `which`
