# Yaf\_Application::run

(Yaf >=1.0.0)

Yaf\_Application::run — Запустити Yaf\_Application

### Опис

```methodsynopsis
public Yaf_Application::run(): void
```

Запускає Yaf\_Application, дозволяючи Yaf\_Application приймати запити та маршрутизувати їх, направляти в контролер/дія (Action) та малювати. І, зрештою, надсилати відповідь клієнту.

### Список параметрів

Ця функція не має параметрів.
