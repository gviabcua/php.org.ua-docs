# YafApplication::run

(Yaf >=1.0.0)

YafApplication::run — Запустити YafApplication

### Опис

```methodsynopsis
public Yaf_Application::run(): void
```

Запускає YafApplication, дозволяючи YafApplication приймати запити та маршрутизувати їх, направляти в контролер/дія (Action) та малювати. І, зрештою, надсилати відповідь клієнту.

### Список параметрів

Ця функція не має параметрів.
