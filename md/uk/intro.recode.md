# Вступ

Модуль надає інтерфейс для бібліотеки GNU Recode. Ця бібліотека конвертує файли з одного кодування в інше. Якщо конвертація не може бути здійснена абсолютно точно, помилкові символи можуть бути пропущені або замінені найбільш відповідними. Бібліотека розпізнає і може оперувати близько 150 кодування. Підтримується більшість кодувань з [» RFC 1345](http://www.faqs.org/rfcs/rfc1345)

> **Зауваження**: Модуль відділений та переміщений [» PECL](https://pecl.php.net/package/recode)починаючи з PHP 7.4.0. Натомість спробуйте використовувати модулі [Многобайтовые строки](book.mbstring.html) або [iconv](book.iconv.html)

> **Зауваження**: Для Windows-платформ цей модуль недоступний.
