# ReflectionParameter::isPassedByReference

(PHP 5, PHP 7, PHP 8)

ReflectionParameter::isPassedByReference — Перевіряє, чи передано параметр за посиланням

### Опис

```methodsynopsis
public ReflectionParameter::isPassedByReference(): bool
```

Перевіряє, що параметр передано за посиланням.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

**`true`**, якщо параметр передано за посиланням, **`false`** в іншому випадку.
