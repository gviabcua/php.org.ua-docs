## Установка

## PHP 7.4

Цей модуль був переміщений до репозиторію [» PECL](https://pecl.php.net/) і більше не постачається з PHP 7.4.0

Інформація щодо встановлення цього модуля PECL може бути знайдена у розділі посібника [Встановлення PECL модулів](install.pecl.md). Додаткову інформацію, таку як нові версії, завантаження, вихідні файли, інформація про розробника та CHANGELOG, можна знайти тут: [» https://pecl.php.net/package/wddx](https://pecl.php.net/package/wddx)

## PHP < 7.4

После установки требуемой библиотеки expat, скомпилируйте PHP с**\--enable-wddx**и используйте**\--with-libexpat-dir**для expat.

Версія PHP для Windows має вбудовану підтримку цього модуля. Тобто для виклику цих функцій не потрібно завантаження додаткових модулів.
