# ParleRLexer::reset

(PECL parle >= 0.7.1)

ParleRLexer::reset — Скидає лексер

### Опис

```methodsynopsis
public Parle\RLexer::reset(int $pos): void
```

Скидання лексингу, за бажання надаючи бажане усунення.

### Список параметрів

`pos`

Позиція скидання.

### Значення, що повертаються

Функція не повертає значення після виконання.
