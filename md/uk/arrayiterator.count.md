# ArrayIterator::count

(PHP 5, PHP 7, PHP 8)

ArrayIterator::count — Порахувати кількість елементів

### Опис

```methodsynopsis
public ArrayIterator::count(): int
```

Повертає кількість елементів у масиві (array) чи кількість громадських властивостей об'єкта (object).

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Кількість елементів чи публічних властивостей в асоціативному масиві (array) чи об'єкті (object) відповідно.
