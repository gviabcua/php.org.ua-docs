## Типи значень

Типи значень - це ті, які перевіряють тип значення та саме значення. PHP підтримує два типи значень: false з PHP 8.0.0 та true з PHP 8.2.0.

**Увага**

До PHP 8.2.0 логічний тип false можна було використовувати лише як частину [об'єднання типів](language.types.type-system.md#language.types.type-system.composite.union)

> **Зауваження**: Неможливо визначити типи значень користувача. Натомість користуються [переліками](language.types.enumerations.md)
