# sodium\_crypto\_secretstream\_xchacha20poly1305\_rekey

(PHP 7 >= 7.2.0, PHP 8)

sodium\_crypto\_secretstream\_xchacha20poly1305\_rekey - Явно повертає ключ у стан secretstream

### Опис

```methodsynopsis
sodium_crypto_secretstream_xchacha20poly1305_rekey(string &$state): void
```

Явно повертає ключ у стан secretstream. Замінює передане значення.

### Список параметрів

`state`

Стан secretstream

### Значення, що повертаються

Функція не повертає значення після виконання.
