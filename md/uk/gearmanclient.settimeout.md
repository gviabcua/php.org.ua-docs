# GearmanClient::setTimeout

(PECL gearman >= 0.6.0)

GearmanClient::setTimeout — Встановлення часу очікування для введення/виводу

### Опис

```methodsynopsis
public GearmanClient::setTimeout(int $timeout): bool
```

Встановлює час очікування для введення/виведення.

### Список параметрів

`timeout`

Тимчасовий інтервал у мілісекундах

### Значення, що повертаються

Завжди повертає **`true`**
