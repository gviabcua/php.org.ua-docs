# IntlTimeZone::getEquivalentID

# intltz\_get\_equivalent\_id

(PHP 5 >= 5.5.0, PHP 7, PHP 8, PECL >= 3.0.0a1)

IntlTimeZone::getEquivalentID -- intltz\_get\_equivalent\_id — Отримати ідентифікатор у групі схожих часових поясів, включно із заданим ідентифікатором.

### Опис

Об'єктно-орієнтований стиль (метод):

```methodsynopsis
public static IntlTimeZone::getEquivalentID(string $timezoneId, int $offset): string|false
```

Процедурний стиль:

```methodsynopsis
intltz_get_equivalent_id(string $timezoneId, int $offset): string|false
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

### Список параметрів

`timezoneId`

`offset`
