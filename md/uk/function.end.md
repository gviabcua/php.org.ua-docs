# end

(PHP 4, PHP 5, PHP 7, PHP 8)

end - Встановлює внутрішній покажчик масиву на його останній елемент

### Опис

```methodsynopsis
end(array|object &$array): mixed
```

**end()** встановлює внутрішній покажчик `array` на останній елемент та повертає його значення.

### Список параметрів

`array`

Масив. Цей масив передається за посиланням, оскільки він модифікується цією функцією. Це означає, що ви повинні передати його як реальну змінну, а не як функцію, що повертає масив, оскільки за посиланням можна передавати лише фактичні змінні.

### Значення, що повертаються

Повертає значення останнього елемента або **`false`** для порожнього масиву.

### Приклади

**Приклад #1 Приклад використання **end()****

```php
<?php

$fruits = array('apple', 'banana', 'cranberry');
echo end($fruits); // cranberry

?>
```
