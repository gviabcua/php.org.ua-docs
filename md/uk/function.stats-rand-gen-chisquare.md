# stats\_rand\_gen\_chisquare

(PECL stats >= 1.0.0)

stats\_rand\_gen\_chisquare - Обчислює випадкове відхилення від розподілу хі-квадрат

### Опис

```methodsynopsis
stats_rand_gen_chisquare(float $df): float
```

Обчислює випадкове відхилення від розподілу хі-квадрат із заданим числом ступенів свободи `df`

### Список параметрів

`df`

Кількість ступенів свободи

### Значення, що повертаються

Випадкове відхилення
