# ісreal

(PHP 4, PHP 5, PHP 7, PHP 8)

ісreal - Псевдонім [ісfloat()](function.is-float.html)

### Опис

Ця функція є псевдонімом: [ісfloat()](function.is-float.html)

**Увага**

Псевдонім оголошено *Застарілим* в PHP 7.4.0 та *ВИДАЛЕНО* у PHP 8.0.0.
