# Yaf\_View\_Interface::display

(Yaf >=1.0.0)

Yaf\_View\_Interface::display — Відображає та виводить шаблон

### Опис

```methodsynopsis
abstract public Yaf_View_Interface::display(string $tpl, array $tpl_vars = ?): bool
```

Малює шаблон і негайно виводить результат.

**Увага**

`tpl_vars`
