# YafViewInterface::display

(Yaf >=1.0.0)

YafViewInterface::display — Відображає та виводить шаблон

### Опис

```methodsynopsis
abstract public Yaf_View_Interface::display(string $tpl, array $tpl_vars = ?): bool
```
