# Imagick::setType

(PECL imagick 2, PECL imagick 3)

Imagick::setType — Встановлює атрибут типу зображення

### Опис

```methodsynopsis
public Imagick::setType(int $image_type): bool
```

Встановлює атрибут типу зображення.

### Список параметрів

`image_type`

### Значення, що повертаються
