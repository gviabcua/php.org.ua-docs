# Вступ

Модуль дозволяє завантажувати загальні (shared) бібліотеки (.DLL або .so), викликати C-функції та отримувати доступ до структур мови C з PHP без необхідності глибоко вивчати API модулів Zend і сторонньої, "проміжної" мови. Публічний API реалізований як єдиний клас [FFI](class.ffi.html) з деяким набором статичних методів (деякі можуть викликатися динамічно) і перевизначеними методами об'єкта, що дозволяє взаємодіяти з даними З.

**Застереження**

Модуль FFI досить небезпечний, оскільки дозволяє взаємодіяти із системою на дуже низькому рівні. Модуль FFI повинен використовуватися тільки розробниками, знайомими з мовою С і вміють використовувати його API. Для мінімізації ризику модуль може бути заборонений директивою [ffi.enable](ffi.configuration.html#ini.ffi.enable) у php.ini.

> **Зауваження**
> 
> Модуль FFI не робить класичний API модулів PHP застарілим; він призначений для взаємодії з функціями та структурами даних C.

**Підказка**

Зараз доступ до структур даних FFI приблизно вдвічі повільніший ніж доступ до об'єктів і масивів PHP. Таким чином, немає підстав використовувати FFI для прискорення; однак його можна використовувати для скорочення пам'яті, що споживається.
