# Вступ

Функції readline реалізують інтерфейс бібліотеки GNU Readline. Це функції, що надають командні рядки, що редагуються. Наприклад, Bash дозволяє використовувати стрілки для навігації по рядку чи прокручування історії команд. Так як ця бібліотека за своєю суттю інтерактивна, то нормально використовувати її можна хіба що для скриптів, що запускаються в [командной строке](features.commandline.html). А для веб-застосунків вона не особливо підходить.

Починаючи з PHP 7.1.0 модуль працює у Windows.

**Застереження**

Модуль readline не є безпечним! Відповідно украй не рекомендується використовувати його з потокобезпечними SAPI (наприклад, Apache modwinnt).
