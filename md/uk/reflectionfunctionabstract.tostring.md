# ReflectionFunctionAbstract::toString

(PHP 5> = 5.2.0, PHP 7, PHP 8)

ReflectionFunctionAbstract::toString — Перетворення на рядок

### Опис

```methodsynopsis
abstract public ReflectionFunctionAbstract::__toString(): void
```

Подання у вигляді рядка.

**Увага**
