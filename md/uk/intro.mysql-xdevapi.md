# Вступ

Модуль забезпечує доступ до сховища документів MySQL через X DevAPI. X DevAPI - це загальний API, що надається кількома конекторами MySQL, що забезпечує легкий доступ до реляційних таблиць, а також колекцій документів, які представлені в JSON, з API з операціями в стилі CRUD.

X DevAPI використовує X Protocol, клієнт-серверний протокол нового покоління MySQL 8.0.

Для отримання загальної інформації про сховище документів MySQL, будь ласка, зверніться до розділу [» Хранилище документов MySQL](https://dev.mysql.com/doc/refman/8.0/en/document-store.html) у посібнику MySQL.
