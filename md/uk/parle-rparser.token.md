# ParleRParser::token

(PECL parle >= 0.7.0)

ParleRParser::token — Оголошує токен

### Опис

```methodsynopsis
public Parle\RParser::token(string $tok): void
```

Оголошує термінал, який використовуватиметься у граматиці.

### Список параметрів

`tok`

Ім'я токена.

### Значення, що повертаються

Функція не повертає значення після виконання.
