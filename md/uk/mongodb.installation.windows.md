## Встановлення драйвера PHP MongoDB під Windows

Попередньо скомпіловані виконувані файли прикріплюються до [» випусків проекту на GitHub](https://github.com/mongodb/mongo-php-driver/releases/). Архіви публікуються для різних комбінацій версії PHP, потокової безпеки (TS чи NTS) та архітектури (x86 чи x64). Визначте правильний архів для PHP-оточення та вийміть файл php\_mongodb.dll - директорію з модулями (за замовчуванням "ext").

Додайте наступний рядок у файл php.ini для кожного оточення, в якому буде використано драйвер:

extension=php\_mongo.dll

Неправильний вибір двійкового файлу призведе до помилки під час спроби завантажити DLL-модуль під час виконання:

PHP Warning: PHP Startup: Unable to load dynamic library 'mongodb'

Переконайтеся, що завантажена DLL відповідає наступним властивостям середовища виконання PHP:

-   Версія PHP (**`PHP_VERSION`**) .
-   Поточна безпека (**`PHP_ZTS`**) .
-   Архітектура (**`PHP_INT_SIZE`**) .

Ці властивості можна побачити у виведенні функції [phpinfo()](function.phpinfo.md). Якщо в системі встановлено кілька версій PHP, перевірте двічі, що висновок функції [phpinfo()](function.phpinfo.md) відповідає оточенню.

> **Зауваження** **Додаткові залежності DLL для користувачів Windows**
> 
> Для роботи цього модуля системної змінної Windows PATH повинні бути доступні DLL-файли. Щоб дізнатися, як цього досягти, зверніться до розділу FAQ "[Як додати мою директорію з PHP до змінної Windows PATH](faq.installation.md#faq.installation.addtopath)".
