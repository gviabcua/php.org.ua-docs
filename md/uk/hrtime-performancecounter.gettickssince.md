# HRTime\\PerformanceCounter::getTicksSince

(PECL hrtime >= 0.6.0)

HRTime\\PerformanceCounter::getTicksSince — Кількість тиків, що пройшли із заданого значення

### Опис

```methodsynopsis
public static HRTime\PerformanceCounter::getTicksSince(int $start): int
```

Повертає тиків, що пройшли із заданого початкового значення.

### Список параметрів

`start`

Початкове значення.

### Значення, що повертаються

Повертає ціле число (int), що дорівнює кількості тиків.
