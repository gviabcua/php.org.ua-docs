## З'єднання з базою даних

Ви можете використовувати безпечні SSL або ssh з'єднання для шифрування даних, якими обмінюються клієнт і сервер. При реалізації будь-якого з перерахованого вище моніторинг трафіку і збір даних про вашу базу даних істотно ускладниться для потенційного зломщика.
