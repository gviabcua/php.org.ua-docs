# sleep

(PHP 4, PHP 5, PHP 7, PHP 8)

sleep — Затримка виконання

### Опис

```methodsynopsis
sleep(int $seconds): int
```

Відкладає виконання програми на секунду, вказану в параметрі `seconds`

> **Зауваження**
> 
> Щоб відкласти виконання програми на частки секунди, скористайтеся функцією [usleep()](function.usleep.md), оскільки функція **sleep()** очікує ціле число (int). Наприклад, `sleep(0.25)` призупинить виконання програми на `0` секунд.

### Список параметрів

`seconds`

Час зупинки за секунди.

### Значення, що повертаються

Повертає 0 у разі успішного виконання або **`false`** у разі виникнення помилки.

Якщо виклик перервано сигналом, функція **sleep()** поверне значення, що не дорівнює нулю. У Windows це значення завжди буде рівним `192` (значення константи Windows API **`WAIT_IO_COMPLETION`**). На інших платформах значенням, що повертається, буде кількість секунд, що залишилися до затримки.

### Помилки

Якщо вказана кількість секунд у параметрі `seconds` негативне, то функція згенерує помилку рівня **`E_WARNING`**

### Приклади

**Приклад #1 Приклад використання **sleep()****

```php
<?php

// текущее время
echo date('h:i:s') . "\n";

// ожидание в течениe 10 секунд
sleep(10);

// завершение ожидания
echo date('h:i:s') . "\n";

?>
```

Цей приклад виведе (через 10 секунд)

```
05:31:23
05:31:33
```
