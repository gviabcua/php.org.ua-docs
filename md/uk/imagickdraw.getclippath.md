# ImagickDraw::getClipPath

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getClipPath — Повертає ідентифікатор поточного відсічного контуру

### Опис

```methodsynopsis
public ImagickDraw::getClipPath(): string
```

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

Отримує ідентифікатор поточного відсічного контуру.

### Значення, що повертаються

Повертає рядок, що містить ідентифікатор відсічного контуру або false, якщо відсічного контуру не існує.
