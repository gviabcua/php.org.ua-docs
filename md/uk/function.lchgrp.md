# lchgrp

(PHP 5> = 5.1.3, PHP 7, PHP 8)

lchgrp — Змінює групу, якій належить символічне посилання

### Опис

```methodsynopsis
lchgrp(string $filename, string|int $group): bool
```

Намагається змінити групу, якій належить символічне посилання `filename` на вказану в `group`

Тільки суперкористувач може змінювати групу символічного посилання довільно, інші користувачі можуть змінювати групу символічного посилання на будь-яку, в якій користувач є членом.

### Список параметрів

`filename`

Шлях до заслання.

`group`

Група, вказана на ім'я або номер.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.

### Приклади

**Приклад #1 Приклад зміни групи символічного посилання**

```php
<?php
$target = 'output.php';
$link = 'output.html';
symlink($target, $link);

lchgrp($link, 8);
?>
```

### Примітки

> **Зауваження**: Ця функція не застосовується для роботи з [удалёнными файлами](features.remote-files.html)оскільки файл повинен бути доступний через файлову систему сервера.

> **Зауваження**: Для Windows-платформ ця функція не реалізована.
