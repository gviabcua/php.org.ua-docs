# GmagickDraw::ellipse

(PECL gmagick >= Unknown)

GmagickDraw::ellipse — Малює еліпс на зображенні

### Опис

```methodsynopsis
public GmagickDraw::ellipse(    float $ox,    float $oy,    float $rx,    float $ry,    float $start,    float $end): GmagickDraw
```

Малює еліпс на зображенні.

### Список параметрів

`ox`

Початкова координата x.

`oy`

Вихідна координата y.

`rx`

Радіус x.

`ry`

Радіус у y.

`start`

Початковий градус повороту.

`end`

Кінцевий градус повороту.
