# ImagickDraw::getFont

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getFont — Повертає шрифт

### Опис

```methodsynopsis
public ImagickDraw::getFont(): string
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

Повертає рядок, що визначає шрифт, який використовується під час анотації тексту.

### Значення, що повертаються

Повертає рядок у разі успішного виконання або false, якщо шрифт не визначено.
