# cosh

(PHP 4> = 4.1.0, PHP 5, PHP 7, PHP 8)

cosh - Гіперболічний косинус

### Опис

```methodsynopsis
cosh(float $num): float
```

Повертає гіперболічний косинус `num`, що визначається як `(exp(arg) + exp(-arg))/2`

### Список параметрів

`num`

Вхідне значення

### Значення, що повертаються

Гіперболічний косинус `num`
