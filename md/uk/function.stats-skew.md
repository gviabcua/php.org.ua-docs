# stats\_skew

(PECL stats >= 1.0.0)

stats\_skew — Обчислює асиметрію даних у масиві

### Опис

```methodsynopsis
stats_skew(array $a): float
```

Повертає асиметрію значень у `a`

### Список параметрів

`a`

Вхідний масив

### Значення, що повертаються

Повертає асиметрію значень у `a`или\*\*`false`\*\*, якщо `a` порожній чи не є масивом.
