# statsskew

(PECL stats >= 1.0.0)

statsskew — Обчислює асиметрію даних у масиві

### Опис

```methodsynopsis
stats_skew(array $a): float
```

Повертає асиметрію значень у `a`

### Список параметрів

`a`

Вхідний масив

### Значення, що повертаються

Повертає асиметрію значень у `a` або **`false`**, якщо `a` порожній чи не є масивом.
