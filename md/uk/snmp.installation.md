## Установка

Пакет[» Net-SNMP](http://www.net-snmp.org/) Windows містить файли для підтримки SNMP в директорії mibs. Цю директорію необхідно додати в змінні оточення Windows з ім'ям MIBDIRS і значенням, що дорівнює повному шляху до директорії mibs: наприклад, c:\\usr\\mibs.
