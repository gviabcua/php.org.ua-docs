# statsrandsetall

(PECL stats >= 1.0.0)

statsrandsetall - Встановлює початкові значення для генератора випадкових чисел

### Опис

```methodsynopsis
stats_rand_setall(int $iseed1, int $iseed2): void
```

Встановлює `iseed1` і `iseed2` як початкові значення для генератора випадкових чисел, що використовується в статистичних функціях.

### Список параметрів

`iseed1`

Значення, яке використовується як випадкове початкове число

`iseed2`

Значення, яке використовується як випадкове початкове число

### Значення, що повертаються

Жодні значення не повертаються.
