# sodium\_crypto\_sign\_keypair

(PHP 7 >= 7.2.0, PHP 8)

sodium\_crypto\_sign\_keypair — Довільно згенерувати секретний ключ та відповідний відкритий ключ

### Опис

```methodsynopsis
sodium_crypto_sign_keypair(): string
```

Створює випадкову пару ключів Ed25519 у вигляді рядка.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Ключова пара Ed25519.
