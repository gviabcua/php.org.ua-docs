# rnp\_key\_get\_info

(PECL rnp >= 0.1.1)

rnp\_key\_get\_info — Отримує інформацію про ключ

### Опис

```methodsynopsis
rnp_key_get_info(RnpFFI $ffi, string $key_fp): array|false
```

### Список параметрів

`ffi`

Об'єкт FFI, що повертається функцією rnp\_ffi\_create.

`key_fp`

Цифровий відбиток ключа.

### Значення, що повертаються

Асоціативний масив з інформацією про ключ або \*\*`false`\*\*в случае возникновения ошибки.

| Ключ | Тип данных | Опис |
| --- | --- | --- |
| `"is_primary"` | boolean | \*\*`true`\*\*якщо ключ є первинним ключем. |
| `"is_sub"` | boolean | \*\*`true`\*\*якщо цей ключ є дочірнім ключем. |
| `"is_valid"` | boolean | \*\*`true`\*\*якщо відкритий ключ дійсний. Сюди входить перевірка підписів, термінів дії, анулювання тощо. |
| `"is_revoked"` | boolean | **`true`**, якщо ключ відкликано. |
| `"is_superseded"` | boolean | **`true`**, якщо ключ замінено. Є тільки у випадку відкликання ключа. |
| `"is_compromised"` | boolean | \*\*`true`\*\*якщо цей ключ скомпрометований. Є тільки у випадку відкликання ключа. |
| `"is_retired"` | boolean | **`true`**, якщо цей ключ відкликано. Є тільки у випадку відкликання ключа. |
| `"is_expired"` | boolean | \*\*`true`\*\*якщо термін дії цього ключа минув. |
| `"have_secret"` | boolean | **`true`**, якщо ключ має закриту частину. |
| `"is_locked"` | boolean | **`true`**, якщо ключ в даний час заблоковано. Є тільки для закритих ключів. |
| `"is_protected"` | boolean | **`true`**, якщо цей ключ захищений. Є тільки для закритих ключів. Захищений ключ - це ключ, який зашифрований і може безпечно зберігатися в пам'яті та блокується/розблокується за необхідності. |
| `"have_public"` | boolean | \*\*`true`\*\*якщо у ключа є відкрита частина. Як правило, у всіх ключів є відкрита частина. |
| `"valid_till"` | integer | Мітка часу, до якої ключ вважається дійсним. Примітка: при цьому враховується не лише закінчення терміну дії ключа, але його відгук. Для дочірнього ключа, що підключається, також буде перевірятися час дії первинного ключа. |
| `"bits"` | integer | Кількість бітів у ключі. Для ключів на основі EC буде міститися розмір кривої. |
| `"alg"` | string | Назва алгоритму ключа. |
| `"subkeys"` | array | Індексований масив, що містить рядки цифрових відбитків дочірніх ключів. Є тільки для первинних ключів. Може бути порожнім, якщо первинний ключ не має дочірніх ключів. |
| `"uids"` | array | Індексований масив, що містить рядки ідентифікаторів користувачів. Є тільки для первинних ключів. Може бути порожнім, якщо первинний ключ не містить ідентифікаторів користувачів. |
