# gmprandom

(PHP 4> = 4.0.4, PHP 5, PHP 7)

gmprandom — Випадкове число

**Увага**

Ця функція оголошена *Застарілої*, починаючи з PHP 7.2.0 і була *ВИДАЛЕНО* у версії PHP 8.0.0. Використовувати цю функцію не рекомендується.

### Опис

```methodsynopsis
gmp_random(int $limiter = 20): GMP
```

Генерує випадкове число. Число буде лежати в межах від нуля до (2 n) - 1, де n - це число, що представляє собою добуток числа `limiter` та кількості біт у лімбі (limb). Якщо число `limiter` негативне, то буде повернено негативний результат.

Лімб (дослівно кінцівка) – внутрішній GMP механізм. Технічно це частина числа, що міститься в одне машинне слово. Кількість біт у ньому може різнитися у різних системах. В основному це або 32 або 64, але це не гарантується.

### Список параметрів

`limiter`

Обмежувач.

Об'єкт [GMP](class.gmp.md), ціле число (int) або числовий рядок (string).

### Значення, що повертаються

Випадкове число GMP.

### Приклади

**Приклад #1 Приклад використання **gmprandom()****

```php
<?php
$rand1 = gmp_random(1); // случайное число от 0 до 1 * бит на лимб
$rand2 = gmp_random(2); // случайное число от 0 до 2 * бит на лимб

echo gmp_strval($rand1) . "\n";
echo gmp_strval($rand2) . "\n";
?>
```

Результат виконання цього прикладу:

```
1915834968
8642564075890328087
```
