# Вступ

Swoole - це високопродуктивний мережевий фреймворк, що використовує асинхронну неблокуючу модель введення-виводу, що керується подіями. Він може використовуватися для розробки високопродуктивних, масштабованих паралельних TCP, UDP, сокетів Unix, HTTP, Websocket сервісів. Для початку, будь ласка, подивіться [» Документацію Swoole](https://wiki.swoole.com/)

> **Зауваження**: Для Windows-платформ цей модуль недоступний.
