# chroot

(PHP 4 >= 4.0.5, PHP 5, PHP 7, PHP 8)

chroot — Змінює кореневий каталог

### Опис

```methodsynopsis
chroot(string $directory): bool
```

Змінює кореневий каталог поточного процесу на переданий як параметр `directory` та змінює поточний каталог на "/".

Ця функція доступна лише на системах GNU та BSD, а також лише при використанні CLI, CGI або Embed SAPI. Крім того, ця функція вимагає прав суперкористувача.

Виклик цієї функції не змінює значення магічних констант `__DIR__`и`__FILE__`

### Список параметрів

`directory`

Шлях для зміни кореневої директорії.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або \*\*`false`\*\*в случае возникновения ошибки.

### Приклади

**Пример #1 Пример использования**chroot()\*\*\*\*

```php
<?php
chroot("/path/to/your/chroot/");
echo getcwd();
?>
```

Результат виконання наведеного прикладу:

```
/
```

### Примітки

> **Зауваження**: Для Windows-платформ ця функція не реалізована.

> **Зауваження**: Ця функція недоступна в інтерпретаторах PHP, зібраних із підтримкою ZTS (Zend Thread Safety). Використовуйте **php -i** або вбудовану константу **`PHP_ZTS`**, щоб дізнатися, чи зібрано PHP з ZTS.
