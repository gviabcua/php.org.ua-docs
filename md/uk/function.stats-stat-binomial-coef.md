# statsстатиbinomialcoef

(PECL stats >= 1.0.0)

statsстатиbinomialcoef — Отримати біномінальний коефіцієнт

### Опис

```methodsynopsis
stats_stat_binomial_coef(int $x, int $n): float
```

Повертає біномінальний коефіцієнт для `n` елементів та `x` вибірок.

### Список параметрів

`x`

Кількість вибірок із набору

`n`

Кількість елементів у наборі

### Значення, що повертаються

Повертає біномінальний коефіцієнт
