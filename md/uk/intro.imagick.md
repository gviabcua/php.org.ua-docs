# Вступ

Imagick є вбудованим PHP модулем для створення і зміни зображень використовуючи API ImageMagick.

ImageMagick є набір програмного забезпечення для створення, редагування та компонування растрових зображень. Він може читати, конвертувати та записувати зображення у різних форматах (більше 100), включаючи DPX, EXR, GIF, JPEG, JPEG-2000, PDF, PhotoCD, PNG, Postscript, SVG, та TIFF.

ImageMagick Studio LLC, некомерційна організація, що спеціалізується на розробці графічного програмного забезпечення, доступного безкоштовно.
