# Вступ

Модуль HRTime реалізує клас високої роздільної здатності StopWatch. Він використовує найкращий доступний API на різних платформах, що дозволяє оперувати часом з точністю до наносекунд. Він також дозволяє реалізовувати користувацькі секундоміри, використовуючи низькорівневі тики, що надаються нижче API.

> **Зауваження**: Починаючи з PHP 7.3.0, пов'язана функція [hrtime()](function.hrtime.md)является частью ядра.
