# Вступ

Модуль використовує бібліотеку libzookeeper, щоб надавати API для зв'язку із сервером ZooKeeper.

ZooKeeper – це проект Apache, що надає централізований сервіс для підтримки конфігураційної інформації та найменувань, надання можливості розподіленої синхронізації та групових сервісів.
