## Рядкові оператори

У PHP є два оператори для роботи з рядками (string). Перший - оператор конкатенації ('.'), який повертає рядок, що є з'єднанням лівого і правого аргументу. Другий - оператор присвоєння з конкатенацією ('`.=`'), який приєднує правий аргумент до лівого. Для отримання більш повної інформації перегляньте розділ [Оператори присвоєння](language.operators.assignment.md)

```php
<?php
$a = "Привет, ";
$b = $a . "Мир!"; // $b теперь содержит строку "Привет, Мир!"

$a = "Привет, ";
$a .= "Мир!";     // $a теперь содержит строку "Привет, Мир!"
?>
```
