svnфсtxnroot — Створює та повертає корінь транзакції.

### Опис

```methodsynopsis
svn_fs_txn_root(resource $txn): resource
```

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

### Примітки
