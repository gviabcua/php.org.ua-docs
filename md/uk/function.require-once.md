## require\_once

(PHP 4, PHP 5, PHP 7, PHP 8)

Вираз `require_once` аналогічно [require](function.require.md) за винятком того, що PHP перевірить, чи включався вже даний файл, і якщо так, не включатиме його ще раз.

Смотрите документацию[include\_once](function.include-once.md)для информации по поведению`_once` і чим він відрізняється від таких самих функцій, без `_once`
