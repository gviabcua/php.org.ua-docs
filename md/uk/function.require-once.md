## requireonce

(PHP 4, PHP 5, PHP 7, PHP 8)

Вираз `require_once` аналогічно [require](function.require.md) за винятком того, що PHP перевірить, чи включався цей файл, і якщо так, не буде включати його ще раз.

Дивіться документацію [includeonce](function.include-once.html) для інформації щодо поведінки `_once` і чим він відрізняється від таких самих функцій, без `_once`
