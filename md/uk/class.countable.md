# Інтерфейс Countable

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

## Вступ

Класи, що реалізують інтерфейс **Countable**, можуть бути використані з функцією [count()](function.count.md)

## Огляд інтерфейсів

```classsynopsis

    
     interface Countable {

    /* Методы */
    
   public count(): int

   }
```
