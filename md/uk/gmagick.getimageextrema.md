# Gmagick::getimageextrema

(PECL gmagick >= Unknown)

Gmagick::getimageextrema — Отримує екстремуми для зображення

### Опис

```methodsynopsis
public Gmagick::getimageextrema(): array
```

Повертає асоціативний масив із ключами "min" та "max". Викликає **GmagickException**в случае возникновения ошибки.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає асоціативний масив із ключами "min" та "max".

### Помилки

Викликає **GmagickException**в случае возникновения ошибки.
