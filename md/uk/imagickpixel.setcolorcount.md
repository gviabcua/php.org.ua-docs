# ImagickPixel::setColorCount

(PECL imagick 2, PECL imagick 3)

ImagickPixel::setColorCount — Опис

### Опис

```methodsynopsis
public ImagickPixel::setcolorcount(int $colorCount): bool
```

Встановлює кількість кольорів, пов'язаних із цим кольором.

### Список параметрів

`colorCount`
