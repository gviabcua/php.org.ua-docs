## Значення оброблювача виводу, що повертаються

Значення оброблювача виводу, що повертаються, внутрішньо перетворюються в рядок, слідуючи стандартній семантиці PHP-типів, з двома винятками: масиви (array) і логічні значення (bool).

Масиви перетворюються на рядок `«Array»`, але попередження «`Array to string conversion`не спрацьовує.

Якщо обробник повертає значення \*\*`false`\*\*повертається вміст буфера. Якщо обробник повертає \*\*`true`\*\*повертається порожній рядок.

> **Зауваження**: Якщо обробник повертає значення **`false`** або викидає виняток, PHP встановлює для нього прапор стану **`PHP_OUTPUT_HANDLER_DISABLED`**
