## Арифметичні оператори

Пам'ятаєте шкільні засади арифметики? Наведені нижче оператори працюють так само.

**Арифметичні операції**

| Пример | Название | Результат |
| --- | --- | --- |
| +$a | Ідентичність | Конвертація $a в int або float, що підходить. |
| $а | Заперечення | Змінити знак $a. |
| $a + $b | Додавання | Сума $a та $b. |
| $a - $b | Віднімання | Різниця $a та $b. |
| $а $в | Розмноження | Твір $a та $b. |
| $a / $b | Поділ | Приватне від поділу $a на $b. |
| $a % $b | Розподіл по модулю | Цілочисельний залишок від поділу $a на $b. |
| $а $в | Зведення в ступінь | Зведення $a до ступеня $b. |

Операція поділу ("/") повертає число з плаваючою точкою, крім випадку, коли обидва значення є цілими числами (або рядками, які перетворюються на цілі числа), які діляться націло - у разі повертається ціле значення. Для цілого розподілу використовуйте [intdiv()](function.intdiv.html)

При розподілі по модулю операнди перетворюються на цілі числа (int) (шляхом видалення дробової частини) на початок операції. Для поділу за модулем чисел із плаваючою точкою використовуйте [fmod()](function.fmod.html)

Результат операції залишку від розподілу `%` матиме той самий знак, що й ділене — тобто результат `$a % $b` матиме той самий знак, що й $a. Наприклад:

```php
<?php

echo (5 % 3)."\n";           // выводит 2
echo (5 % -3)."\n";          // выводит 2
echo (-5 % 3)."\n";          // выводит -2
echo (-5 % -3)."\n";         // выводит -2

?>
```
