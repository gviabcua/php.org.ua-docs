# SplDoublyLinkedList::bottom

(PHP 5> = 5.3.0, PHP 7, PHP 8)

SplDoublyLinkedList::bottom — Отримує вузол, що знаходиться на початку двозв'язкового списку

### Опис

```methodsynopsis

```

Значення першого вузла.
