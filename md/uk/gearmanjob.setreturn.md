# GearmanJob::setReturn

(PECL gearman >= 0.5.0)

GearmanJob::setReturn — Встановлення значення, що повертається.

### Опис

```methodsynopsis
public GearmanJob::setReturn(int $gearman_return_t): bool
```

Задає значення, що повертається для поточного завдання. Це значення вказує на завершення завдання.

### Список параметрів

`gearman_return_t`

Допустиме в Gearman значення, що повертається.
