# ImagickDraw::getStrokeDashArray

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getStrokeDashArray — Повертає масив, що представляє патерн зі штрихів та пробілів, що використовуються для обведення контурів

### Опис

```methodsynopsis
public ImagickDraw::getStrokeDashArray(): array
```

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

Повертає масив, що представляє патерн зі штрихів та пробілів, що використовуються для обведення контурів.

### Значення, що повертаються

Повертає масив у разі успішного виконання чи порожній масив, якщо патерн не заданий.
