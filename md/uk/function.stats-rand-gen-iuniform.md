# statsrandгенiuniform

(PECL stats >= 1.0.0)

statsrandгенuniform - Генерує ціле число рівномірно розподілене в діапазоні LOW (включаючи) та HIGH (включаючи)

### Опис

```methodsynopsis
stats_rand_gen_iuniform(int $low, int $high): int
```

Генерує випадкове ціле число на основі рівномірного розподілу в діапазоні `low` (включаючи) та `high` (Включаючи).

### Список параметрів

`low`

Нижня границя

`high`

Верхня межа

### Значення, що повертаються

Випадкове ціле число
