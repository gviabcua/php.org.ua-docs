# Вступ

Функції gettext реалізують NLS (Native Language Support) API, який можна використовувати для інтернаціоналізації ваших PHP-додатків. Зверніться до документації gettext для вашої системи або відкрийте її за адресою [» http://www.gnu.org/software/gettext/manual/gettext.md](http://www.gnu.org/software/gettext/manual/gettext.md)
