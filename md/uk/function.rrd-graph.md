# rrdgraph

(PECL rrd >= 0.9.0)

rrdgraph — Створює зображення з даних

### Опис

```methodsynopsis
rrd_graph(string $filename, array $options): array
```

Створює зображення для певних даних із RRD-файлу.

### Список параметрів

`filename`

Ім'я файлу для виведення графіка. Це, як правило, закінчується або `.png` `.svg` або `.eps`, Залежно від потрібного формату виведення.

`options`

Опції для створення зображення. Дивіться сторінку керівництва для команди rrd graph, щоб отримати всі можливі варіанти. Усі опції (визначення даних, визначення змінних тощо) дозволені.

### Значення, що повертаються

Повертається масив з інформацією про згенероване зображення або **`false`** у разі виникнення помилки.
