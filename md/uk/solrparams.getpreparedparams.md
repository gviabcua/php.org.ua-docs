# Solr Params::get Prepared Params

(PECL solr> = 0.9.2)

SolrParams::getPreparedParams — Повертає масив параметрів в URL-адресі

### Опис

```methodsynopsis
final public SolrParams::getPreparedParams(): array
```

Повертає масив параметрів у URL-закодованому вигляді
