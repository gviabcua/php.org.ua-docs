# Random\\Engine\\PcgOneseq128XslRr64::generate

(PHP 8 >= 8.2.0)

Random\\Engine\\PcgOneseq128XslRr64::generate — Створює 64 біти випадкової послідовності

### Опис

```methodsynopsis
public Random\Engine\PcgOneseq128XslRr64::generate(): string
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає рядок, що становить 32-бітове ціле число без знака в порядку little-endian.
