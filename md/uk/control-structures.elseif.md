## elseif/else if

(PHP 4, PHP 5, PHP 7, PHP 8)

Конструкція `elseif`, як її ім'я і каже є поєднання `if` і `else`. Аналогічно `else`, вона розширює оператор `if` для виконання різних виразів у разі, коли умова початкового оператора `if` еквівалентно **`false`**. Однак, на відміну від `else`, виконання альтернативного виразу відбудеться лише тоді, коли умова оператора `elseif` буде рівним **`true`**. Наприклад, наступний код може виводити a більше, ніж b, a дорівнює b або менше, ніж b:

```php
<?php
if ($a > $b) {
    echo "a больше, чем b";
} elseif ($a == $b) {
    echo "a равен b";
} else {
    echo "a меньше, чем b";
}
?>
```

Можливо кілька `elseif` в одному виразі `if`. Перший вираз `elseif` (якщо воно є) рівне **`true`** буде виконано. У PHP ви також можете написати 'else if' (у два слова), і тоді поведінка буде ідентичною 'elseif' (в одне слово). Синтаксично значення трохи відрізняється (якщо ви знайомі з мовою С, це те саме поведінка), але в кінцевому підсумку обидва висловлювання приведуть до одного і того ж результату.

Вираз `elseif` виконається, якщо попередній вираз `if` та попередні вирази `elseif` еквівалентні **`false`**, а поточний `elseif` дорівнює **`true`**

> **Зауваження**: Зауважте, що `elseif` і `else if` будуть рівнозначні лише за використання фігурних дужок, як у прикладах вище. Якщо використовуються синтаксис з двокрапкою для визначення умов `if``elseif`, ви не повинні розділяти `else if` на два слова, інакше це викликає фатальну помилку в PHP.

```php
<?php

/* Некорректный способ: */
if($a > $b):
    echo $a." больше, чем ".$b;
else if($a == $b): // Не скомпилируется.
    echo "Строка выше вызывает фатальную ошибку.";
endif;


/* Корректный способ: */
if($a > $b):
    echo $a." больше, чем ".$b;
elseif($a == $b): // Заметьте, тут одно слово.
    echo $a." равно ".$b;
else:
    echo $a." не больше и не равно ".$b;
endif;

?>
```
