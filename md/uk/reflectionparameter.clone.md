# ReflectionParameter::\_\_clone

(PHP 5, PHP 7, PHP 8)

ReflectionParameter::\_\_clone — Клонувати

### Опис


Клонує.

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.
