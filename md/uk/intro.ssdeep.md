# Вступ

ssdeep - це утиліта для створення та порівняння нечітких хешів або [» зависящее от контекста частичное хеширование](http://dfrws.org/2006/proceedings/12-Kornblum.pdf)

Нечітке хешування може порівнювати сигнатури, які мають "...послідовності однакових байт в однаковому порядку, хоча байти цих послідовностей можуть бути різними за змістом та довжиною".[»  страница проекта ssdeep](http://ssdeep.sourceforge.net)

Модуль надає функції для створення та порівняння нечітких хешів.
