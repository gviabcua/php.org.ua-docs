# ZMQPoll::clear

(PECL zmq >= 0.5.0)

ZMQPoll::clear — Очистити пул опитування

### Опис

```methodsynopsis
public ZMQPoll::clear(): ZMQPoll
```

Видаляє всі елементи з пулу опитування.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає поточний об'єкт.
