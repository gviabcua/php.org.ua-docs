# SplDoublyLinkedList::offsetExists

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SplDoublyLinkedList::offsetExists — Перевіряє, чи існує запитуваний індекс

### Опис

```methodsynopsis
public SplDoublyLinkedList::offsetExists(int $index): bool
```

### Список параметрів

`index`

Індекс, що перевіряється.

### Значення, що повертаються

\*\*`true`\*\*якщо запитуваний індекс `index` існує, в іншому випадку **`false`**
