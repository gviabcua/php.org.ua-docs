# V8Js::getExtensions

(PECL v8js >= 0.1.0)

V8Js::getExtensions — Повертає масив зареєстрованих модулів

### Опис

```methodsynopsis
public static V8Js::getExtensions(): array
```

Повертає масив зареєстрованих за допомогою [V8Js::registerExtension()](v8js.registerextension.html) модулів Javascript

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає масив зареєстрованих модулів чи порожній масив, якщо їх немає.
