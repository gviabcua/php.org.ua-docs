# Вступ

Модуль дозволяє прозоро читати та записувати стислі файли gzip (.gz), використовуючи більшість функцій [файловой системы](book.filesystem.html), які працюють зі стислими файлами gzip (і стиснутими файлами теж, але без використання сокетів).

> **Зауваження**
> 
> PHP поставляється з fopen-оберткою для .gzфайлів. Більше інформації можна знайти в розділі про [zlib://](wrappers.compression.html)
