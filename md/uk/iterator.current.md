# Iterator::current

(PHP 5, PHP 7, PHP 8)

Iterator::current — Повернення поточного елемента

### Опис

```methodsynopsis
public Iterator::current(): mixed
```

Повертає поточний елемент.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Може повертати будь-який тип.
