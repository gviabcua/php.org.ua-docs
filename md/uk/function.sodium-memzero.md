# sodiummemzero

(PHP 7> = 7.2.0, PHP 8)

sodiummemzero — Перезаписує рядок символами NULL

### Опис

```methodsynopsis
sodium_memzero(string &$string): void
```

**sodiummemzero()** обнуляє рядок, переданий за посиланням.

### Список параметрів

`string`

Рядок.

### Значення, що повертаються

Функція не повертає значення після виконання.
