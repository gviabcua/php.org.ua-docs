# str\_ends\_with

(PHP 8)

str\_ends\_with — Перевіряє, чи закінчується рядок заданим підрядком

### Опис

```methodsynopsis
str_ends_with(string $haystack, string $needle): bool
```

Виконує перевірку з урахуванням регістру, що вказує, чи закінчується `haystack`подстрокой`needle`

### Список параметрів

`haystack`

Рядок для пошуку.

`needle`

Подстрока для поиска в`haystack`

### Значення, що повертаються

Повертає **`true`**, якщо `haystack` закінчується підрядком `needle` **`false`** в іншому випадку.

### Приклади

**Приклад #1 Использование пустой строки`''`**

```php
<?php
if (str_ends_with('абв', '')) {
    echo "Все строки заканчиваются пустой строкой";
}
?>
```

Результат виконання наведеного прикладу:

```
Все строки заканчиваются пустой строкой
```

**Приклад #2 Демонстрація чутливості до регістру**

```php
<?php
$string = 'ленивая лиса перепрыгнула через забор';

if (str_ends_with($string, 'забор')) {
    echo "Строка заканчивается на 'забор'\n";
}

if (str_ends_with($string, 'Забор')) {
    echo 'Строка заканчивается на "Забор"';
} else {
    echo '"Забор" не найден, потому что регистр не соответствует';
}

?>
```

Результат виконання наведеного прикладу:

```
Строка заканчивается на 'забор'
"Забор" не найден, потому что регистр не соответствует
```

### Примітки

> **Зауваження**: Ця функція безпечна для обробки даних у двійковій формі.
