# rrderror

(PECL rrd >= 0.9.0)

rrderror — Отримати останнє повідомлення про помилку

### Опис

```methodsynopsis
rrd_error(): string
```

Повертає останнє повідомлення про помилку.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Останнє повідомлення про помилку.
