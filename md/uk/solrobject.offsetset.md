# SolrObject::offsetSet

(PECL solr> = 0.9.2)

SolrObject::offsetSet - Встановлює значення властивості

### Опис

```methodsynopsis
public SolrObject::offsetSet(string $property_name, string $property_value): void
```

Встановлює значення якості. Використовується, коли об'єкт обробляється масивом. Цей об'єкт доступний лише для читання. Цього ніколи не слід робити.

### Список параметрів

`property_name`

Ім'я якості.

`property_value`

Нове значення.

### Значення, що повертаються

Нічого не вертає.
