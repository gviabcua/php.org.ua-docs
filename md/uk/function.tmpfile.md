# tmpfile

(PHP 4, PHP 5, PHP 7, PHP 8)

tmpfile — Створює тимчасовий файл

### Опис

```methodsynopsis
tmpfile(): resource|false
```

Створює тимчасовий файл з унікальним ім'ям, відкриваючи його в режимі читання та запису (w+) та повертає файловий покажчик.

Цей файл автоматично видаляється після закриття (наприклад, шляхом виклику функції [fclose()](function.fclose.html) або якщо не залишилося жодного посилання на покажчик файлу, що повертається **tmpfile()**), або після завершення роботи скрипта.

**Застереження**

Якщо скрипт несподівано завершиться, тимчасовий файл не можна видалити.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає дескриптор файлу, аналогічний тому, що повертає функцію [fopen()](function.fopen.html) для нових файлів або **`false`** у разі виникнення помилки.

### Приклади

**Приклад #1 Приклад використання функції **tmpfile()****

```php
<?php
$temp = tmpfile();
fwrite($temp, "записываем во временный файл");
fseek($temp, 0);
echo fread($temp, 1024);
fclose($temp); // происходит удаление файла
?>
```

Результат виконання цього прикладу:

```
записываем во временный файл
```
