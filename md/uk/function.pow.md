# pow

(PHP 4, PHP 5, PHP 7, PHP 8)

pow — Зводить до ступеня

### Опис

```methodsynopsis
pow(mixed $num, mixed $exponent): int|float|object
```

Повертає число `num`в степени`exponent`

> **Зауваження** :
> 
> Допустимо вказувати оператор [\*\*](language.operators.arithmetic.md) замість дзвінка функції.

### Список параметрів

`num`

Заснування.

`exponent`

Показник ступеня.

### Значення, що повертаються

Повертає основу `num`в степени`exponent`. Якщо обидва аргументи - невід'ємні цілі і результат можна подати як ціле число, то функція повертає ціле число (int). В іншому випадку функція повертає число з плаваючою точкою (float).

### Приклади

**Приклад #1 Приклад використання функції **pow()****

```php
<?php

var_dump(pow(2, 8)); // int(256)
echo pow(-1, 20); // 1
echo pow(0, 0); // 1
echo pow(10, -1); // 0.1

echo pow(-1, 5.5); // NAN

?>
```

### Примітки

> **Зауваження** :
> 
> Функція перетворює вхідні значення, навіть нескалярні, на числові, що іноді дає *дивний*результат.
