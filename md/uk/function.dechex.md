# dechex

(PHP 4, PHP 5, PHP 7, PHP 8)

dechex - Переводить число з десяткової системи числення в шістнадцяткову

### Опис

```methodsynopsis
dechex(int $num): string
```

Повертає рядок, що містить шістнадцяткове уявлення заданого беззнакового аргументу `num`

Найбільше число, яке функція перетворює, **`PHP_INT_MAX`** `* 2 + 1`(или`-1`): на 32-бітних платформах це число `4 294 967 295` у десятковій системі, для якого функція **dechex()** поверне значення `ffffffff`

### Список параметрів

`num`

Десятичне значення перетворення.

Оскільки в PHP значення цілого типу (int) знакові, а функція **dechex()** працює з беззнаковими цілими числами, то негативні цілі числа будуть оброблятися як беззнакові.

### Значення, що повертаються

Возвращает шестнадцатеричное строковое представление числа`num`

### Приклади

**Приклад #1 Приклад використання функції** dechex()\*\*\*\*

```php
<?php

echo dechex(10) . "\n";
echo dechex(47);

?>
```

Результат виконання наведеного прикладу:

```
a
2f
```

**Приклад #2 Приклад використання функції** dechex()\*\* з великими цілими числами\*\*

```php
<?php
// Вывод подразумевает 32-битную платформу.
// Обратите внимание, что вывод одинаков для всех значений.
echo dechex(-1)."\n";
echo dechex(PHP_INT_MAX * 2 + 1)."\n";
echo dechex(pow(2, 32) - 1)."\n";

?>
```

Результат виконання наведеного прикладу:

```
ffffffff
ffffffff
ffffffff
```
