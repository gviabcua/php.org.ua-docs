## Глобальний простір

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

Без визначення простору імен визначення класів і функцій розміщуються у глобальному просторі — так само, як це було в PHP до введення просторів імен. Додавання префіксу `\` до імен вказує, що потрібне ім'я із глобального простору, навіть у контексті простору імен.

**Приклад #1 Специфікація глобального простору**

```php
<?php

namespace A\B\C;

/* Функция — это A\B\C\fopen */
function fopen() {
     /* ... */
     $f = \fopen(...); // Вызов глобальной функции fopen
     return $f;
}

?>
```
