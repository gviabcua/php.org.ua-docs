## Глобальний простір

(PHP 5> = 5.3.0, PHP 7, PHP 8)

Без визначення простору імен, визначення всіх класів та функцій перебувають у глобальному просторі - так само, як це було у PHP до введення просторів імен. Додавання префіксу `\` до імен означає, що це ім'я має знаходитись у глобальному просторі, навіть якщо ви знаходитесь у контексті певного простору імен.

**Приклад #1 Використання глобального простору та його завдання**

```php
<?php
namespace A\B\C;

/* Эта функция является A\B\C\fopen */
function fopen() {
     /* ... */
     $f = \fopen(...); // вызов глобальной функции fopen
     return $f;
}
?>
```
