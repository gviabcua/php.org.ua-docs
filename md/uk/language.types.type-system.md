## Система типів

Мова PHP використовує систему номінальних типів із сильним ставленням поведінкового підтипування. PHP перевіряє ставлення підтипів під час компіляції, тоді як контроль типів мову виконує динамічно під час виконання.

Система типів PHP підтримує атомарні типи, які можна об'єднувати, щоб створювати складні типи. Частину цих типів можна записувати в [оголошеннях типів](language.types.declarations.md)

### Атомарні типи

Окремі атомарні типи - це вбудовані типи, які тісно інтегровані з мовою і які неможливо відтворити визначенням типів користувача.

Список базових типів:

-   Вбудовані типи
    -   null
    -   Скалярні типи:
        -   bool
        -   int
        -   float
        -   string
    -   array
    -   object
    -   resource
    -   never
    -   void
    -   « [Відносні типи класів](language.types.relative-class-types.md) »: self, parent і static
-   [Типи значень](language.types.value.md)
    -   false
    -   true
-   Типи, що визначаються користувачем (часто звані клас-типами)
    -   [Інтерфейси](language.oop5.interfaces.md)
    -   [Класи](language.oop5.basic.md#language.oop5.basic.class)
    -   [Перерахування](language.types.enumerations.md)
-   [callable](language.types.callable.md)

### Складові типи

Можливе поєднання кількох атомарних типів у складові типи. PHP дозволяє об'єднувати типи такими способами:

-   Перетин клас-типів (інтерфейсів та назв класів).
-   Об'єднання типів.

#### Перетин типів

Перетин типів приймає значення, які задовольняють не одному, а одночасно кільком оголошенням клас-типів. Окремі типи, що перетинаються, з'єднуються символом `&`. Тому перетин типів `T` `U`и`V`, буде записано як `T&U&V`

#### Об'єднання типів

Об'єднання типів приймає значення кількох різних типів, а чи не одного. Окремі типи, що об'єднуються, з'єднуються символом . Тому об'єднання типів `T` `U`и`V`, буде записано як `T|U|V`. Якщо при об'єднанні типів додають перетин типів, то перетин типів беруть у дужки для запису в диз'юнктивній нормальній формі (DNF): `T|(X&Y)`

### Псевдоніми типів

PHP поддерживает два псевдонима типов:[mixed](language.types.declarations.md#language.types.declarations.mixed) і [iterable](language.types.iterable.md), які відповідають (у тому самому порядку) [об'єднанню типів](language.types.type-system.md#language.types.type-system.composite.union) `object|resource|array|string|float|int|bool|null`и`Traversable|array`

> **Зауваження**: PHP не підтримує типи, що визначаються користувачем псевдоніми.
