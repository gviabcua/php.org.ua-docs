# spl\_object\_id

(PHP 7 >= 7.2.0, PHP 8)

spl\_object\_id — Отримати цілий ідентифікатор об'єкта

### Опис

```methodsynopsis
spl_object_id(object $object): int
```

Ця функція повертає унікальний ідентифікатор об'єкта. Ідентифікатор об'єкта є унікальним протягом життя об'єкта. Коли об'єкт буде знищений, його ідентифікатор може бути повторно використаний інших об'єктів. Ця поведінка схожа на [spl\_object\_hash()](function.spl-object-hash.md)

### Список параметрів

`object`

Будь-який об'єкт.

### Значення, що повертаються

Цілочисельний ідентифікатор, який є унікальним для кожного існуючого об'єкта в даний час і завжди є однаковим для кожного об'єкта.

### Приклади

**Приклад #1 Приклад використання** spl\_object\_id()\*\*\*\*

```php
<?php
$id = spl_object_id($object);
$storage[$id] = $object;
?>
```

### Примітки

> **Зауваження** :
> 
> Коли об'єкт буде знищений, його ідентифікатор може бути повторно використаний інших об'єктів.
