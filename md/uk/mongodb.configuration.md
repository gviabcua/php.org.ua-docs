# Налаштування під час виконання

Поведінка цих функцій залежить від установок у php.ini.

**Опції налаштування mongodb**

| Имя | По умолчанию | Место изменения | Список изменений |
| --- | --- | --- | --- |
| [mongodb.debug](mongodb.configuration.html#ini.mongodb.debug) | "" | PHPINIALL |  |

Коротке пояснення конфігураційних директив.

`mongodb.debug` string

Цей параметр можна використовувати для увімкнення або вимкнення ведення журналу налагодження на рівні трасування в драйвері (і libmongoc).

Вкажіть порожній рядок, `"0"` `"off"` `"no"` або `"false"`, щоб вимкнути ведення журналу.

Вкажіть `"stderr"` або `"stdout"`, щоб вести журнал у `stderr` або `stdout`відповідно.

Вкажіть `"1"` `"on"` `"yes"` або `"true"` для ведення журналу новий тимчасовий файл в системному тимчасовому каталозі за промовчанням (тобто . [sysgettempdir()](function.sys-get-temp-dir.html)

Вкажіть будь-який інший рядок для ведення журналу до нового тимчасового файлу в цьому каталозі. Якщо каталог не може бути використаний, замість нього використовуватиметься системний часовий каталог за замовчуванням.

> **Зауваження**
> 
> Зверніть увагу, що журнал налагодження може містити конфіденційну інформацію, таку як облікові дані на сервері MongoDB та повні документи, записані чи прочитані з сервера. Будь ласка, перегляньте всі журнали налагодження, перш ніж ділитися ними з іншими людьми.
