# pi

(PHP 4, PHP 5, PHP 7, PHP 8)

pi — Повертає число Пі

### Опис

```methodsynopsis
pi(): float
```

Повертає приблизне значення числа Пі. Константа **`M_PI`** дає ідентичний результат.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає приблизне значення числа Пі як числа з плаваючою точкою (float).

### Приклади

**Приклад #1 Приклад використання функції** pi()\*\*\*\*

```php
<?php

echo pi(); // 3.1415926535898
echo M_PI; // 3.1415926535898

?>
```
