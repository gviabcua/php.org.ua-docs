# Iterator::next

(PHP 5, PHP 7, PHP 8)

Iterator::next — Переходить до наступного елементу

### Опис

```methodsynopsis
public Iterator::next(): void
```

Переміщення поточної позиції до наступного елемента.

> **Зауваження** :
> 
> Метод викликається *після* кожної ітерації [foreach](control-structures.foreach.md)

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Будь-яке значення, що повертається, ігнорується.
