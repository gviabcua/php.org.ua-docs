# MongoDB\\BSON\\Iterator::valid

(mongodb >=1.16.0)

MongoDB\\BSON\\Iterator::valid — Перевіряє, чи є поточна позиція коректною

### Опис

```methodsynopsis
public MongoDB\BSON\Iterator::valid(): bool
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає \*\*`true`\*\*якщо поточна позиція ітератора коректна, інакше повертає **`false`**
