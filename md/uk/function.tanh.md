# tanh

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

tanh - Обчислює гіперболічний тангенс

### Опис

```methodsynopsis
tanh(float $num): float
```

Повертає гіперболічний тангенс числа `num`, який визначено в результаті обчислення `sinh($num)/cosh($num)`

### Список параметрів

`num`

Аргумент для обчислення.

### Значення, що повертаються

Повертає гіперболічний тангенс числа `num`
