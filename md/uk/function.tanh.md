# tanh

(PHP 4> = 4.1.0, PHP 5, PHP 7, PHP 8)

tanh - Гіперболічний тангенс

### Опис

```methodsynopsis
tanh(float $num): float
```

Повертає гіперболічний тангенс `num`, що визначається як `sinh(arg)/cosh(arg)`

### Список параметрів

`num`

Вхідне значення

### Значення, що повертаються

Гіперболічний тангенс `num`
