# SolrQuery::setFacet

(PECL solr> = 0.9.2)

SolrQuery::setFacet — Відповідний параметр фасету. Включає або вимикає фасетування

### Опис

```methodsynopsis
public SolrQuery::setFacet(bool $flag): SolrQuery
```

Включає або вимикає фасетування.

### Список параметрів

`value`

**`true`** включає, а **`false`** відключає.

### Значення, що повертаються

Повертає поточний об'єкт SolrQuery, якщо використовується значення, що повертається.
