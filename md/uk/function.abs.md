# abs

(PHP 4, PHP 5, PHP 7, PHP 8)

abs - Повертає абсолютну величину (модуль числа)

### Опис

```methodsynopsis
abs(int|float $num): int|float
```

Повертає абсолютну величину числа `num`

### Список параметрів

`num`

Числове значення.

### Значення, що повертаються

Повертає абсолютну величину числа `num`. Якщо параметр `num` передано аргумент у вигляді числа з плаваючою точкою (float), функція також повертає число з плаваючою точкою (float), інакше - ціле число (int) (оскільки у float часто більший діапазон значень, ніж у int).

### список змін

| Версия | Опис |
| --- | --- |
| 8.0.0 | Параметр`num` більше не приймає внутрішні об'єкти, що підтримують числове перетворення. |

### Приклади

**Приклад #1 Приклад використання функції** abs()\*\*\*\*

```php
<?php

var_dump(abs(-4.2));
var_dump(abs(5));
var_dump(abs(-5));
?>
```

Результат виконання наведеного прикладу:

```
float(4.2)
int(5)
int(5)
```
