# ParleParser::token

(PECL parle >= 0.5.1)

ParleParser::token — Оголошує токен

### Опис

```methodsynopsis
public Parle\Parser::token(string $tok): void
```

Оголошує термінал, який використовуватиметься у граматиці.

### Список параметрів

`tok`

Ім'я токена.

### Значення, що повертаються

Функція не повертає значення після виконання.
