# GmagickDraw::setfontstyle

(PECL gmagick >= Unknown)

GmagickDraw::setfontstyle — Встановлює стиль шрифту для використання під час анотації тексту

### Опис

```methodsynopsis
public GmagickDraw::setfontstyle(int $style): GmagickDraw
```

Встановлює стиль шрифту, який буде використовуватися під час анотації тексту. Використання AnyStyle діє як "будь-який".

### Список параметрів

`style`

Стиль шрифту (NormalStyle, ItalicStyle, ObliqueStyle, AnyStyle).

### Значення, що повертаються

Об'єкт [Gmagick](class.gmagick.html)
