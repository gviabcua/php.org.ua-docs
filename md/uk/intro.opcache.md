# Вступ

OPcache покращує продуктивність PHP шляхом збереження скомпільованого байт-коду скриптів в пам'яті, що розділяється, тим самим позбавляючи PHP від ​​необхідності завантажувати і аналізувати скрипти при кожному запиті.

Цей модуль доступний за замовчуванням з PHP 5.5.0 та [» доступний у PECL](https://pecl.php.net/package/ZendOpcache) для версій 5.2, 5.3 та 5.4.
