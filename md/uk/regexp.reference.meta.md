## Метасимволи

Сила регулярних виразів виходить із можливості використовувати умови та повторення у шаблоні. Вони записуються за допомогою *метасимволів*, які спеціальним чином інтерпретуються.

Існують два різні набори метасимволів: ті, які використовуються всередині квадратних дужок, і ті, що використовуються поза квадратними дужками. Поза квадратними дужками використовуються такі метасимволи:

**Метасимволи поза квадратними дужками**

| Метасимвол | Описание                                                                                                                                                                                       |
|------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
|            | загальний екрануючий символ, що припускає кілька варіантів застосування                                                                                                                        |
|            | декларує початок даних (або рядки у багаторядковому режимі)                                                                                                                                    |
|            | декларує кінець даних або до завершення рядка (або закінчення рядка у багаторядковому режимі)                                                                                                  |
|            | відповідає будь-якому символу, крім перекладу рядка (за замовчуванням)                                                                                                                         |
|            | початок опису символьного класу                                                                                                                                                                |
|            | кінець опису символьного класу                                                                                                                                                                 |
|            |                                                                                                                                                                                                |
|            | початок підмаски                                                                                                                                                                               |
|            | кінець підмаски                                                                                                                                                                                |
|            | розширює зміст метасимволу (, є також квантифікатором, що означає 0 або 1 входження, також перетворює жадібні квантифікатори в ліниві (дивіться [повторение](regexp.reference.repetition.html) |
|            | квантифікатор, що означає 0 або більше входжень                                                                                                                                                |
|            | квантифікатор, що означає 1 або більше входжень                                                                                                                                                |
|            | початок кількісного квантифікатора                                                                                                                                                             |
|            | кінець кількісного квантифікатора                                                                                                                                                              |

Частина шаблону, укладена у квадратні дужки, називається [символьным классом](regexp.reference.character-classes.html). У символьному класі використовуються такі метасимволи:

**Метасимволи всередині квадратних дужок (*символьному класі***

| Метасимвол | Описание                                                   |
|------------|------------------------------------------------------------|
|            | загальний екрануючий символ                                |
|            | означає заперечення класу, допустимо лише на початку класу |
|            | означає символьний інтервал                                |

Наступні розділи детально описують кожен із перерахованих метасимволів.
