## . Объекты

### Ініціалізація об'єкту

Для створення нового об'єкта, використовуйте вираз `new`, що створює в змінній екземпляр класу:

```php
<?php
class foo
{
    function do_foo()
    {
        echo "Код foo.";
    }
}

$bar = new foo;
$bar->do_foo();
?>
```

Полное рассмотрение производится в разделе[Класи та Об'єкти](language.oop5.md)

### Перетворення на об'єкт

Якщо object перетворюється на object, об'єкт не зміниться. Якщо значення іншого типу перетворюється на object, створюється новий екземпляр вбудованого класу [stdClass](class.stdclass.md). Якщо значення було **`null`**, новий екземпляр буде порожнім. Масиви перетворюються на об'єкт з іменами полів, названими згідно з ключами масиву та відповідними їм значеннями. Зауважте, що в цьому випадку до PHP 7.2.0 числові ключі не будуть доступні, поки не проітерувати об'єкт.

```php
<?php
$obj = (object) array('1' => 'foo');
var_dump(isset($obj->{'1'})); // выводит 'bool(true)', начиная с PHP 7.2.0; 'bool(false)' ранее
var_dump(key($obj)); // выводит 'string(1) "1"', начиная с PHP 7.2.0; 'int(1)' ранее
?>
```

При преобразовании любого другого значения, оно будет помещено в поле с именем`scalar`соответствующему типу.

```php
<?php
$obj = (object) 'привет';
echo $obj->scalar;  // выведет 'привет'
?>
```
