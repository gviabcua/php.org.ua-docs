# ReflectionClassConstant::getModifiers

(PHP 7> = 7.1.0, PHP 8)

ReflectionClassConstant::getModifiers — Отримує модифікатори константи класу

### Опис

```methodsynopsis
public ReflectionClassConstant::getModifiers(): int
```

Повертає бітове поле модифікаторів доступу для цієї константи класу.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Числове уявлення модифікаторів. Фактичні значення цих модифікаторів описані в [визначених константах](class.reflectionclassconstant.html#reflectionclassconstant.constants.modifiers)
