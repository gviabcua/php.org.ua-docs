# Вступ

Цей модуль застарів з версії PHP 5.5.0 і видалено у версії 7.0.0. Замість нього використовуйте [mysqli](book.mysqli.md) або [PDOMySQL](ref.pdo-mysql.md). Ознайомтеся також з [оглядом MySQL API](mysqlinfo.api.choosing.md) для більш детальної інформації щодо вибору MySQL API.

Ці функції дозволяють працювати з MySQL. За інформацією про MySQL звертайтесь до [» http://www.mysql.com/](http://www.mysql.com/)

Документація MySQL знаходиться за адресою [» http://dev.mysql.com/doc/](http://dev.mysql.com/doc/)
