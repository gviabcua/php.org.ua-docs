# Parle\\RParser::sigilName

(PECL parle >= 0.8.4)

Parle\\RParser::sigilName — Отримує ім'я правила або токена

### Опис

```methodsynopsis
public Parle\RParser::sigilName(int $idx): string
```

Отримання часткового імені правила або імені токена, що відповідає індексу.

### Список параметрів

`idx`

Індекс відповідності, відлік починається з нуля

### Значення, що повертаються

Повертає рядок (string) із знайденою частиною.
