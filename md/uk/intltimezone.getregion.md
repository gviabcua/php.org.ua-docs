# IntlTimeZone::getRegion

# intltz\_get\_region

(PHP 5 >= 5.5.0, PHP 7, PHP 8)

IntlTimeZone::getRegion -- intltz\_get\_region — Отримати код регіону, який відповідає заданому ідентифікатору системного часового поясу

### Опис

Об'єктно-орієнтований стиль (метод):

```methodsynopsis
public static IntlTimeZone::getRegion(string $timezoneId): string|false
```

Процедурний стиль:

```methodsynopsis
intltz_get_region(string $timezoneId): string|false
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

### Список параметрів

`timezoneId`

### Значення, що повертаються

Повертає регіон або \*\*`false`\*\*в случае возникновения ошибки.
