# SQLite3::lastErrorCode

(PHP 5> = 5.3.0, PHP 7, PHP 8)

SQLite3::lastErrorCode — Повертає числовий код результату останнього невдалого запиту SQLite

### Опис

```methodsynopsis
public SQLite3::lastErrorCode(): int
```

Повертає числовий код результату останнього запиту SQLite.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає ціле значення, яке представляє числовий код результату останнього невдалого запиту SQLite.
