# GmagickDraw::setfont

(PECL gmagick >= Unknown)

GmagickDraw::setfont — Встановлює вказаний шрифт для використання під час анотації тексту

### Опис

```methodsynopsis
public GmagickDraw::setfont(string $font): GmagickDraw
```

Встановлює повністю вказаний шрифт для використання під час анотації тексту.

### Список параметрів

`font`

Назва шрифту.

### Значення, що повертаються

Об'єкт [Gmagick](class.gmagick.md)
