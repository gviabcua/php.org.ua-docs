# PDO::pgsqlGetPid

(PHP 5> = 5.6.0, PHP 7, PHP 8)

PDO::pgsqlGetPid — Отримує PID сервера

### Опис

```methodsynopsis
public PDO::pgsqlGetPid(): int
```

Повертає сервер PID.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

PID сервер.
