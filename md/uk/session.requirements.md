## Вимоги

Для збирання цього модуля не потрібні зовнішні бібліотеки.

> **Зауваження** :
> 
> При бажанні можна використовувати квотування пам'яті, що розділяється (mm), розроблене Ralf S. Engelschall, для зберігання сесії. Для цього необхідно завантажити та встановити [» mm](http://www.ossp.org/pkg/lib/mm/). Ця опція недоступна для платформ на базі Windows. Слід також звернути увагу на те, що модуль зберігання даних сесії при використанні mm не гарантує блокування інформації сесії при паралельних запитах. Можливо, в такому випадку доцільне використання файлової системи в пам'яті, що розділяється (Наприклад, TMPFS в Solaris/Linux або /dev/md для BSD), щоб зберігати сесії у файлах, блокованих при зверненні до них. Оскільки дані сесії зберігаються в пам'яті, що розділяється, перезапуск веб-сервера призведе до їх видалення.
