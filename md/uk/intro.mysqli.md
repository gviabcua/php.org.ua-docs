# Вступ

Модуль`mysqli` дозволяє отримати доступ до функціональності, яку надає MySQL версії 4.1 і вище. Більше інформації про сервер баз даних MySQL ви можете отримати за адресою [» http://www.mysql.com/](http://www.mysql.com/)

Информация об использовании MySQL из PHP находится по адресу[Короткий огляд](mysqli.overview.md)

Документация MySQL находится по адресу[» http://dev.mysql.com/doc/](http://dev.mysql.com/doc/)

Деякі частини цієї документації включені з посібника MySQL з дозволу корпорації Oracle.

Всі приклади, наведені в документації, використовують доступні вільно бази даних [» world](http://dev.mysql.com/doc/world-setup/en/index.md) і [» sakila](http://dev.mysql.com/doc/sakila/en/index.md)
