# Вступ

Модуль `mysqli` дозволяє отримати доступ до функціональності, яку надає MySQL версії 4.1 і вище. Більше інформації про сервер баз даних MySQL ви можете отримати за адресою [» http://www.mysql.com/](http://www.mysql.com/)

Інформація про використання MySQL із PHP знаходиться за адресою [Краткий обзор](mysqli.overview.html)

Документація MySQL знаходиться за адресою [» http://dev.mysql.com/doc/](http://dev.mysql.com/doc/)

Деякі частини цієї документації включені з посібника MySQL з дозволу корпорації Oracle.

Всі приклади, наведені в документації, використовують доступні вільно бази даних [» world](http://dev.mysql.com/doc/world-setup/en/index.html) і [» sakila](http://dev.mysql.com/doc/sakila/en/index.html)
