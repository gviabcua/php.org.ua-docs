## Повернення значень

Значення повертаються з допомогою необов'язкового оператора повернення. Значення, що повертаються можуть бути будь-якого типу, в тому числі це можуть бути масиви і об'єкти. Повернення призводить до завершення виконання функції та передачі управління назад до того рядка коду, в якому ця функція була викликана. Для отримання більш детальної інформації ознайомтеся з описом [return](function.return.html)

> **Зауваження**
> 
> Якщо конструкція [return](function.return.html) не вказано, то функція поверне значення **`null`**

### Використання виразу return

**Приклад #1 Використання конструкції [return](function.return.html)**

```php
<?php
function square($num)
{
    return $num * $num;
}
echo square(4);   // выводит '16'.
?>
```

Функція не може повертати кілька значень, але аналогічного результату можна досягти, повертаючи масив.

**Приклад #2 Повернення кількох значень як масиву**

```php
<?php
function small_numbers()
{
    return [0, 1, 2];
}
// Деструктуризация массива будет собирать каждый элемент массива индивидуально
[$zero, $one, $two] = small_numbers();

// До версии 7.1.0 единственной эквивалентной альтернативой было использование конструкции list().
list($zero, $one, $two) = small_numbers();

?>
```

Для того, щоб функція повертала результат за посиланням, вам необхідно використовувати оператор & і при описі функції, і при присвоєнні змінної значення, що повертається:

**Приклад #3 Повернення результату за посиланням**

```php
<?php
function &returns_reference()
{
    return $someref;
}

$newref =& returns_reference();
?>
```

Для отримання більш детальної інформації про посилання зверніться до розділу документації [Подробно о ссылках](language.references.html)
