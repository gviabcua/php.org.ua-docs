# Pool::resize

(PECL pthreads >= 2.0.0)

Pool::resize — Змінює розмір пулу

### Опис

```methodsynopsis
public Pool::resize(int $size): void
```

Змінює розмір пулу

### Список параметрів

`size`

Максимальна кількість воркерів, яка може створити цей пул

### Значення, що повертаються

Функція не повертає значення після виконання.
