# ReflectionClass::isIterateable

(PHP 5, PHP 7, PHP 8)

ReflectionClass::isIterateable - Псевдонім [ReflectionClass::isIterable()](reflectionclass.isiterable.html)

### Опис

Псевдонім [ReflectionClass::isIterable()](reflectionclass.isiterable.html)

Починаючи з PHP 7.2.0, замість методу **RefectionClass::isIterateable()**, , в ім'я якого закралася синтаксична помилка, варто використати правильно іменований метод [ReflectionClass::isIterable()](reflectionclass.isiterable.html)
