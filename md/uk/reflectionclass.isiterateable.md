# ReflectionClass::isIterateable

(PHP 5, PHP 7, PHP 8)

ReflectionClass::isIterateable - Псевдонім [ReflectionClass::isIterable()](reflectionclass.isiterable.md)

### Опис

Псевдонім [ReflectionClass::isIterable()](reflectionclass.isiterable.md)
