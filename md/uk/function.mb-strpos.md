# мбstrpos

(PHP 4> = 4.0.6, PHP 5, PHP 7, PHP 8)

мбstrpos — Пошук позиції першого входження одного рядка до іншого

### Опис

```methodsynopsis
mb_strpos(    string $haystack,    string $needle,    int $offset = 0,    ?string $encoding = null): int|false
```

Шукає позицію першого входження одного рядка string до іншого рядка string.

Виконує безпечну з погляду багатобайтних кодувань операцію [strpos()](function.strpos.html), яка спирається на число символів у рядку. Перший символ стоїть позиції 0, позиція другого 1 тощо.

### Список параметрів

`haystack`

Рядок string в якому провадиться пошук.

`needle`

Рядок, пошук якого проводиться у рядку `haystack`. На відміну від [strpos()](function.strpos.html), числові значення не застосовуються як порядкові номери символів.

`offset`

Зміщення початку пошуку. Якщо не заданий, використовується 0. Якщо значення є негативним, відлік йде з кінця рядка.

`encoding`

Параметр `encoding` є символьним кодуванням. Якщо він опущений або дорівнює **`null`**, замість нього буде використано значення внутрішнього кодування.

### Значення, що повертаються

Повертає число – позицію першого входження рядка `needle` у рядок `haystack` string. Якщо рядок `needle` не знайдено, функція поверне **`false`**

### список змін

| Версия | Описание                                                    |
|--------|-------------------------------------------------------------|
|        | `needle` тепер приймає порожній рядок.                      |
|        | Тепер параметр `encoding` може набувати значення **`null`** |
|        | Додано підтримку негативних значень `offset`                |
