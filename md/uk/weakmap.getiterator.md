# WeakMap::getIterator

(PHP 8)

WeakMap::getIterator — Отримує зовнішній ітератор

### Опис

```methodsynopsis
public WeakMap::getIterator(): Iterator
```

Повертає зовнішній ітератор.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Примірник об'єкта, що реалізує [Iterator](class.iterator.md) або [Traversable](class.traversable.md)

### Помилки

Викидає [Exception](class.exception.md)в случае возникновения ошибки.
