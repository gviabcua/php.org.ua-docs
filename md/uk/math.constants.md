# Обумовлені константи

Наведені нижче константи завжди доступні як частина ядра PHP.

**Математичні константи**

| Константа                 | Значение        | Описание                      | Доступна с версии |
|---------------------------|-----------------|-------------------------------|-------------------|
| **`M_PI`**                |                 | число Пі                      |                   |
| **`M_E`**                 |                 | число Ейлера (e)              |                   |
| **`M_LOG2E`**             |                 | log2 e                        |                   |
| **`M_LOG10E`**            |                 | lg e                          |                   |
| **`M_LN2`**               |                 | ln 2                          |                   |
| **`M_LN10`**              |                 | ln 10                         |                   |
| **`M_PI_2`**              |                 | Пі/2                          |                   |
| **`M_PI_4`**              |                 | Пі/4                          |                   |
| **`M_1_PI`**              |                 | 1/Пі                          |                   |
| **`M_2_PI`**              |                 | 2/Пі                          |                   |
| **`M_SQRTPI`**            |                 | sqrt(Пі)                      | PHP 5.2.0         |
| **`M_2_SQRTPI`**          |                 | 2/sqrt(Пі)                    |                   |
| **`M_SQRT2`**             |                 | sqrt(2)                       |                   |
| **`M_SQRT3`**             |                 | sqrt(3)                       |                   |
| **`M_SQRT1_2`**           |                 | 1/sqrt(2)                     |                   |
| **`M_LNPI`**              |                 | Натуральний логарифм числа пі |                   |
| **`M_EULER`**             |                 | Постійна Ейлера               |                   |
| **`PHP_ROUND_HALF_UP`**   |                 | Округлення до більшого цілого |                   |
| **`PHP_ROUND_HALF_DOWN`** |                 | Округлення до меншого цілого  |                   |
| **`PHP_ROUND_HALF_EVEN`** |                 | Округлення до парного числа   |                   |
| **`PHP_ROUND_HALF_ODD`**  |                 | Округлення до непарного числа |                   |
| **`NAN`**                 | NAN (тип float) | Чи не є числом (Not A Number) |                   |
| **`INF`**                 | INF (тип float) | Нескінченність                |                   |
