# GmagickDraw::setstrokeopacity

(PECL gmagick >= Unknown)

GmagickDraw::setstrokeopacity — Визначає прозорість обведення контурів об'єкта

### Опис

```methodsynopsis
public GmagickDraw::setstrokeopacity(float $stroke_opacity): GmagickDraw
```

Визначає прозорість обведення контурів об'єкта.

### Список параметрів

`stroke_opacity`

Прозорість обведення. При значенні 1.0 – непрозора.

### Значення, що повертаються

Об'єкт [GmagickDraw](class.gmagickdraw.html)
