# statsharmonicmean

(PECL stats >= 1.0.0)

statsharmonicmean — Повертає середній гармонійний масив значень

### Опис

```methodsynopsis
stats_harmonic_mean(array $a): number
```

Повертає середній гармонійний масив значень `a`

### Список параметрів

`a`

Масив значень

### Значення, що повертаються

Повертає середній гармонійний масив значень `a` або **`false`**, якщо `a` порожній чи не є масивом.
