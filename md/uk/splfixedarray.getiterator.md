# SplFixedArray::getIterator

(PHP 8)

SplFixedArray::getIterator - Отримує ітератор для переходу по масиву

### Опис

```methodsynopsis
public SplFixedArray::getIterator(): Iterator
```

Повертає ітератор для переходу по масиву

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Примірник об'єкта, що реалізує [Iterator](class.iterator.md)для прохождения по массиву.
