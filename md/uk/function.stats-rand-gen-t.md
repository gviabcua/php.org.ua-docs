# statsrandгент

(PECL stats >= 1.0.0)

statsrandгенt - Генерує одне випадкове відхилення від t-розподілу

### Опис

```methodsynopsis
stats_rand_gen_t(float $df): float
```

Повертає випадкове відхилення від t-розподілу зі ступенями свободи, `df`

### Список параметрів

`df`

Ступені свободи

### Значення, що повертаються

Випадкове відхилення
