# variantset

(PHP 5, PHP 7, PHP 8)

variantset — Присвоєння нового значення об'єкту варіанта

### Опис

```methodsynopsis
variant_set(variant $variant, mixed $value): void
```

Перетворює `value` у варіант і надає його об'єкту `variant`; при цьому новий об'єкт не створюється і старе значення `variant` пропадає.

### Список параметрів

`variant`

Різновид.

`value`

### Значення, що повертаються

Функція не повертає значення після виконання.
