# spl\_object\_hash

(PHP 5 >= 5.2.0, PHP 7, PHP 8)

spl\_object\_hash — Повертає хеш-ідентифікатор об'єкту

### Опис

```methodsynopsis
spl_object_hash(object $object): string
```

Функція повертає унікальний ідентифікатор заданого об'єкта. Цей ідентифікатор можна використовувати як хеш-ключ для зберігання об'єктів або для ідентифікації об'єктів, доки об'єкт не буде знищений. Як тільки об'єкт буде знищений, його хеш може бути повторно використано інших об'єктів. Поведінка схожа на функцію [spl\_object\_id()](function.spl-object-id.md)

### Список параметрів

`object`

Будь-який об'єкт.

### Значення, що повертаються

Унікальний рядок символів для кожного об'єкта, що існує на даний момент, і завжди однаковий для кожного конкретного об'єкта.

### Приклади

**Пример #1 Пример использования**spl\_object\_hash()\*\*\*\*

```php
<?php
$id = spl_object_hash($object);
$storage[$id] = $object;
?>
```

### Примітки

> **Зауваження** :
> 
> Після знищення об'єкта його хеш-ідентифікатор може бути використаний для інших об'єктів.

> **Зауваження** :
> 
> Хеші об'єктів слід порівнювати на ідентичність за допомогою `===`и`!==`оскільки повертається хеш може бути [числовим рядком](language.types.numeric-strings.md)Например,`0000000000000e600000000000000000`
