# splobjecthash

(PHP 5> = 5.2.0, PHP 7, PHP 8)

splobjecthash — Повертає хеш-ідентифікатор об'єкту

### Опис

```methodsynopsis
spl_object_hash(object $object): string
```

Функція повертає унікальний ідентифікатор заданого об'єкта. Цей ідентифікатор можна використовувати як хеш-ключ для зберігання об'єктів або для ідентифікації об'єктів, доки об'єкт не буде знищений. Як тільки об'єкт буде знищений, його хеш може бути повторно використано інших об'єктів. Поведінка схожа на функцію [splobjectid()](function.spl-object-id.html)

### Список параметрів

`object`

Будь-який об'єкт.

### Значення, що повертаються

Унікальний рядок символів для кожного об'єкта, що існує на даний момент, і завжди однаковий для кожного конкретного об'єкта.

### Приклади

**Приклад #1 Приклад використання **splobjecthash()****

```php
<?php
$id = spl_object_hash($object);
$storage[$id] = $object;
?>
```

### Примітки

> **Зауваження**
> 
> Після знищення об'єкта його хеш-ідентифікатор може бути використаний для інших об'єктів.
