# DOMChildNode::remove

(PHP 8)

DOMChildNode::remove — Видаляє вузол.

### Опис

```methodsynopsis
public DOMChildNode::remove(): void
```

Видаляє вузол.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Функція не повертає значення після виконання.
