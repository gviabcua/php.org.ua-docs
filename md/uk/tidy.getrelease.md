# tidy::getRelease

# tidy\_get\_release

(PHP 5, PHP 7, PHP 8, PECL tidy >= 0.5.2)

tidy::getRelease -- tidy\_get\_release — Отримати дату випуску (версію) для бібліотеки Tidy

### Опис

Об'єктно-орієнтований стиль

```methodsynopsis
public tidy::getRelease(): string
```

Процедурний стиль

```methodsynopsis
tidy_get_release(): string
```

Отримує дату випуску для бібліотеки Tidy.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає рядок з датою релізу бібліотеки Tidy, яка може бути `'unknown'`
