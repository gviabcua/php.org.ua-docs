# sinh

(PHP 4> = 4.1.0, PHP 5, PHP 7, PHP 8)

sinh - Гіперболічний синус

### Опис

```methodsynopsis
sinh(float $num): float
```

Повертає гіперболічний синус `num`, що визначається як `(exp(arg) - exp(-arg))/2`

### Список параметрів

`num`

Вхідне значення

### Значення, що повертаються

Гіперболічний синус `num`
