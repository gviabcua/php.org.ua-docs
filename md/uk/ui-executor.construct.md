# ОЙExecutor::construct

(UI 2.0.0)

ОЙExecutor::construct — Створити новий об'єкт Executor

### Опис

public **ОЙExecutor::construct**

public **ОЙExecutor::construct**(int `$microseconds`

### Список параметрів

`microseconds`

Мікросекунди між виконаннями
