# SolrQuery::setTermsIncludeLowerBound

(PECL solr >= 0.9.2)

SolrQuery::setTermsIncludeLowerBound — Включає нижню межу виразу в набір результатів

### Опис

```methodsynopsis
public SolrQuery::setTermsIncludeLowerBound(bool $flag): SolrQuery
```

Включає нижню межу вираження у набір результатів

### Список параметрів

`flag`

Включає нижню межу вираження у набір результатів

### Значення, що повертаються

Повертає поточний об'єкт SolrQuery, якщо використовується значення, що повертається.
