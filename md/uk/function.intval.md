# intval

(PHP 4, PHP 5, PHP 7, PHP 8)

intval — Повертає ціле значення змінної

### Опис

```methodsynopsis
intval(mixed $value, int $base = 10): int
```

Повертає ціле (int) значення змінної `value`, використовуючи вказану основу системи числення `base` для перетворення (за умовчанням основа дорівнює 10) . **intval()** не можна використовувати з об'єктами, спроба це зробити викликає помилку рівня **`E_NOTICE`** та поверне значення 1.

### Список параметрів

`value`

Скалярне значення для конвертації в ціле

`base`

Підстава системи числення для перетворення

> **Зауваження**
> 
> Якщо `base` дорівнює 0, основа системи числення визначається форматом `value`
> 
> -   якщо рядок включає префікс "0x" (або "0X"), підстава вибирається рівним 16 (шістнадцятковим);
> -   інакше, якщо рядок починається з "0", основа дорівнює 8 (вісімкова);
> -   інакше, підстава вибирається рівним 10 (десятковим).

### Значення, що повертаються

Ціле значення змінної `value` у разі успішного виконання, або 0 у разі невдачі. Для порожніх масивів повертається 0, для порожніх повертається 1.

Максимальне значення залежить від операційної системи. У 32-х бітних системах максимальне знакове ціле лежить у діапазоні від -2147483648 до 2147483647. Так, наприклад, у такій системі, `intval('1000000000000')` поверне 2147483647. Максимальне знакове ціле значення для 64-х бітних систем – 9223372036854775807.

Для рядків функція швидше за все поверне 0, хоча залежить від першого символу в рядку. Застосовуються загальні правила [преобразований в целые](language.types.integer.html#language.types.integer.casting)

### Приклади

**Приклад #1 Приклади використання **intval()****

Наступні приклади наведені для 32 бітної системи.

```php
<?php
echo intval(42);                      // 42
echo intval(4.2);                     // 4
echo intval('42');                    // 42
echo intval('+42');                   // 42
echo intval('-42');                   // -42
echo intval(042);                     // 34
echo intval('042');                   // 42
echo intval(1e10);                    // 1410065408
echo intval('1e10');                  // 1
echo intval(0x1A);                    // 26
echo intval(42000000);                // 42000000
echo intval(420000000000000000000);   // 0
echo intval('420000000000000000000'); // 2147483647
echo intval(42, 8);                   // 42
echo intval('42', 8);                 // 34
echo intval(array());                 // 0
echo intval(array('foo', 'bar'));     // 1
echo intval(false);                   // 0
echo intval(true);                    // 1
?>
```

### Примітки

> **Зауваження**
> 
> Параметр `base` застосовується лише в тому випадку, якщо параметр `value` є рядком.
