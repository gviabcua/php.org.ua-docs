# псshow

(PECL ps >= 1.1.0)

псshow — Виводить текст

### Опис

```methodsynopsis
ps_show(resource $psdoc, string $text): bool
```

Виводить текст у поточній позиції. Положення тексту можна встановити, erfpfd координати X та Y у значеннях `textx` і `texty` за допомогою функції [ps\_set\_value()](function.ps-set-value.html). Функція видасть помилку, якщо шрифт не було встановлено раніше за допомогою [ps\_setfont()](function.ps-setfont.html)

Функція **псshow()** оцінює наступні параметри та значення, встановлені [ps\_set\_parameter()](function.ps-set-parameter.html) і [ps\_set\_value()](function.ps-set-value.html)

charspacing (значення)

Відстань між двома послідовними гліфами. Якщо це значення не дорівнює нулю, всі лігатури будуть дозволені. Допускаються значення менше нуля.

kerning (параметр)

Установка параметра "false" відключить кернінг. За замовчуванням кернінг увімкнено.

ligatures (параметр)

Встановлення параметра "false" вимкне використання лігатур. За замовчуванням увімкнено лігатуру.

underline (параметр)

Налаштування параметра "true" увімкне підкреслення. За промовчанням підкреслення вимкнено.

overline (параметр)

Установка параметра "true" увімкне накладення. Стандартне накладення вимкнено.

strikeout (параметр)

Налаштування параметра "true" увімкне закреслювання. За замовчуванням закреслення вимкнено.

### Список параметрів

`psdoc`

Ідентифікатор ресурсу файлу postscript, повернутий функцією [ps\_new()](function.ps-new.html)

`text`

Текст для виведення.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.
