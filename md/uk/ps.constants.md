# Обумовлені константи

Наведені нижче константи визначені даним модулем і можуть бути доступні тільки в тому випадку, якщо PHP був зібраний за допомогою цього модуля або в тому випадку, якщо даний модуль був динамічно завантажений під час виконання.

Дві наступні таблиці містять усі константи, що визначаються модулем PS.

**Константи для закінчення ліній**

| Константа | Значение |
| --- | --- |
| псLINECAPBUTT | Загострені |
| псLINECAPROUND | Закруглені |
| псLINECAPSQUARED | Прямокутні |

**Константи, що визначають вид перетину ліній**

| Константа | Значение |
| --- | --- |
| псLINEJOINMITER | Прямокутне з'єднання |
| псLINEJOINROUND | Заокруглене |
| псLINEJOINBEVEL | Скошене |
