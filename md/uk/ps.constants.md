# Обумовлені константи

Наведені нижче константи визначені цим модулем і доступні або в збірках PHP з підтримкою цього модуля, або коли модуль динамічно завантажений під час виконання коду.

Дві наступні таблиці містять усі константи, що визначаються модулем PS.

**Константи для кінцівок ліній**

| Константа | Значение |
| --- | --- |
| ps\_LINECAP\_BUTT | Загострені |
| ps\_LINECAP\_ROUND | Закруглені |
| ps\_LINECAP\_SQUARED | Прямокутні |

**Константи, що визначають вид перетину ліній**

| Константа | Значение |
| --- | --- |
| ps\_LINEJOIN\_MITER | Прямокутне з'єднання |
| ps\_LINEJOIN\_ROUND | Заокруглене |
| ps\_LINEJOIN\_BEVEL | Скошене |
