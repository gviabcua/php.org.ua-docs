# ZMQ::\_\_construct

(PECL zmq >= 0.5.0)

ZMQ::\_\_construct — Конструктор ZMQ

### Опис

```methodsynopsis
private ZMQ::__construct()
```

Приватний конструктор для запобігання прямій ініціалізації. Цей клас містить константи для модуля ZMQ.

### Список параметрів

Ця функція не має параметрів.
