# SVM::train

(PECL svm >= 0.1.0)

SVM::train — Створити SVMModel на основі навчальних даних

### Опис

```methodsynopsis
public svm::train(array $problem, array $weights = ?): SVMModel
```

Створює SVMModel на основі навчальних даних.

### Список параметрів

`problem`

Є три шляхи надання навчальних даних: файл, потік та масив. Якщо дані надаються за допомогою файлу або потоку, то на кожному рядку повинен міститися один навчальний приклад, відформатований наступним чином: на початку має бути ціле число (зазвичай 1 або -1), це число позначається терміном "клас", а слідом за ним перерахування пар ознака: значення в порядку збільшення ознаки. Ознаки мають бути цілими числами, які значення раціональними, зазвичай, на діапазоні 0-1. У разі використання масиву дані повинні бути представлені у вигляді масиву масивів, в якому кожен вкладений масив повинен першим елементом містити клас, а всі наступні елементи містити пари "ознака" => "значення".

`weights`

Необов'язковий набір вагових коефіцієнтів для різних класів, які допомагають працювати з незбалансованими навчальними наборами. Наприклад, якщо у нас два класи, 1 і -1, і навчальних прикладів для класу -1 дуже більше, то вага для -1 треба виставити рівним 0.5. Вага має бути в діапазоні від 0 до 1.

### Значення, що повертаються

Повертає об'єкт класу SVMModel. У разі виникнення помилки викидає SVMException
