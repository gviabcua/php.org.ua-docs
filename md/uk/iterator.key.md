# Iterator::key

(PHP 5, PHP 7, PHP 8)

Iterator::key — Повертає ключ поточного елемента

### Опис

```methodsynopsis
public Iterator::key(): mixed
```

Повертає ключ поточного елемента.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає скалярне значення (scalar) у разі успішного виконання або **`null`** у разі виникнення помилки.

### Помилки

Створює помилку рівня **`E_NOTICE`** у разі виникнення помилки.
