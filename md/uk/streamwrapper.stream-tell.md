# streamWrapper::stream\_tell

(PHP 4 >= 4.3.2, PHP 5, PHP 7, PHP 8)

streamWrapper::stream\_tell — Визначення поточної позиції потоку

### Опис

```methodsynopsis
public streamWrapper::stream_tell(): int
```

Цей метод викликається у процесі виконання [fseek()](function.fseek.md) визначення поточної позиції.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повинен повертати поточну позицію потоку.
