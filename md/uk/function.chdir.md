# chdir

(PHP 4, PHP 5, PHP 7, PHP 8)

chdir — Змінює каталог

### Опис

```methodsynopsis
chdir(string $directory): bool
```

Змінює поточний каталог PHP на вказаний як параметр `directory`

### Список параметрів

`directory`

Новий поточний каталог

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.

### Помилки

У разі невдалого виконання видає помилку рівня **`E_WARNING`**

### Приклади

**Приклад #1 Приклад використання **chdir()****

```php
<?php

// текущий каталог
echo getcwd() . "\n";

chdir('public_html');

// текущий каталог
echo getcwd() . "\n";

?>
```

Результатом виконання цього прикладу буде щось подібне:

```
/home/vincent
/home/vincent/public_html
```

### Примітки

**Застереження**

Якщо PHP-інтерпретатор зібраний за допомогою ZTS (Zend Thread Safety), будь-які зміни в поточному каталозі, зроблені за допомогою **chdir()**, не буде виявлено операційною системою. Всі вбудовані функції PHP, як і раніше, враховуватимуть зміни у поточному каталозі; це не відноситься до викликаних функцій зовнішніх бібліотек, підключених через [FFI](book.ffi.html). Використовуйте **php -i** або вбудовану константу **`PHP_ZTS`**, щоб дізнатися, чи зібрано PHP з ZTS.
