# Imagick::getImageGamma

(PECL imagick 2, PECL imagick 3)

Imagick::getImageGamma — Повертає гаму зображення

### Опис

```methodsynopsis
public Imagick::getImageGamma(): float
```

Повертає гаму зображення.

### Список параметрів

Ця функція не має параметрів.
