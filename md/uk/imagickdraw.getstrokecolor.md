# ImagickDraw::getStrokeColor

(PECL imagick 2, PECL imagick 3)

ImagickDraw::getStrokeColor — Повертає колір для обведення контурів об'єкта.

### Опис

```methodsynopsis
public ImagickDraw::getStrokeColor(): ImagickPixel
```

**Увага**

Функція поки що не документована; для знайомства доступний лише перелік аргументів.

Повертає колір для обведення контурів об'єкта.

### Список параметрів

`stroke_color`

### Значення, що повертаються

Повертає об'єкт [ImagickPixel](class.imagickpixel.md), що описує колір.
