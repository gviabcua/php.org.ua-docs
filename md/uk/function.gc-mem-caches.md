# gc\_mem\_caches

(PHP 7, PHP 8)

gc\_mem\_caches — Звільняє пам'ять, яку використовує менеджер пам'яті Zend Engine

### Опис

```methodsynopsis
gc_mem_caches(): int
```

Звільняє пам'ять, яку використовує менеджер пам'яті Zend Engine.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає кількість звільнених байт.
