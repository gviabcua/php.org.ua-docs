# trader\_get\_compat

(PECL trader >= 0.2.2)

trader\_get\_compat — Отримує режим сумісності

### Опис

```methodsynopsis
trader_get_compat(): int
```

Отримує режим сумісності, який впливає на спосіб виконання обчислень усіма функціями модуля.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає ідентифікатор режиму сумісності, який можна визначити за константами [TRADER\_COMPATIBILITY\_\*](trader.constants.md)
