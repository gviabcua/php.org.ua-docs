# tradergetcompat

(PECL trader >= 0.2.2)

tradergetcompat — Отримує режим сумісності

### Опис

```methodsynopsis
trader_get_compat(): int
```

Отримує режим сумісності, який впливає на спосіб виконання обчислень усіма функціями модуля.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає ідентифікатор режиму сумісності, який можна визначити за константами [TRADERCOMPATIBILITY](trader.constants.html)
