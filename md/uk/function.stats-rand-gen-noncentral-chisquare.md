# stats\_rand\_gen\_noncentral\_chisquare

(PECL stats >= 1.0.0)

stats\_rand\_gen\_noncentral\_chisquare - Генерує випадкове відхилення нецентрального розподілу хі-квадрат

### Опис

```methodsynopsis
stats_rand_gen_noncentral_chisquare(float $df, float $xnonc): float
```

Повертає випадкове відхилення нецентрального розподілу хі-квадрат з `df` ступенями свободи та параметром нецентральності `xnonc`

### Список параметрів

`df`

Ступені свободи

`xnonc`

Параметр нецентральності

### Значення, що повертаються

Випадкове відхилення
