# GmagickDraw::point

(PECL gmagick >= Unknown)

GmagickDraw::point — Малює точку

### Опис

```methodsynopsis
public GmagickDraw::point(float $x, float $y): GmagickDraw
```

Малює точку, використовуючи поточний колір та товщину обведення за вказаними координатами.

### Список параметрів

`x`

Координата x.

`y`

Координата y.
