# Ev::verify

(PECL ev >= 0.2.0)

Ev::verify - Здійснює внутрішню перевірку цілісності (для налагодження)

### Опис

```methodsynopsis
final
   public
   static
   Ev::verify(): void
```

Здійснює внутрішню перевірку цілісності (для налагодження *libev*) і припиняє виконання програми, якщо виявлено помилки.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Функція не повертає значення після виконання.
