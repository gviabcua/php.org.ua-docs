# мбcheckencoding

(PHP 4> = 4.4.3, PHP 5> = 5.1.3, PHP 7, PHP 8)

мбcheckencoding — Перевіряє, що кодування рядків вибрано правильно

### Опис

```methodsynopsis
mb_check_encoding(array|string|null $value = null, ?string $encoding = null): bool
```

Перевіряє, що для потоку даних підходить вибране кодування. Якщо `value` є масивом (array), всі ключі та значення перевіряються рекурсивно. Функція може бути корисною для запобігання так званому "Атаки неправильним кодуванням".

### Список параметрів

`value`

Потік даних чи масив (array) для перевірки. Якщо не встановлено, функція перевірить усі вхідні дані з початку запиту.

`encoding`

Очікуване кодування.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.

### список змін

| Версия | Описание                                                                                      |
|--------|-----------------------------------------------------------------------------------------------|
|        | Параметри `value` і `encoding` можуть набувати значення **`null`**                            |
|        | Функція тепер також приймає масив (array) `value`. Раніше підтримувалися лише рядки (string). |
