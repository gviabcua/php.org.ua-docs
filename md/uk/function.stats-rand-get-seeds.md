# stats\_rand\_get\_seeds

(PECL stats >= 1.0.0)

stats\_rand\_get\_seeds — Отримує початкові значення генератора випадкових чисел

### Опис

```methodsynopsis
stats_rand_get_seeds(): array
```

Повертає початкові значення генератора випадкових чисел.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає масив із двох цілих чисел.
