# Imagick::clipImagePath

(PECL imagick 2> = 2.3.0, PECL imagick 3)

Imagick::clipImagePath — Опис

### Опис

```methodsynopsis
public Imagick::clipImagePath(string $pathname, string $inside): void
```

Кліпи вздовж іменованих шляхів із профілю 8BIM, якщо вони є. Пізніші операції набувають чинності всередині шляху. Ідентифікатор може бути числом, якщо передує # для роботи з пронумерованим шляхом, наприклад, вкажіть "#1", щоб використовувати перший шлях.

**Увага**

На цей час ця функція ще була документована; для ознайомлення доступний лише перелік аргументів.

### Список параметрів

`pathname`

`inside`
