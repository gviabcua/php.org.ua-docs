# mb\_strcut

(PHP 4 >= 4.0.6, PHP 5, PHP 7, PHP 8)

mb\_strcut — Отримує частину рядка

### Опис

```methodsynopsis
mb_strcut(    string $string,    int $start,    ?int $length = null,    ?string $encoding = null): string
```

Функция**mb\_strcut()** вирізає підрядок з рядка також, як функція [mb\_substr()](function.mb-substr.md)але оперує байтами замість символів. Якщо початок частини, що вирізається, потрапляє між байтами одного символу, функція виріже підрядок, починаючи з першого байта цього символу. Це суттєва відмінність від функції [substr()](function.substr.md), яка просто виріже підрядок, починаючи з середини символу, і цим порушить послідовність байтів у рядку.

### Список параметрів

`string`

Обрізний рядок.

`start`

Если значение параметра`start` невід'ємний, повернутий рядок почнеться з позиції `start`в*байтах*от начала строки`string`з нуля. Наприклад, у рядку «`abcdef`» байт у позиції - Це «`a`», байт у позиції `c`" і т.д.

Если значение параметра`start` негативний рядок, що повертається, буде починатися з байта `start` зі зворотним відліком від кінця рядка `string`. Однак, якщо величина негативного `start` перевищує довжину рядка, частина, що повертається, почнеться з початку параметра `string`

`length`

Длина подстроки*у байтах*. Якщо не вказано, то передається значення `NULL` і вирізають усі байти до кінця рядка.

Если значение параметра`length` негативний рядок, що повертається, буде закінчуватися байтом `length` зі зворотним відліком від кінця рядка `string`. Однак, якщо величина негативного значення `length` перевищує кількість символів після позиції `start`, буде повернуто порожній рядок.

`encoding`

Параметр`encoding` - Це кодування символів. Якщо він опущений або дорівнює **`null`**, для нього буде встановлено внутрішнє кодування символів.

### Значення, що повертаються

Повертає частину рядка `string`, заданную аргументами`start`и`length`
