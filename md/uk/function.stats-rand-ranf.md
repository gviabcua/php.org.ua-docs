# stats\_rand\_ranf

(PECL stats >= 1.0.0)

stats\_rand\_ranf - Генерує випадкове число з плаваючою точкою від 0 до 1

### Опис

```methodsynopsis
stats_rand_ranf(): float
```

Повертає випадкове число з плаваючою точкою з рівномірного розподілу від 0 (за винятком) до 1 (за винятком).

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Випадкове число з плаваючою точкою
