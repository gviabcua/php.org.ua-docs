# ParleRLexer::advance

(PECL parle >= 0.5.1)

ParleRLexer::advance - Обробка наступного правила лексера

### Опис

```methodsynopsis
public Parle\RLexer::advance(): void
```

Обробляє таке правило та готує отримані дані токена.
