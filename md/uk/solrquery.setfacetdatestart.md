# SolrQuery::setFacetDateStart

(PECL solr >= 0.9.2)

SolrQuery::setFacetDateStart — Відповідає facet.date.start

### Опис

```methodsynopsis
public SolrQuery::setFacetDateStart(string $value, string $field_override = ?): SolrQuery
```

Відповідає facet.date.start

### Список параметрів

`value`

Дивіться facet.date.start

`field_override`

Ім'я поля.

### Значення, що повертаються

Повертає поточний об'єкт SolrQuery, якщо використовується значення, що повертається.
