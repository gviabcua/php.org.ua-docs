# Обумовлені константи

Наведені нижче константи визначені цим модулем і доступні або в збірках PHP з підтримкою цього модуля, або коли модуль динамічно завантажений під час виконання коду.

**`DBA_LMDB_USE_SUB_DIR`**(int)

Прапор LMDB Driver, що дозволяє створення підкаталогу для файлів бази даних. Доступно, починаючи з PHP 8.2.0.

**`DBA_LMDB_NO_SUB_DIR`**(int)

Прапор LMDB Driver забороняє створення підкаталогу для файлів бази даних. Доступно, починаючи з PHP 8.2.0.
