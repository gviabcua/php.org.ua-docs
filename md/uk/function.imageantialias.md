# imageantialias

(PHP 4> = 4.3.2, PHP 5, PHP 7, PHP 8)

imageantialias — Чи потрібно застосовувати функції згладжування чи ні

### Опис

```methodsynopsis
imageantialias(GdImage $image, bool $enable): bool
```

Задіює механізми згладжування при малюванні ліній та замкнутих багатокутників. Не підтримує компоненти альфа. Використовує операції прямого змішування кольорів. Працює тільки з truecolor-зображеннями.

Стилі та товщина не підтримуються.

Використання згладжених примітивів разом із прозорим тлом може дати несподіваний результат. Метод змішування використовує колір тла нарівні з будь-яким іншим кольором. Відсутність підтримки компонента альфа не дозволяє застосовувати заснований на ньому метод згладжування.

### Список параметрів

`image`

Об'єкт [GdImage](class.gdimage.md), що повертається однією з функцій створення зображень, наприклад, такий як [imagecreatetruecolor()](function.imagecreatetruecolor.md)

`enable`

Увімкнути згладжування чи ні.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.

### список змін

| Версия | Описание |
| --- | --- |
|  | `image` тепер чекає екземпляр [GdImage](class.gdimage.md); раніше очікували ресурс (resource). |
|  | Функція **imageantialias()** доступна без обмежень. Раніше вона була доступна тільки якщо PHP був зібраний з використанням бібліотеки GD, що йде з ним в комплекті. |

### Приклади

**Приклад #1 Порівняння двох ліній, одна з яких малюється із включеним згладжуванням**

```php
<?php
// Создание двух изображений
$aa = imagecreatetruecolor(400, 100);
$normal = imagecreatetruecolor(200, 100);

// Включение сглаживания для одного из них
imageantialias($aa, true);

// Задание цветов
$red = imagecolorallocate($normal, 255, 0, 0);
$red_aa = imagecolorallocate($aa, 255, 0, 0);

// Рисование линий
imageline($normal, 0, 0, 200, 100, $red);
imageline($aa, 0, 0, 200, 100, $red_aa);

// Размещение изображений рядом для сравнения (АА: слева, Обычное: справа)
imagecopymerge($aa, $normal, 200, 0, 0, 0, 200, 100, 100);

// Вывод результата
header('Content-type: image/png');

imagepng($aa);
imagedestroy($aa);
imagedestroy($normal);
?>
```
