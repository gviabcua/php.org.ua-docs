# Yac::flush

(PECL yac >= 1.0.0)

Yac::flush — Очищає кеш

### Опис

```methodsynopsis
public Yac::flush(): bool
```

Видаляє всі значення кеші.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Логічне значення завжди повертає true
