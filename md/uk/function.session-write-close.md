# sessionwriteclose

(PHP 4> = 4.0.4, PHP 5, PHP 7, PHP 8)

sessionwriteclose - Записує дані сесії і завершує її

### Опис

```methodsynopsis
session_write_close(): bool
```

Завершує поточну сесію та зберігає її дані.

Ці сесії зазвичай зберігаються після завершення виконання скрипту без необхідності виклику **sessionwriteclose()**. Оскільки дані сесії заблоковані для запобігання одночасному конкуруючого запису, тільки один скрипт може працювати з сесією будь-якої миті часу. Через це блокування, при використанні кадрів разом із сесіями, кадри будуть завантажуватися по черзі один за одним. Можна зменшити час завантаження всіх кадрів шляхом завершення сесії одразу, як тільки будуть виконані всі необхідні зміни змінних сесії.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.
