# AppendIterator::next

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

AppendIterator::next — Переходить до наступного елемента

### Опис

```methodsynopsis
public AppendIterator::next(): void
```

Переходить до наступного елемента. Якщо елемент це інший ітератор, то перемотує цей ітератор.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Функція не повертає значення після виконання.
