# Zookeeper::close

(PECL zookeeper >= 0.5.0)

Zookeeper::close — Закриває обробник zookeeper та звільняє будь-які ресурси

### Опис

```methodsynopsis
public
   Zookeeper::close(): void
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Функція не повертає значення після виконання.

### Помилки

Метод генерує [ZookeeperException](class.zookeeperexception.html) та його похідні при закритті неініціалізованого екземпляра.
