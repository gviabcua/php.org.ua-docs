# stats\_absolute\_deviation

(PECL stats >= 1.0.0)

stats\_absolute\_deviation - Повертає абсолютне відхилення масиву значень

### Опис

```methodsynopsis
stats_absolute_deviation(array $a): float
```

Возвращает абсолютное отклонение значений в`a`

### Список параметрів

`a`

Вхідний масив

### Значення, що повертаються

Возвращает абсолютное отклонение значений в`a`или\*\*`false`\*\*, якщо `a` порожній чи не є масивом.
