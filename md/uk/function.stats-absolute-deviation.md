# statsabsolutedeviation

(PECL stats >= 1.0.0)

statsabsolutedeviation - Повертає абсолютне відхилення масиву значень

### Опис

```methodsynopsis
stats_absolute_deviation(array $a): float
```

Повертає абсолютне відхилення значень у `a`

### Список параметрів

`a`

Вхідний масив

### Значення, що повертаються

Повертає абсолютне відхилення значень у `a` або **`false`**, якщо `a` порожній чи не є масивом.
