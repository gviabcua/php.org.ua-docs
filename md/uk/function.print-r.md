# printр

(PHP 4, PHP 5, PHP 7, PHP 8)

printr — Виводить інформацію, що читає легко, про змінну

### Опис

```methodsynopsis
print_r(mixed $value, bool $return = false): string|bool
```

**printr()** виводить інформацію про змінну у легкочитаному вигляді.

**printr()** [vardump()](function.var-dump.html) і [varexport()](function.var-export.md) можуть також відображати захищені та закриті атрибути об'єктів. Статичні елементи класу не відображатимуться.

### Список параметрів

`value`

Вираз для виведення на екран.

`return`

Якщо ви бажаєте перехопити висновок **printr()**, використовуйте параметр `return`. Якщо його значення дорівнює **`true`**, то **printr()** поверне інформацію замість виведення у браузер.

### Значення, що повертаються

Якщо функцію передаються значення типів string, int або float, буде надруковано саме значення. Якщо передається масив array, значення буде надруковано у форматі, що показує ключі та елементи масиву. Аналогічний формат виводу використовується для об'єктів.

Якщо параметр `return` встановлений в **`true`**, дана функція поверне рядок (string). В іншому випадку значення, що повертається, буде одно **`true`**

### Приклади

**Приклад #1 Приклад використання **printr()****

```php
<pre>
<?php
$a = array ('a' => 'apple', 'b' => 'banana', 'c' => array ('x', 'y', 'z'));
print_r ($a);
?>
</pre>
```

Результат виконання цього прикладу:

```
<pre>
Array
(
    [a] => apple
    [b] => banana
    [c] => Array
        (
            [0] => x
            [1] => y
            [2] => z
        )
)
</pre>
```

**Приклад #2 Приклад використання параметра `return`**

```php
<?php
$b = array ('m' => 'monkey', 'foo' => 'bar', 'x' => array ('x', 'y', 'z'));
$results = print_r($b, true); // $results теперь содержит вывод print_r
?>
```

### Примітки

> **Зауваження**
> 
> Коли використовується параметр `return`, функція використовує внутрішню буферизацію виведення до PHP 7.1.0, тому її не можна використовувати всередині callback-функції [проstart()](function.ob-start.md)
