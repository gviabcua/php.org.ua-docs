# Безпека сесій

## Зміст

-   [Базові засади управління сесіями](features.session.security.management.md)
-   [INI-налаштування безпеки сесій](session.security.ini.md)

Зовнішні посилання: [» Атака "Фіксація сесії"](http://www.acros.si/papers/session_fixation.pdf)

Робота з HTTP-сесіями є основою веб-безпеки . *Повинні* бути застосовані всі можливі заходи для забезпечення безпеки сесії. Розробник повинен включати та використовувати відповідні налаштування відповідним чином.
