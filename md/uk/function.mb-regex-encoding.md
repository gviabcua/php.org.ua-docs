# mb\_regex\_encoding

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

mb\_regex\_encoding — Встановлює/отримує кодування символів для багатобайтового регулярного виразу

### Опис

```methodsynopsis
mb_regex_encoding(?string $encoding = null): string|bool
```

Встановлює/отримує кодування символів для регулярного виразу однобайтового.

### Список параметрів

`encoding`

Параметр`encoding` - Це кодування символів. Якщо він опущений або дорівнює **`null`**, для нього буде встановлено внутрішнє кодування символів.

### Значення, що повертаються

Если задан параметр`encoding`, то Повертає **`true`** у разі успішного виконання або \*\*`false`\*\*в случае возникновения ошибки. Если кодировка задана, то внутренняя кодировка символов не изменяется. Если параметр`encoding` не вказано, то повертається ім'я поточного кодування символів для регулярних багатобайтових виразів.
