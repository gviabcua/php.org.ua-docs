## Оператори, що працюють із масивами

**Оператори, що працюють із масивами**

| Пример | Название | Результат |
| --- | --- | --- |
| $a + $b | Об'єднання | Об'єднання масиву $a та масиву $b. |
| $a == $b | Рівно | **`true`** у випадку, якщо $a і $b містять одні й самі пари ключ/значення. |
| $a === $b | Тотожно | **`true`** у випадку, якщо $a і $b містять одні й ті ж пари ключ/значення в тому самому порядку і того ж типу. |
| $a != $b | Не дорівнює | \*\*`true`\*\*якщо масив $a не дорівнює масиву $b. |
| $a <> $b | Не дорівнює | \*\*`true`\*\*якщо масив $a не дорівнює масиву $b. |
| $a !== $b | Тотожно не одно | **`true`**, якщо масив $a не дорівнює тотожному масиву $b. |

Оператор `+` повертає лівий масив, до якого було приєднано правий масив. Для ключів, які існують в обох масивах, будуть використані значення з лівого масиву, а відповідні елементи з правого масиву будуть проігноровані.

```php
<?php
$a = array("a" => "apple", "b" => "banana");
$b = array("a" => "pear", "b" => "strawberry", "c" => "cherry");

$c = $a + $b; // Объединение $a и $b
echo "Объединение \$a и \$b: \n";
var_dump($c);

$c = $b + $a; // Объединение $b и $a
echo "Объединение \$b и \$a: \n";
var_dump($c);

$a += $b; // Объединение $a += $b, это $a и $b
echo "Объединение \$a += \$b: \n";
var_dump($a);
?>
```

Після виконання скрипт надрукує наступне:

```
Объединение $a и $b:
array(3) {
  ["a"]=>
  string(5) "apple"
  ["b"]=>
  string(6) "banana"
  ["c"]=>
  string(6) "cherry"
}
Объединение $b и $a:
array(3) {
  ["a"]=>
  string(4) "pear"
  ["b"]=>
  string(10) "strawberry"
  ["c"]=>
  string(6) "cherry"
}

Объединение $a += $b:
array(3) {
  ["a"]=>
  string(5) "apple"
  ["b"]=>
  string(6) "banana"
  ["c"]=>
  string(6) "cherry"
}
```

При порівнянні елементи масиву вважаються ідентичними, якщо збігається і ключ і відповідне йому значення.

**Приклад #1 Порівняння масивів**

```php
<?php
$a = array("apple", "banana");
$b = array(1 => "banana", "0" => "apple");

var_dump($a == $b); // bool(true)
var_dump($a === $b); // bool(false)
?>
```
