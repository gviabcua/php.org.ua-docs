## Визначення просторів імен

(PHP 5> = 5.3.0, PHP 7, PHP 8)

Хоча будь-який коректний PHP-код може знаходитися всередині простору імен, тільки класи (включаючи абстрактні та трейти), інтерфейси, функції та константи залежать від нього.

Простори імен оголошуються за допомогою зарезервованого слова `namespace`. Файл, що містить простір імен, повинен містити його оголошення на початку будь-якого іншого коду, крім зарезервованого слова [declare](control-structures.declare.html)

**Приклад #1 Оголошення єдиного простору імен**

```php
<?php
namespace MyProject;

const CONNECT_OK = 1;
class Connection { /* ... */ }
function connect() { /* ... */  }

?>
```

> **Зауваження**: Абсолютні імена (тобто імена, що починаються із зворотної косою межі) не допускаються в оголошеннях простору імен, оскільки такі конструкції інтерпретуються як відносні вирази простору імен.

Тільки вираз `declare` може бути перед оголошенням простору імен для вказівки кодування файлу. Крім того, оголошення простору імен не повинен передувати не PHP-код, у тому числі зайві прогалини:

**Приклад #2 Оголошення простого простору імен**

```php
<html>
<?php
namespace MyProject; // fatal error - объявление пространства имён должно быть первым выражением в скрипте
?>
```

Крім того, на відміну від будь-якої іншої конструкції PHP, той самий простір імен можна визначати в декількох файлах, що дозволяє розподіляти їх вміст по файловій системі.
