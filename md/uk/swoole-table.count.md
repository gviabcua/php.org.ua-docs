# SwooleTable::count

(PECL swoole >= 1.9.0)

SwooleTable::count — Підраховує рядки у таблиці або підраховує всі елементи у таблиці, якщо $mode = 1

### Опис

```methodsynopsis
public Swoole\Table::count(): int
```

### Список параметрів

Ця функція не має параметрів.
