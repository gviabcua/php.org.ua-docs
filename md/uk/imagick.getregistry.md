# Imagick::getRegistry

(PECL imagick 3 >= 3.3.0)

Imagick::getRegistry — Отримує запис StringRegistry

### Опис

```methodsynopsis
public static Imagick::getRegistry(string $key): string
```

Отримує запис StringRegistry для вказаного ключа або false, якщо не встановлено.

### Список параметрів

`key`

Запис, який потрібно отримати.
