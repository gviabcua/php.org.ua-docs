# GmagickDraw::setfillopacity

(PECL gmagick >= Unknown)

GmagickDraw::setfillopacity — Встановлює непрозорість заливки

### Опис

```methodsynopsis
public GmagickDraw::setfillopacity(float $fill_opacity): GmagickDraw
```

Встановлює непрозорість при малюванні за допомогою кольору заливки або текстури заливки. Налаштування значення 1.0 зробить заливку повністю непрозорою.

### Список параметрів

`fill_opacity`

Непрозорість заливання

### Значення, що повертаються

Об'єкт [Gmagick](class.gmagick.md)
