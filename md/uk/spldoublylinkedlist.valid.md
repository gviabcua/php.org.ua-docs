# SplDoublyLinkedList::valid

(PHP 5> = 5.3.0, PHP 7, PHP 8)

SplDoublyLinkedList::valid — Перевіряє, чи містить вузли двозв'язковий список

### Опис

```methodsynopsis
public SplDoublyLinkedList::valid(): bool
```

Перевіряє, чи містить вузли двозв'язний список.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає **`true`**, якщо список містить вузли після поточного покажчика, **`false`** в іншому випадку.
