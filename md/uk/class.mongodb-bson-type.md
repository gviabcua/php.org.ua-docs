(mongodb >=1.0.0)

## Вступ

Анотація базовий інтерфейс, який не повинен бути реалізований безпосередньо.

## Огляд інтерфейсів

   }

Цей інтерфейс немає методів. Його єдине призначення – бути базовим інтерфейсом для всіх класів типів BSON.
