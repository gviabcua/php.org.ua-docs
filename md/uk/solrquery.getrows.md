# SolrQuery::getRows

(PECL solr> = 0.9.2)

SolrQuery::getRows — Повертає максимальну кількість документів

### Опис

```methodsynopsis
public SolrQuery::getRows(): int
```

Повертає максимальну кількість документів з повного набору результатів, які клієнт повертає для кожного запиту.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

У разі успішного виконання повертає ціле число та **`null`**, якщо значення не встановлено.
