# SolrQuery::setFacetDateEnd

(PECL solr >= 0.9.2)

SolrQuery::setFacetDateEnd — Відповідає facet.date.end

### Опис

```methodsynopsis
public SolrQuery::setFacetDateEnd(string $value, string $field_override = ?): SolrQuery
```

Відповідає facet.date.end

### Список параметрів

`value`

Дивіться facet.date.end

`field_override`

Ім'я поля

### Значення, що повертаються

Повертає поточний об'єкт SolrQuery, якщо використовується значення, що повертається.
