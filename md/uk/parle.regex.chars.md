## Подання символів

**Подання символів**

| Последовательность | Опис |
| --- | --- |
| \\a | Символ оповіщення, сигнал. |
| \\b | Символ повернення "Backspace". |
| \\e | Символ ESC, \\x1b. |
| \\n | Новий рядок. |
| \\r | Повернення каретки. |
| \\f | Розрив сторінки, \\x0c. |
| \\t | Горизонтальна табуляція, \\x09. |
| \\v | Вертикальна табуляція, \\x0b. |
| \\oct | Символ, заданий тризначним вісімковим кодом. |
| \\xhex | Символ, заданий шістнадцятковим кодом. |
| \\cchar | Названий керуючий символ. |
