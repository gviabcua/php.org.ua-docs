## Подання символів

**Подання символів**

| Последовательность | Описание |
| --- | --- |
| а | Символ оповіщення, сигнал. |
| в | Символ повернення "Backspace". |
| е | Символ ESC, x1b. |
| н | Новий рядок. |
| р | Повернення каретки. |
| ф | Розрив сторінки, x0c. |
| т | Горизонтальна табуляція, x09. |
| в | Вертикальна табуляція, x0b. |
| oct | Символ, заданий тризначним вісімковим кодом. |
| xhex | Символ, заданий шістнадцятковим кодом. |
| cchar | Названий керуючий символ. |
