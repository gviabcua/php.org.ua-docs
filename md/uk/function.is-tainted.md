# ісtainted

(PECL taint >=0.1.0)

ісtainted — Перевіряє, чи зіпсований рядок

### Опис

```methodsynopsis
is_tainted(string $string): bool
```

Перевіряє, чи рядок зіпсований

### Список параметрів

`string`

### Значення, що повертаються

Повертає **`true`**, якщо рядок зіпсовано або **`false`** в іншому випадку.
