# ReflectionFunction::getClosure

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

ReflectionFunction::getClosure — Повертає динамічно створене замикання функції

### Опис

```methodsynopsis
public ReflectionFunction::getClosure(): Closure
```

Отримує динамічно створене замикання функції.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає щойно створене замикання ([Closure](class.closure.md)
