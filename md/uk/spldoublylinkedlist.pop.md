# SplDoublyLinkedList::pop

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

SplDoublyLinkedList::pop - Видаляє (виштовхує) вузол, що знаходиться в кінці двозв'язкового списку

### Опис

```methodsynopsis
public SplDoublyLinkedList::pop(): mixed
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Значення віддаленого вузла.

### Помилки

Викидає виняток [RuntimeException](class.runtimeexception.md)коли структура даних порожня.
