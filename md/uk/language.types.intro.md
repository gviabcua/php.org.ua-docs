## Вступ

PHP підтримує десять найпростіших типів.

Чотири скалярні типи:

-   bool
-   int
-   float (число з плаваючою точкою, також відоме як double)
-   string

Чотири змішані типи:

-   array
-   object
-   [callable](language.types.callable.html)
-   [iterable](language.types.iterable.html)

І, нарешті, два спеціальні типи:

-   resource
-   NULL

Ви можете знайти кілька згадок типу подвійний точності (double). Розглядайте його як число з точкою, що плаває, два імені існують тільки з історичних причин.

Як правило, програміст не встановлює тип змінної; зазвичай, це робить PHP під час виконання програми залежно від контексту, в якому використовується змінна.

> **Зауваження**: Якщо ви бажаєте перевірити тип і значення певного [выражения](language.expressions.html), використовуйте [vardump()](function.var-dump.html)
> 
> Якщо ж вам для налагодження необхідно просто представлення типу, що легко читається, використовуйте [gettype()](function.gettype.html). Щоб перевірити певний тип, *не* використовуйте [gettype()](function.gettype.html), застосовуйте для цього функції `is_type`. Ось кілька прикладів:
> 
> ```php
> <?php
> $a_bool = TRUE;   // логический
> $a_str  = "foo";  // строковый
> $a_str2 = 'foo';  // строковый
> $an_int = 12;     // целочисленный
> 
> echo gettype($a_bool); // выводит: boolean
> echo gettype($a_str);  // выводит: string
> 
> // Если это целое, увеличить на четыре
> if (is_int($an_int)) {
>     $an_int += 4;
> }
> 
> // Если $a_bool - это строка, вывести её
> // (ничего не выводит)
> if (is_string($a_bool)) {
>     echo "Строка: $a_bool";
> }
> ?>
> ```

Якщо ви хочете примусово змінити тип змінної, ви можете або [привести](language.types.type-juggling.html#language.types.typecasting) змінну, або використовувати функцію [settype()](function.settype.html)

Зверніть увагу, що змінна, залежно від її типу зараз, у певних ситуаціях може мати різні значення. Більш детальну інформацію дивіться у розділі [Маніпуляції з типами](language.types.type-juggling.html). Також вам, можливо, буде цікаво подивитись [таблицы сравнения типов](types.comparisons.html), оскільки в них наведено приклади різних порівнянь, пов'язаних із типами.
