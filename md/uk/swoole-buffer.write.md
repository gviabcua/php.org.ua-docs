# Swoole\\Buffer::write

(PECL swoole >= 1.9.0)

Swoole\\Buffer::write — записує дані до буфера пам'яті. Пам'ять виділена для буфера не буде змінена

### Опис

```methodsynopsis
public Swoole\Buffer::write(int $offset, string $data): void
```

### Список параметрів

`offset`

Зміщення.

`data`

Дані для запису буфер пам'яті.
