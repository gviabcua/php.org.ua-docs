# Огляд PHP драйверів MySQL

# Вступ

Для доступу до бази даних MySQL існує декілька API-інтерфейсів PHP. Користувачі можуть вибирати між модулями [mysqli](book.mysqli.md) або [PDO\_MySQL](ref.pdo-mysql.md)

У цьому посібнику описується [термінологія](mysqlinfo.terminology.md) використовується в описах кожного API, інформація по [вибору відповідного API](mysqlinfo.api.choosing.md), та інформація яка допоможе вам вибрати підходящу, для використання з цим API, MySQL [бібліотеку](mysqlinfo.library.choosing.md)
