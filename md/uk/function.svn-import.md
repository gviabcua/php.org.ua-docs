# svnimport

(PECL svn >= 0.2.0)

svnimport — Імпорт шляху без версії до репозиторії

### Опис

```methodsynopsis
svn_import(string $path, string $url, bool $nonrecursive): bool
```

Додавання неверсіонованого шляху `path` у репозиторій за адресою `url`. Якщо `path` є директорією та параметр `nonrecursive` має значення **`false`**, директорія буде додана до репозиторію рекурсивно.

### Список параметрів

`path`

Шлях до файлу чи директорії для імпорту.

> **Зауваження**: Відносні шляхи будуть обчислені, якби поточна робоча директорія була домашньою папкою самого PHP Щоб використовувати робочу директорію скрипта, що викликає, використовуйте [realpath()](function.realpath.md) або dirname(FILE

`url`

URL-адреса репозиторію.

`nonrecursive`

Чи слід обробити директорії рекурсивно чи ні.

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.

### Приклади

**Приклад #1 Простий приклад**

Цей приклад ілюструє базове використання цієї функції. Імпорт директорії з ім'ям `new-files` у репозиторій з адресою `http://www.example.com/svnroot/incoming/abc` виглядає наступним чином:

```php
<?php
svn_import(realpath('new-files'), 'http://www.example.com/svnroot/incoming/abc', false);
?>
```

### Примітки

**Увага**

Ця функція є *ЕКСПЕРИМЕНТАЛЬНОЇ*. Поведінка цієї функції, її ім'я та документація, що до неї належить, можуть змінитися в наступних версіях PHP без повідомлення. Використовуйте цю функцію на свій страх та ризик.
