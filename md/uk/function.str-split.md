# str\_split

(PHP 5, PHP 7, PHP 8)

str\_split — Перетворює рядок на масив

### Опис

```methodsynopsis
str_split(string $string, int $length = 1): array
```

Перетворює рядок на масив.

### Список параметрів

`string`

Вхідний рядок.

`length`

Максимальна довжина фрагмента.

### Значення, що повертаються

Якщо вказано необов'язковий параметр `length`, що повертається масив буде розбитий на фрагменти, кожен з яких матиме довжину `length`, За винятком останнього фрагмента, який може бути коротшим, якщо рядок ділиться нерівномірно. За замовчуванням параметр `length`равен тобто розмір кожного фрагмента буде один байт.

### Помилки

Якщо параметр `length`меньше , буде викинута помилка [ValueError](class.valueerror.md)

### список змін

| Версия | Опис |
| --- | --- |
| 8.2.0 | Якщо параметр `string` не заданий, тепер вертається порожній масив (array). Раніше повертався масив (array), що містить один порожній рядок |
| 8.0.0 | Тепер якщо параметр `length`меньше , буде викинута помилка [ValueError](class.valueerror.md); раніше, натомість видавалася помилка рівня **`E_WARNING`**, а функція повертала **`false`** |

### Приклади

**Приклад #1 Приклад використання функції** str\_split()\*\*\*\*

```php
<?php

$str = "Hello Friend";

$arr1 = str_split($str);
$arr2 = str_split($str, 3);

print_r($arr1);
print_r($arr2);

?>
```

### Примітки

> **Зауваження** :
> 
> Функция**str\_split()** здійснює розбивку за байтами, а не за символами, у разі використання рядків у багатобайтних кодуваннях. Використовуйте функцію [mb\_str\_split()](function.mb-str-split.md), щоб розбити рядок кодових точок.
