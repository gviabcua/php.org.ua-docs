# Вступ

Модуль для ефективного моніторингу процесів введення/виводу, та подій таймера та сигналів, що використовує найкращий з доступних для конкретної платформи механізмів сповіщення про події введення/виведення. Модуль є переписаною для PHP бібліотеки libevent.

> **Зауваження**
> 
> Зверніть увагу, що підтримка Windows з'явилася в `event-1.9.0`

У версії додано нове об'єктно-орієнтоване API (що ламає зворотну сумісність) та підтримка libevent 2+, включаючи HTTP, DNS, OpenSSL та перехоплювач сигналів.

> **Зауваження**
> 
> Зверніть увагу, що `event-1.0.0` та вище не сумісні з попередніми версіями.
