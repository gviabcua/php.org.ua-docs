# Fiber::isStarted

(PHP 8> = 8.1.0)

Fiber::isStarted — Визначає, чи запущено файбер

### Опис

```methodsynopsis
public Fiber::isStarted(): bool
```

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає **`true`** тільки після того, як файбер був запущений, інакше повертає **`false`**
