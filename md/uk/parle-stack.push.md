# ParleStack::push

(PECL parle >= 0.5.1)

ParleStack::push — Поміщає елемент у стек

### Опис

```methodsynopsis
public Parle\Stack::push(mixed $item): void
```

### Список параметрів

`item`

Змінна, яку необхідно помістити у стек
