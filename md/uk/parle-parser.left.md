# ParleParser::left

(PECL parle >= 0.5.1)

ParleParser::left - Оголошує токен з лівою асоціативністю

### Опис

```methodsynopsis
public Parle\Parser::left(string $tok): void
```

Оголошує токен з лівою асоціативністю.

### Список параметрів
