# ReflectionClass::getModifiers

(PHP 5, PHP 7, PHP 8)

ReflectionClass::getModifiers — Повертає інформацію про модифікаторів класу

### Опис

```methodsynopsis
public ReflectionClass::getModifiers(): int
```

Повертає бітову маску модифікаторів даного методу.

### Список параметрів

Ця функція не має параметрів.

### Значення, що повертаються

Повертає бітову маску з [констант модифікаторів](class.reflectionclass.html#reflectionclass.constants.modifiers)
