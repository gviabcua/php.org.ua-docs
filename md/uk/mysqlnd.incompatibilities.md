# Несумісності

Вбудований драйвер MySQL (mysqlnd), як правило, сумісний з клієнтською бібліотекою MySQL `libmysql`. Цей розділ описує несумісність цих бібліотек.

-   Значення у полях даних типу `bit` у бібліотеці `libmysql` повертаються як бінарні рядки (наприклад, "0" або "x1F"), а в бібліотеці `mysqlnd` - як десяткові рядки (наприклад, "0" або "31"). Якщо необхідно, щоб код був сумісний з обома бібліотеками, завжди з MySQL повертайте дані з полів типу `bit` у вигляді чисел, використовуючи такий запит: `SELECT bit + 0 FROM table`
