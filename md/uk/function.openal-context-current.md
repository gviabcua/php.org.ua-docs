# openalcontextcurrent

(PECL openal >= 0.1.0)

openalcontextcurrent — Зробити вказаний контекст поточним

### Опис

```methodsynopsis
openal_context_current(resource $context): bool
```

### Список параметрів

`context`

Ресурс [Open AL(Context)](openal.resources.html) (Створений раніше за допомогою [openalcontextcreate()](function.openal-context-create.html)

### Значення, що повертаються

Повертає **`true`** у разі успішного виконання або **`false`** у разі виникнення помилки.
